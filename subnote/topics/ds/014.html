2. IaaS (Infrastructure as a Service)

  • 정의: 서버·스토리지·네트워크 등 IT 인프라를 가상화하여 온디맨드로 제공하고 사용량 기반으로 과금하는 클라우드 서비스 모델
  • 키워드: VM, 가상 네트워크, 스토리지, 온디맨드, 종량과금, AWS EC2, Azure VM
  • 분류: 클라우드 컴퓨팅
  • 특징: 인프라 프로비저닝 자동화, 글로벌 데이터센터 활용, 탄력적 확장·축소, OS 이상은 사용자 관리
  • 주요 사례: AWS EC2·S3, Azure Virtual Machines, GCP Compute Engine, KT Cloud
  • 인사이트: IaaS는 온프레미스 대비 초기 투자 없이 글로벌 인프라 활용이 가능하나, 사용자가 OS·미들웨어·앱 보안을 직접 책임지는 공동 책임 모델을 명확히 이해해야 함.
