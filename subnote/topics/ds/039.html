2. 반응형 웹 디자인 (RWD, Responsive Web Design)

정의: 단일 HTML 코드베이스로 다양한 화면 크기(데스크톱·태블릿·모바일)에 최적화된 레이아웃을 자동 제공하는 웹 디자인 방식
키워드: 미디어 쿼리(Media Query), 유동형 그리드, 유동 이미지, 뷰포트(Viewport)
분류: API/웹 기술
