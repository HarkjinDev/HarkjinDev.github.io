2. 반응형 웹 디자인 (RWD, Responsive Web Design)

  • 정의: 단일 HTML 코드베이스로 다양한 화면 크기(데스크톱·태블릿·모바일)에 최적화된 레이아웃을 자동 제공하는 웹 디자인 방식
  • 키워드: 미디어 쿼리(Media Query), 유동형 그리드, 유동 이미지, 뷰포트(Viewport)
  • 분류: API/웹 기술
  • RWD 핵심 기술:
    기술설명
    미디어 쿼리브레이크포인트별 CSS 적용 (@media screen and (max-width: 768px))
    유동형 그리드고정 픽셀 대신 % 단위 레이아웃
    유동 이미지max-width: 100%로 이미지 자동 조절
    뷰포트 메타태그모바일 브라우저 렌더링 너비 지정
  • RWD vs 적응형 웹(AWD) 비교:
    구분RWDAWD
    방식CSS로 유동적 조절기기별 별도 레이아웃
    코드베이스단일다중
    서버 감지불필요필요 (User-Agent)
    유지보수쉬움복잡
  • 웹 접근성(WA, Web Content Accessibility): WCAG 2.0 표준
    • 인식(Perceivable) 용이성, 운용(Parable) 용이성, 이해(Understand) 용이성, 견고성(Robust)
  • 인사이트: RWD는 “모바일 퍼스트(Mobile First)” 전략과 결합 시 효과적이며, 구글의 모바일 친화성 검색 순위 반영으로 SEO와도 직결됨.
