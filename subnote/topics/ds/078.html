3. 데이터 거래소 (Data Exchange)

  • 정의: 데이터 공급자와 수요자를 연결하여 데이터의 거래·유통·정산을 중개하는 플랫폼 기반 데이터 마켓플레이스
  • 키워드: 데이터 마켓플레이스, 중개 플랫폼, 데이터 품질 인증, 비식별화, KDX, 금융 데이터 거래소
  • 분류: 데이터 경제
  • 구성 요소: 데이터 카탈로그, 가격 산정 엔진, 계약·정산 시스템, 품질 검증 모듈, 보안 전송 채널
  • 주요 이슈: 데이터 품질 신뢰 확보, 개인정보 보호법 준수, 표준 계약서 부재, 가격 산정 객관성
  • 인사이트: 데이터 거래소는 데이터 경제 생태계의 유통 인프라로, 거래 활성화를 위해 데이터 품질 인증 체계와 표준 라이선스 모델 정립이 선결 과제임.
