6. BaaS (Backend as a Service)

  • 정의: 모바일·웹 앱 개발 시 인증·푸시 알림·DB·파일 스토리지 등 공통 백엔드 기능을 API로 제공하는 클라우드 서비스
  • 키워드: Firebase, AWS Amplify, 인증(Auth), 실시간 DB, 푸시 알림, 서버리스 백엔드
  • 분류: 클라우드 컴퓨팅
  • 제공 기능: 사용자 인증(OAuth·SNS 로그인), 실시간 데이터베이스, 파일 스토리지, 푸시 알림, 서버리스 함수
  • 인사이트: BaaS는 프론트엔드 개발자가 백엔드 없이 완성된 앱을 출시할 수 있게 하여 MVP 개발 속도를 혁신적으로 단축하나, 복잡한 비즈니스 로직 구현에는 한계가 있음.
