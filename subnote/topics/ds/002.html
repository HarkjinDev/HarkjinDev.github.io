1. 가상화 (Virtualization)

  • 정의: 물리적 하드웨어 자원을 소프트웨어로 추상화하여 다수의 독립적 논리 환경을 생성·운용하는 기술
  • 키워드: 하이퍼바이저, VM, 추상화, 자원 공유, 격리, 서버 가상화
  • 분류: 가상화/컨테이너
  • 가상화 유형/기술:
    구분유형가상화 기술
    인프라 자원서버 가상화파티셔닝·가상I/O·하이퍼바이저
    스토리지 가상화블록 가상화·가상 테이프라이브러리
    네트워크 가상화L2~L7 활용·VPN·VLAN
    정보파일 가상화클러스터 파일·그리드 파일 시스템
    데이터 가상화데이터 연합·콘솔리데이션
    워크로드트랜잭션 가상화JVM 로드 밸런싱
    태스크 가상화컴퓨팅 그리드
    프레젠테이션 가상화서비스 기반 컴퓨팅 (SBC)
    운영환경전사적 워크로드Enterprise Workload Manager
    전력·냉각 효율화Hibernation·Partition Mobility
    유틸리티 서비스Metering·Provisioning
    백업 가상화VTL 백업·테이프 라이브러리 기반 백업
    클라이언트 데스크탑 가상화PC 가상화·SBC·VDI
  • 가상화 동작 방식: 물리 서버 위에 하이퍼바이저(VMM) 설치 → Guest OS별 독립 VM 생성 → 자원(CPU·메모리·스토리지) 논리 분할
  • 원격 데스크톱 프로토콜: RDP(Microsoft), VNC, ICA(Citrix) - 가상 데스크톱 원격 접속 핵심
  • 인사이트: 가상화는 클라우드 컴퓨팅의 기반 기술로, 서버 통합을 통한 TCO 절감과 빠른 프로비저닝이 핵심 가치이며, 컨테이너와의 비교(오버헤드, 부팅 속도, 격리 수준)를 서술하면 차별화 가능.
