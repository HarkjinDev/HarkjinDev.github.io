3. 웹 성능 최적화 (Web Performance Optimization)

정의: 웹 페이지의 로딩 속도와 사용자 체감 성능을 개선하기 위한 기술적 전략과 기법의 총칭
키워드: Core Web Vitals, LCP, FID, CLS, CDN, 레이지 로딩, 코드 분할, 캐싱, SEO
분류: API/웹 기술

인사이트: 웹 성능은 사용자 이탈률과 전환율에 직접 영향을 미치며, Google의 Core Web Vitals가 검색 순위 알고리즘에 반영되어 SEO와 성능 최적화가 연계된 전략 필요.
