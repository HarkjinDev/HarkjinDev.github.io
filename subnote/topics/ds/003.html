2. 하이퍼바이저 (Hypervisor)

  • 정의: 물리 하드웨어 위에서 다수의 가상머신을 생성·관리하며 자원을 할당하는 가상화 관리 소프트웨어 계층
  • 키워드: Type 1(베어메탈), Type 2(호스트), VMM, KVM, VMware ESXi, Xen
  • 분류: 가상화/컨테이너
  • 유형 비교:
    유형특징사례
    Type 1 (베어메탈)하드웨어 직접 위에 설치, 고성능·고안정성VMware ESXi, Xen, KVM
    Type 2 (호스트)기존 OS 위에 설치, 개발·테스트용VMware Workstation, VirtualBox
  • 인사이트: 엔터프라이즈 환경에서는 Type 1 하이퍼바이저가 표준이며, KVM은 Linux 커널 내장으로 클라우드 IaaS(AWS EC2, KT클라우드)의 핵심 기반임을 강조.
