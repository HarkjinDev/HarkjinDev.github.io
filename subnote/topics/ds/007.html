6. 쿠버네티스 (Kubernetes)

  • 정의: 컨테이너화된 워크로드의 자동 배포·스케일링·복구·서비스 디스커버리를 관리하는 오픈소스 컨테이너 오케스트레이션 플랫폼
  • 키워드: Pod, Node, Deployment, Service, HPA, CNCF, 셀프힐링, 선언적 관리
  • 분류: 가상화/컨테이너
  • 주요 컴포넌트:
    구분항목설명 핵심
    Master NodeKube API Server명령 전달 및 REST 통신REST API 서버
    Kube Scheduler자원 상황별 노드 할당노드 스케줄링
    Kube Controller다양한 컨트롤러 관리제어 루틴 관리
    etcd클러스터 데이터 저장소키-값 저장소
    Worker NodePod가장 기본 배포 단위컨테이너 그룹
    Kubelet노드 에이전트, 실행 관리노드 에이전트
    Kube-Proxy네트워크 관리 및 로드밸런싱네트워킹/LB
    컨테이너 런타임컨테이너 실제 실행 환경실행 환경(CRI)
    Userkubectl쿠버네티스 명령 실행 도구CLI 도구
    애드온DNS서비스용 DNS 레코드 제공네이밍 서비스
    Dashboard웹 기반 클러스터 UI웹 관리 도구
    리소스 모니터링시계열 매트릭스 기록 열람자원 모니터링
    로깅로그 중앙 저장 및 검색통합 로그 관리
  • HPA(Horizontal Pod Autoscaler): CPU·메모리 사용률에 따라 Pod 수 자동 조정
  • 인사이트: Kubernetes는 클라우드 네이티브의 사실상 표준으로, 선언적 구성(Desired State)과 셀프힐링으로 운영 자동화를 실현하며, CNCF 생태계(Helm, Istio, Prometheus)와의 연계를 언급하면 고득점 가능.
