3. 온톨로지(Ontology)

  • 정의: 실세계 개념과 속성, 데이터의 관계를 구조화하고 시멘틱웹으로 표현하는 지식 베이스
  • 키워드: 개념 체계화, 의미 명세화, 추론기반, 클래스, 인스턴스, 속성, 관계, 제약조건, 공리, RDFS, OWL, XTM, SPARQL 쿼리, 추론엔진, 지식 그래프, 시멘틱웹, 대화형 AI, 로보틱스, XAI
  • 분류: 신기술/동향
  • 구성요소: <개속관계공>
    구분설명핵심 역할
    개념(Concepts/Instances)객체, 개념 및 개별 인스턴스지식 표현의 기본 단위
    속성(Properties)개체가 가지는 고유 특성 정보객체의 세부 특징 기술
    관계(Relationships)개체 간의 연관성 및 상호작용지식 간의 연결 고리 형성
    계층구조(Taxonomy)개념 간의 상하위 계층 체계지식의 체계적 분류
    공리(Axioms)관계의 제약조건 및 추론 규칙지식의 정당성 및 논리 검증
  • 온톨로지 프리미티브: 도메인을 일관된 의미(semantics)로 표현하기위해 사용하는 최소단위 모델링 요소
    • 구성요소: RACI(관계, 속성, 클래스, 인스턴스), 로직(공리, 계층구조)
    • 계층 구조[(클래스) → (공리) (클래스) → (인스턴스) ↔ (관계) ↔ (인스턴스)]
  • 지식그래프: 개체-관계-속성을 그래프 구조(노드, 엣지, 속성)로 정리하여 정보를 의미상으로 구조화적으로 표현한 그래프 기반 지식 구조
    • 특징: 의미 계층, 관계 중심 질의, 맥락 유지
    • 구성요소: 노드(Node), 엣지(Edge), 속성(Property)
    • 기술요소: 그래프 데이터 모델링, 지식 추론, 지식 그래프 임베딩(KGE), RDF, SPARQL
  • 온톨로지 언어 및 표현 기술:
    구분설명비고
    OWL(Web Ontology Language)W3C 표준 온톨로지 저작 언어복잡한 논리 표현 가능
    RDF(Resource Description Framework)자원 표현용 기본 데이터 모델주어-서술어-목적어 구조
    SPARQL(Standard Query Language)RDF 데이터용 표준 질의 언어그래프 데이터베이스 조회
    XTM(XML Topic Maps)ISO 표준 기반 지식 표현 체계의미적 상호 연결 강조
  • 인사이트: 온톨로지는 단순한 데이터 저장 구조를 넘어 기계가 의미를 이해하고 추론할 수 있도록 지식을 형식화하는 핵심 기반 기술로, 시맨틱웹의 근간을 형성함. LLM의 할루시네이션 문제해결방법으로 AI 신뢰성 확보의 보완 수단으로 주목받고 있음. 또한, 설명가능한 AI 구현의 기반 인프라로 역할이 확대되고 있음.
