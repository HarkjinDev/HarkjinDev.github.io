1. 슈퍼앱 (Super App)

  • 정의: 메시징·결제·쇼핑·교통 등 다양한 서비스를 단일 앱 생태계에서 통합 제공하여 사용자의 디지털 생활을 포괄하는 올인원 플랫폼 앱
  • 키워드: 미니앱, 플랫폼 락인, 위챗, 카카오, 네트워크 효과, API 통합
  • 분류: 앱/서비스 트렌드
  • 주요 요소: 강력한 사용자 기반, 결제 인프라, 미니앱 생태계, 풍부한 사용자 데이터, 플랫폼 API
  • 슈퍼앱 vs 멀티앱 비교:
    구분슈퍼앱멀티앱
    전략단일 앱 내 통합서비스별 독립 앱
    사용자 경험원스톱, 전환 없음서비스 전문성 높음
    사례위챗, 카카오, 그랩Google(Maps·Pay·Drive), Meta
  • 주요 이슈: 독점·반경쟁 우려, 개인정보 집중, 플랫폼 규제
  • 인사이트: 슈퍼앱의 성공 열쇠는 미니앱 생태계 구축을 통한 개발자 유입과 결제 데이터 기반 초개인화 서비스 제공이며, 국내외 규제 리스크를 균형 있게 논의해야 함.
