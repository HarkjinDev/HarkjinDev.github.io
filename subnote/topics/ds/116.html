6. 인터미턴트 컴퓨팅 (Intermittent Computing)

  • 정의: Energy Harvesting 시스템이 전력 손실로 인한 잦은 중단 환경에서도 연산을 지속적으로 진행할 수 있도록 하는 컴퓨팅 패러다임
  • 키워드: 에너지 하베스팅, 체크포인트, 비휘발성 메모리(NVM), 태스크 기반 실행, 배터리리스, 멱등성, 전력 중단 복구
  • 분류: 신기술/동향
  • 등장 배경:
    항목설명핵심
    배터리 교체 한계오지·극한 환경 배치 IoT 기기 배터리 교체 불가유지보수 불가 환경
    에너지 하베스팅 발전태양광·RF·진동 에너지 수확 기술 성숙대체 전원 가능
    IoT 기기 폭발적 증가수십억 개 IoT 기기의 지속 운용 문제배터리리스 수요
    NVM 기술 발전FRAM·PCM·MRAM 비휘발성 메모리 상용화상태 보존 가능
  • 핵심 기술요소:
    구분기술설명핵심
    에너지에너지 하베스팅태양광·RF·압전·열전 등 주변 에너지 수확배터리리스 전원
    에너지 버퍼수확 에너지 임시 저장 커패시터·슈퍼캡에너지 임시 저장
    전력 관리 IC수확 에너지 안정화·임계 전압 제어전력 안정화
    메모리NVM (비휘발성 메모리)FRAM·PCM·MRAM 전력 끊겨도 데이터 보존상태 영구 보존
    하이브리드 메모리SRAM(속도)+NVM(비휘발성) 혼합 구조성능·보존 균형
    실행 모델체크포인트 실행 중 상태를 NVM에 주기적 저장·복원중단 후 재개
    태스크 기반 실행원자적 태스크 단위로 분할·멱등성 보장원자적 실행
    멱등성 (Idempotency)동일 태스크 반복 실행 시 동일 결과 보장중복 실행 안전
    소프트웨어런타임 체크포인트잔여 에너지 기반 자동 체크포인트 결정에너지 인식 저장
    컴파일러 자동 삽입컴파일 단계에서 체크포인트 코드 자동 생성자동화
  • 에너지 하베스팅 소스: 태양광 (Solar), RF (Radio Frequency), 압전 (Piezoelectric), 열전 (Thermoelectric), 운동 에너지 (Kinetic)
  • 활용 분야:
    분야설명핵심
    배터리리스 IoT교체 불가 환경 장기 운용 센서 노드배터리리스 센서
    헬스케어 웨어러블인체 열·운동 에너지 기반 의료 모니터링무전원 의료 기기
    스마트 농업태양광 기반 토양·기상 센싱오지 환경 모니터링
    구조물 모니터링교량·터널 진동 에너지 기반 상태 감지인프라 모니터링
    RFID·NFC 태그RF 에너지 기반 배터리리스 식별·센싱수동형 태그
    우주·극한 환경배터리 교체 불가 극한 환경 운용극한 환경 센싱
  • 핵심 설계 과제: 데이터 일관성, 체크포인트 오버헤드, 멱등성 보장, 에너지 예측, NVM 수명
  • 인사이트: 인터미턴트 컴퓨팅은 배터리 교체가 불가능한 원격·오지·극한 환경에서 지속가능하고 유지보수 불필요한 기기를 실현하는 핵심 패러다임으로, IoT 기기의 폭발적 증가와 에너지 하베스팅 기술 성숙으로 실용화가 가속되고 있음. 체크포인트·태스크 기반 실행·멱등성 설계의 3축이 인터미턴트 컴퓨팅 안정성의 핵심이며, NVM 기술 고도화와 에너지 인식 컴파일러 자동화가 개발 부담을 낮추는 핵심 방향임
