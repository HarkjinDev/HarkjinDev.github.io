2. 데이터 가치평가 (Data Valuation)

  • 정의: 데이터의 경제적 가치를 비용, 시장, 수익 접근법 등을 통해 정량적으로 산출하는 방법론
  • 키워드: 비용 접근법, 시장 접근법, 수익 접근법, 데이터 품질, 재화 특성, KDATA
  • 분류: 데이터 경제
  • 데이터의 재화적 특성: 비소진성(사용해도 소멸 안 함), 비경합성(동시 사용 가능), 무형성, 맥락 의존성
  • 평가 모델:
    접근법방법특징
    수익 접근법데이터 활용 기대 수익 현재가치미래 가치 반영, 불확실성 존재
    원가 접근법데이터 수집·가공 비용 산정객관적, 가치 저평가 우려
    시장 접근법유사 데이터 시장 거래 가격 비교시장 성숙 시 유효
  • 활용방안: 데이터 자산 회계 처리, 데이터 거래소 가격 산정, M&A 시 데이터 자산 평가
  • 인사이트: 데이터 가치평가는 무형 자산 회계 처리의 신규 영역으로, 표준화된 평가 기준 부재가 최대 과제이며 한국데이터산업진흥원(KDATA)의 가이드라인을 언급하면 차별화 가능.
