1. 데이터 커머스 (Data Commerce)

  • 정의: 데이터를 독립적 상품으로 거래하거나 데이터 분석 기반으로 개인화 상거래를 실현하는 데이터 기반 경제 활동
  • 키워드: 데이터 상품화, 데이터 중개, 개인화 추천, 데이터 마켓플레이스, 데이터 수익화
  • 분류: 데이터 경제
  • 주요 기술: 빅데이터 분석, 추천 엔진, 개인정보 비식별화(k-익명성, 차분 프라이버시), 데이터 거래소
  • 특징: 비소모성(원본 보존), 재사용성, 결합 가치 창출, 네트워크 효과
  • 활용 분야: 마케팅, 금융 신용평가, 의료 임상 데이터, 공공 정책 결정
  • 인사이트: 데이터 커머스는 데이터의 비경쟁재적 특성(동시 사용 가능)으로 전통 재화와 다른 가치 창출 모델을 가지며, 개인정보 보호와의 균형이 핵심 쟁점임.
