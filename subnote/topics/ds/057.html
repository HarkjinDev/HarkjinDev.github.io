2. LOD (Linked Open Data)

정의: 웹 표준(URI, RDF, SPARQL)을 활용하여 기관·도메인 간 데이터를 연결하고 공개하는 구조화된 개방형 데이터 체계
키워드: RDF, URI, SPARQL, 5성급 모델, 시맨틱 웹, 데이터 링크
분류: 디지털 서비스 기초
Tim Berners-Lee 5성급 LOD:
등급
기준
★
어떤 형식이든 오픈 라이선스로 공개
★★
구조화된 기계 판독 가능 형식 (ex. Excel)
★★★
비독점 형식 사용 (ex. CSV)
★★★★
W3C 표준(RDF, SPARQL) 사용
★★★★★
다른 데이터와 링크 연결
인사이트: LOD는 공공 데이터 개방 전략의 최상위 목표로, 데이터 사일로를 해소하고 지식 그래프 구축 기반이 됨. 오픈 데이터와의 차이(연결성 유무)를 명확히 서술.
