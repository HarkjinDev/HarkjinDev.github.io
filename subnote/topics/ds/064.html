1. 오픈뱅킹 (Open Banking)

  • 정의: 금융기관이 계좌·거래 API를 핀테크 기업에 표준 방식으로 개방하여 금융 서비스 혁신과 경쟁을 촉진하는 체계
  • 키워드: Open API, PSD2, 핀테크, 마이데이터, 계좌 통합 조회, 결제 이니시에이션
  • 분류: 핀테크/금융
  • 등장 배경: 영국 PSD2(2018) → 유럽 은행 API 의무 개방 → 국내 2019년 금융결제원 오픈뱅킹 출범
  • 구성 주체 및 역할:
    주체역할
    은행(정보제공자)계좌·거래 API 제공
    핀테크 기업(이용기관)API 활용 서비스 개발
    금융결제원(중개기관)API 게이트웨이 운영, 표준 제정
    고객동의 기반 데이터 제공
  • 기대 효과: 금융 서비스 혁신, 소비자 선택권 확대, 핀테크 산업 활성화, 마이데이터와 연계한 초개인화 금융
  • 인사이트: 오픈뱅킹은 마이데이터(금융 분야 확장)의 기반 인프라로, 보안(OAuth 2.0 기반 인증)·동의 관리가 핵심 쟁점임을 강조.
