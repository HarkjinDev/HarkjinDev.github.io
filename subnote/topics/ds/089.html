2. 미니앱 (Mini App)

  • 정의: 슈퍼앱 플랫폼 내에서 별도 설치 없이 즉시 실행 가능한 경량 서비스 애플리케이션
  • 키워드: 경량 앱, 웹 기반, 슈퍼앱 생태계, 서드파티, 위챗 미니프로그램, 즉시 실행
  • 분류: 앱/서비스 트렌드
  • 특징: 앱 다운로드·설치 불필요, 웹 기술(HTML5) 기반, 슈퍼앱 API 활용, 빠른 로딩
  • 슈퍼앱과의 관계: 미니앱 = 슈퍼앱의 확장 생태계 → 서드파티 기업이 슈퍼앱 내에 서비스를 배포하여 기존 사용자에게 즉시 노출
  • 인사이트: 미니앱은 앱 피로도 해소와 유통 비용 절감 모델로, 중소 서비스 기업의 빠른 시장 진입 수단이 일 수 있음. 플랫폼 의존도 심화에 따른 생태계 종속 리스크도 언급 필요.
