9. 서버리스 컴퓨팅 (Serverless Computing)

  • 정의: 서버 프로비저닝·관리 없이 코드를 실행하고 실제 사용 자원에만 과금하는 클라우드 실행 모델
  • 키워드: FaaS, BaaS, 이벤트 드리븐, 콜드 스타트, AWS Lambda, 무상태 아키텍처
  • 분류: 클라우드 컴퓨팅
  • 서버리스 구성 요소: FaaS(함수 실행) + BaaS(백엔드 서비스) = 서버리스 아키텍처
  • 적합한 워크로드: 이벤트 기반 처리, API 백엔드, 배치 작업, IoT 데이터 처리
  • 인사이트: '서버리스'는 서버가 없는 것이 아니라 서버 관리를 추상화한 것으로, MSA와 결합 시 개별 기능별 독립 배포·확장이 가능해 운영 효율이 극대화됨.
