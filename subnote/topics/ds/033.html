[하위] 1-1. 개방형 API (Open API)

정의: 외부 개발자·사용자가 자유롭게 접근·활용할 수 있도록 공개된 API
키워드: 공공데이터포털, API 키, 플랫폼 생태계, 매시업(Mashup), 디지털 전환
분류: API/웹 기술
Open API 특징:
접근성: 누구나 사용 가능 (API 키 인증 후)
문서화: 공개 명세서 제공 (OAS 기반)
생태계: 서드파티 애플리케이션 생성 유도
수익화: 호출 건수 기반 과금 모델 가능
인사이트: 공공데이터포털 등 정부 개방형 API는 디지털 정부 서비스 혁신의 기반으로, 보안(API 키·OAuth)과 버전 관리가 운영의 핵심 과제.
