5. FaaS (Function as a Service)

정의: 코드를 함수 단위로 배포하고 이벤트 발생 시에만 실행·과금하는 서버리스 컴퓨팅 모델
키워드: 서버리스, Lambda, Azure Functions, 이벤트 드리븐, 콜드 스타트, 무상태
분류: 클라우드 컴퓨팅
특징: 서버 관리 불필요, 이벤트 트리거 실행, 밀리초 단위 과금, 자동 스케일링(0→∞)
장단점:
인사이트: FaaS는 이벤트 기반 경량 작업(알림, 이미지 처리, API 게이트웨이)에 최적이며, 콜드 스타트 문제는 Provisioned Concurrency나 언어 선택(Go vs Python)으로 완화 가능함.
