3. SaaS (Software as a Service)

  • 정의: 인터넷 브라우저로 완성된 소프트웨어를 구독 방식으로 이용하고 인프라·플랫폼 관리를 공급자가 전담하는 클라우드 서비스 모델
  • 키워드: 구독형, 멀티테넌시, 제로 설치, Salesforce, Office 365, 자동 업데이트
  • 분류: 클라우드 컴퓨팅
  • 특징: 즉시 사용(설치 불필요), 자동 업그레이드, 멀티테넌트 아키텍처, 어디서나 접근 가능
  • 장단점: 장점(저TCO, 빠른 도입) / 단점(커스터마이징 제한, 데이터 종속성, 인터넷 의존)
  • 인사이트: SaaS는 기업 IT 운영 부담 최소화 모델로, 벤더 종속(Vendor Lock-in) 리스크를 줄이기 위한 데이터 이식성과 표준 API 지원 여부를 도입 전 반드시 검토해야 함.
