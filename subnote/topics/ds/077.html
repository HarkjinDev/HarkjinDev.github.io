[하위] 2-1. 데이터 자산화

  • 정의: 데이터를 경제적 가치의 무형 자산으로 인식하고 체계적으로 가공·관리하여 비즈니스·의사결정에 활용 가능하도록 전환하는 과정
  • 과정: 데이터 표준화 → 메타데이터 관리 → 접근권한 통제 → 재사용 정책
  • 핵심요소:
    핵심요소설명특징
    데이터 품질수집 단계부터 체계적 품질 관리데이터 신뢰성 확보
    데이터 보안·프라이버시 보호법적 규제 준수·안전한 보호 관리컴플라이언스·보안
    데이터 거버넌스정책·표준화·조직·프로세스 구축데이터 관리 원칙·조직·프로세스·인프라
    데이터 인프라저장·처리·분석 기술 인프라 구축데이터 카탈로그·메타데이터
    데이터 분석 역량데이터 분석으로 인사이트 도출데이터 사이언스·분석
