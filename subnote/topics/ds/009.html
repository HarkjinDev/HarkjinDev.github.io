8. VDI (Virtual Desktop Infrastructure)

  • 정의: 데이터센터 서버에서 개인 데스크톱 환경을 가상화하여 중앙 관리하고, 사용자는 씬 클라이언트나 PC로 원격 접속하는 데스크톱 가상화 인프라
  • 키워드: 씬 클라이언트, 세션 브로커, 중앙 관리, RDP/ICA/PCoIP, 보안 USB, BYOD, 재택근무
  • 분류: 가상화/컨테이너
  • 계층별 구성요소:
    계층구성 요소
    클라이언트씬 클라이언트, PC, 모바일
    프로토콜RDP, ICA(Citrix), PCoIP(VMware)
    가상화하이퍼바이저 + 가상 데스크톱 VM
    스토리지공유 스토리지 (SAN/NFS)
    관리VDI 관리 플랫폼 (Citrix, VMware Horizon)
  • 인사이트: VDI는 재택근무·보안 강화·BYOD 대응의 실용적 솔루션으로, 초기 구축 비용이 높으나 데이터 중앙화로 보안 사고 시 피해 최소화가 가능함. DaaS와의 비교(온프레미스 vs 클라우드)를 언급하면 차별화 가능.
