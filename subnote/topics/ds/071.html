8. 블록체인 (Blockchain)

  • 정의: 거래 데이터를 블록 단위로 묶어 체인 형태로 연결하고 네트워크 참여자 전체가 분산 저장·검증하는 분산 원장 기술(DLT)
  • 키워드: 분산 원장, 합의 알고리즘, 스마트 컨트랙트, 해시, 탈중앙화, 불변성, PoW·PoS, DeFi
  • 분류: 분산 시스템 / 암호화 / 핀테크
  • 블록체인 핵심 구조:
    구성요소설명핵심
    블록 (Block)트랜잭션 데이터·타임스탬프·이전 블록 해시 묶음데이터 저장 단위
    체인 (Chain)이전 블록 해시로 연결된 블록의 순서적 연결불변 연결 구조
    해시 (Hash)블록 데이터를 고정 길이 암호값으로 변환무결성 보장
    노드 (Node)블록체인 네트워크 참여·검증·저장 컴퓨터분산 참여자
    분산 원장 (DLT)모든 노드가 동일 원장 공유·저장중앙화 없는 기록
    스마트 컨트랙트조건 충족 시 자동 실행되는 프로그램 코드자동화 계약
  • 블록 구조:
    항목설명핵심
    블록 헤더이전 블록 해시·타임스탬프·논스·머클루트 포함블록 식별 정보
    이전 블록 해시앞 블록 해시값으로 체인 연결·위변조 방지체인 연결
    머클 트리 (Merkle Tree)트랜잭션 해시를 이진 트리로 요약·검증거래 무결성
    논스 (Nonce)PoW 채굴 시 목표 해시 찾기 위한 임의 숫자채굴 증명값
    트랜잭션 데이터실제 거래 내역·스마트 컨트랙트 실행 기록실제 거래 기록
  • 블록체인 유형:
    유형설명접근성대표 사례
    퍼블릭 블록체인누구나 참여·검증·열람 가능한 완전 공개형완전 개방비트코인·이더리움
    프라이빗 블록체인특정 조직이 운영·허가된 참여자만 접근폐쇄적Hyperledger Fabric
    컨소시엄 블록체인복수 기관 공동 운영·허가된 구성원 참여반개방R3 Corda·Quorum
    하이브리드 블록체인퍼블릭·프라이빗 혼합 구성선택적 개방Dragonchain
  • 블록체인 핵심 특성:
    특성설명핵심
    탈중앙화 (Decentralization)중앙 관리자 없이 참여자 전체가 관리중개자 제거
    불변성 (Immutability)한번 기록된 데이터 수정·삭제 불가데이터 영속성
    투명성 (Transparency)모든 거래 내역 공개·누구나 검증 가능완전 공개
    무결성 (Integrity)해시·암호화로 데이터 위변조 방지데이터 신뢰성
    가용성 (Availability)단일 장애점 없이 분산 노드로 지속 운영고가용성
  • 인사이트: 블록체인은 신뢰의 탈중앙화라는 혁신적 개념을 실현하지만, 트릴레마(탈중앙화·보안·확장성 동시 달성 불가)가 대중화의 핵심 기술 장벽으로 작용하고 있음. Layer 2·샤딩·PoS 전환 등으로 확장성이 빠르게 개선되고 있으며, DeFi·CBDC·공급망 분야에서 실용화가 가속되고 있음. 기술 성숙도보다 규제 명확화·UX 단순화·에너지 효율화가 블록체인 대중화의 실질적 선결 과제임
