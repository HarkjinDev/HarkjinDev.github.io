5. 프로세스 동기화 (Process Synchronization)

정의: 여러 프로세스/스레드가 공유 자원에 접근할 때 데이터 일관성을 보장하기 위한 제어 메커니즘
키워드: 임계영역, 상호배제, 세마포어, 모니터, 스핀락, Race Condition
분류: 프로세스 관리
임계영역 문제 해결 조건: 상호배제(Mutual Exclusion) / 진행(Progress) / 한정 대기(Bounded Waiting)
동기화 기법:
기법
수준
특징
세마포어 (Semaphore)
저수준
P/V 연산, 이진/카운팅 세마포어
뮤텍스 (Mutex)
저수준
이진 세마포어의 특수 형태, 소유권 개념 있음
스핀 락 (Spin Lock)
하드웨어 수준
바쁜 대기(Busy Wait), 단기 임계영역에 적합
모니터 (Monitor)
고수준
언어 수준 추상화, 상호배제 자동 보장
인사이트: 동기화 기법 선택 시 임계영역 진입 시간의 길이가 핵심 기준 — 짧은 임계영역은 스핀락, 긴 임계영역은 세마포어/뮤텍스가 효율적
