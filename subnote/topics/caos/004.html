[하위] 1-2. 엔디언 (Endianness)

  • 정의: 다중 바이트 데이터를 메모리에 저장할 때의 바이트 순서 방식
  • 키워드: MSB, LSB, 빅 엔디언, 리틀 엔디언, 네트워크 바이트 순서, x86 아키텍처
  • 분류: 컴퓨터 구조
    구분빅 엔디언 (Big Endian)리틀 엔디언 (Little Endian)
    정의최상위 바이트(MSB)를 낮은 주소에 먼저 저장최하위 바이트(LSB)를 낮은 주소에 먼저 저장
    예시 (0x12345678)주소 0: 0x12 / 1: 0x34 / 2: 0x56 / 3: 0x78주소 0: 0x78 / 1: 0x56 / 2: 0x34 / 3: 0x12
    사용 환경네트워크 프로토콜(TCP/IP), SPARC, PowerPCx86, x86-64 (Intel/AMD), ARM(기본)
    가독성메모리 덤프 시 직관적산술 연산 처리 용이
  • 인사이트: 네트워크 통신 시 서로 다른 엔디언 시스템 간 데이터 교환을 위해 htons()/ntohl() 같은 바이트 순서 변환 함수를 반드시 사용해야 하며, 네트워크 바이트 순서는 빅 엔디언이 표준임
