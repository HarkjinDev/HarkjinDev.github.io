[하위] 2-1. 멀티 GPU (Multi-GPU)

  • 정의: 다수의 GPU를 활용하여 대규모 신경망을 분산 병렬로 작동하기 위해 사용하는 기술
  • 키워드: 데이터 병렬화, 모델 병렬화, 텐서 병렬화, NVLink, PCIe, Infinity Fabric, PSU, 침지냉각
  • 분류: 컴퓨터 구조
  • 유형:
    전략설명적합 상황
    데이터 병렬화 동일 모델 복사본에 다른 데이터 배치 분배모델이 GPU 메모리에 수용 가능할 때
    모델 병렬화 모델을 여러 GPU에 분할 배치모델이 단일 GPU 메모리 초과 시
    텐서 병렬화 개별 텐서 연산을 GPU에 분할대형 LLM (GPT, LLaMA 등)
    파이프라인 병렬화 레이어 그룹을 GPU에 순서대로 배치매우 깊은 신경망
  • 멀티 GPU 구성 요소:
    구성요소설명
    NVLink / NVSwitchNVIDIA GPU 간 고대역폭 직접 연결. PCIe 대비 10배 이상 대역폭 제공
    Infinity FabricAMD GPU·CPU 간 고속 인터커넥트
    PCIe 스위치표준 인터페이스 기반 GPU 연결. 비용 저렴, 대역폭 상대적으로 낮음
    PSU (Power Supply Unit)다수 GPU의 전력 공급 관리. 고전력 소비 대비 충분한 용량 확보 필요
    냉각 시스템공랭(Air) / 수랭(Liquid) / 침지냉각(Immersion) 방식
  • 고려사항: GPU 간 통신 대역폭(Bandwidth), 동기화 오버헤드, 전원 공급(PSU), 냉각
  • 인사이트: LLM 학습에는 수백~수천 GPU가 필요하며, GPU 간 통신 대역폭이 학습 효율을 결정하는 핵심 병목이 됨. NVIDIA NVLink/NVSwitch 기반 DGX 시스템이 대형 모델 학습의 표준 플랫폼으로 자리잡음
