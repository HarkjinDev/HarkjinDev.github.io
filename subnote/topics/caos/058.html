1. TPU (Tensor Processing Unit)

  • 정의: 딥러닝 행렬 연산(텐서 연산)에 특화된 구글의 AI 전용 ASIC 프로세서
  • 키워드: 구글, 텐서 연산 특화, ASIC, MXU, Systolic Array, HBM, 추론/학습 가속
  • 분류: AI 연산 하드웨어
  • TPU 특화 구조:
    • MXU (Matrix Multiply Unit): 행렬 곱셈 전용 하드웨어 — Systolic Array 방식
    • VPU (Vector Processing Unit): 활성화 함수 등 벡터 연산
    • HBM: 고대역폭 메모리로 데이터 병목 최소화
  • 구성 요소:
  • TPU 사용 이유:
    • GPU 대비 딥러닝 연산에서 전력 효율 3~5배 높음
    • 대규모 클러스터(TPU Pod) 구성으로 수천 개 TPU 연동 가능
  • CPU vs GPU vs TPU:
    구분CPUGPUTPU
    설계 목적범용 직렬 처리그래픽 + 병렬 연산딥러닝 텐서 연산 특화
    코어 구조소수 복잡 코어수천 개 단순 코어행렬곱 가속 전용 유닛 (MXU)
    유연성최고높음낮음 (특정 연산에 최적화)
    전력 효율중간높음매우 높음 (TOPS/W)
    프로그래밍범용 언어CUDA/OpenCLTensorFlow/JAX (구글 생태계)
    제조사Intel, AMD, ARMNVIDIA, AMDGoogle (자체 개발)
  • 인사이트: TPU는 구글 독점 하드웨어로 Google Cloud에서만 사용 가능하며, 마이크로소프트-NVIDIA, AWS Inferentia/Trainium 등 자체 AI 칩 개발 경쟁이 촉발됨
