2. GPU(Graphics Processing Unit)

  • 정의: 수천개 소형코어로 부동소수점 기반의 대규모 행렬 병렬 연산을 수행하는 처리 장치
  • 키워드: 부동소수점연산(FLOPs), 병렬 연산, CUDA, 행렬 연산, 학습 가속, ASIC, 추론/학습, 전력 효율
  • 분류: 컴퓨터 구조
  • GPU 구성 요소:
    구분주요 요소설명
    연산 장치 (Processing)코어 (CUDA/Stream)수천 개의 ALU로 구성되어 실제 부동 소수점 연산을 병렬로 수행
    Tensor CoreAI 및 딥러닝 가속을 위한 행렬 연산(Matrix Multiplication) 특화 코어
    RT Core레이 트레이싱(Ray Tracing) 등 고사양 그래픽 렌더링 가속 코어
    제어 및 스케줄링스케줄러 (Scheduler)수만 개의 스레드를 관리하고 연산 장치에 작업을 배분(Warp Scheduler)
    디스패처 (Dispatcher)스케줄링된 명령어를 실제 실행 유닛으로 전달
    메모리 계층 (Memory)레지스터 (Register)각 스레드에 할당된 가장 빠른 기억 공간
    L1/L2 캐시데이터 재사용성 향상을 위한 내부 캐시 메모리
    비디오 메모리 (VRAM)GDDR6/GDDR7 또는 고대역폭 메모리인 HBM을 사용하여 대용량 데이터 저장
    인터페이스 (Bus)PCI Express (PCIe)호스트 CPU와 GPU 간의 데이터 전송 통로
    NVLink / CXLGPU 간 직접 통신을 통해 병목 현상을 제거하는 고속 인터페이스
  • 인사이트: GPU는 유연성과 생태계(CUDA)로 AI 표준 플랫폼이 되었으나, LLM 추론 서비스에서 비용 대비 성능을 위해 TPU나 자체 NPU를 도입하는 하이퍼스케일러들이 늘고 있음
