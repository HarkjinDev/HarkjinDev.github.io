6. UNIX 파일 시스템 (UNIX File System)

  • 정의: UNIX 운영체제에서 파일 저장 및 관리를 위한 계층적 디렉토리 구조와 i-node 기반의 파일 시스템
  • 키워드: 부트블록, 슈퍼블록, i-node, 데이터블록, 접근제어
  • 분류: 메모리 관리

    슈퍼 블록 (Super Block)파일 시스템 전체 정보 저장 (블록 크기·빈 블록 수·i-node 수)파일 시스템 메타

  • 인사이트: UNIX 파일 시스템은 모든 것을 파일로 추상화하는 철학(Everything is a file)이 특징이며, 하드 링크(i-node 공유)와 심볼릭 링크(경로 참조)의 차이를 시험에서 자주 출제함
