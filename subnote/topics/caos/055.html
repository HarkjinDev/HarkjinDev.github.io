2. 스토리지 계층 (블록 / 파일 / 오브젝트 스토리지)

  • 정의: 데이터를 저장하고 접근하는 방식에 따른 스토리지 유형 분류
  • 키워드: 블록, 파일, 오브젝트, LUN, 계층 구조, 키-값, REST API, S3
  • 분류: 스토리지
  • 비교:
    구분블록 스토리지파일 스토리지오브젝트 스토리지
    접근 단위고정 크기 블록파일 및 디렉토리오브젝트 (키-값)
    접근 방식LUN/볼륨 마운트파일 경로REST API / HTTP
    성능최고 (낮은 레이턴시)중간높은 처리량, 높은 레이턴시
    확장성제한적중간무제한 (수평 확장)
    적합 용도DB, VM 디스크파일 공유, NAS미디어, 백업, 빅데이터, 클라우드 스토리지
    대표 제품AWS EBS, SANNFS, AWS EFSAWS S3, Azure Blob
  • 인사이트: 클라우드 네이티브 환경에서 오브젝트 스토리지가 빠르게 성장하고 있으며, 이머지블(Immutable) 특성과 무제한 확장성으로 AI/ML 데이터 레이크의 표준 저장소로 자리잡음
