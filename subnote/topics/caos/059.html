4. CXL (Compute Express Link)

정의: CPU와 가속기(GPU/FPGA) 및 메모리 장치 간 고속·저지연·일관성 있는 연결을 제공하는 인터커넥트 표준
키워드: CXL.io, CXL.cache, CXL.mem, Type1/2/3, 메모리 일관성, PCIe 기반
분류: AI 연산 하드웨어
CXL 프로토콜:
프로토콜
역할
CXL.io
PCIe 기반 I/O 프로토콜 (초기화, 레지스터 접근)
CXL.cache
가속기가 CPU의 메모리(호스트 메모리)를 캐싱하는 프로토콜
CXL.mem
CPU가 가속기의 메모리에 직접 접근하는 프로토콜
CXL 타입:
유형
사용 프로토콜
예시
Type 1
CXL.io + CXL.cache
스마트 NIC, 암호화 가속기
Type 2
CXL.io + CXL.cache + CXL.mem
GPU, FPGA (양방향 메모리 공유)
Type 3
CXL.io + CXL.mem
메모리 확장 모듈 (DRAM/Persistent Memory)
등장 배경: AI/HPC 워크로드에서 CPU-가속기 간 메모리 공유의 비효율성 해결 (기존 PCIe는 일관성 없는 별도 메모리 공간)
인사이트: CXL은 메모리 분해(Memory Disaggregation)를 가능하게 하여 대규모 AI 클러스터에서 메모리 활용 효율을 극대화하는 핵심 기술로 부상하고 있으며, DDR6/HBM4와 함께 차세대 AI 서버 인프라의 표준이 될 전망
