[하위] 2-2. 다중연결리스트 (Multi-Linked List)

정의: 하나의 노드가 다수의 포인터를 갖는 연결 리스트 자료구조
키워드: 다중 포인터, 희소 행렬(Sparse Matrix), 행 헤더, 열 헤더, 다차원 접근
분류: 선형 자료구조
특징: 하나의 노드에서 여러 방향으로 연결이 가능하여 행·열 방향 모두 순회 가능
적용:
적용 분야
설명
희소 행렬(Sparse Matrix)
0이 아닌 원소만 노드로 저장하여 메모리 절약
다차원 연결 구조
행 헤더·열 헤더로 2차원 방향 임의 접근
스파스 텐서
딥러닝에서 희소 가중치 행렬 저장
인사이트: 다중연결리스트는 희소 행렬의 효율적 표현에 핵심적으로 사용되며, 전체 원소 대비 비영(非零) 원소 비율이 낮을수록 배열 대비 공간 절약 효과가 극대화됨을 언급
