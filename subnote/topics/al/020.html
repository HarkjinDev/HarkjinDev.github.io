3. 방향성 비순환 그래프 (Directed Acyclic Graph, DAG)

  • 정의: 정점을 방향 간선으로 연결하되 순환 경로가 존재하지 않는 그래프 자료구조
  • 키워드: 방향 그래프, 비순환, 위상정렬, 진입 차수(In-degree), 의존성 분석
  • 분류: 비선형 자료구조
  • 주요 특징:
    특징설명핵심
    비순환성임의 정점에서 출발해 자신으로 돌아오는 경로 없음사이클 부재
    위상정렬 가능DAG일 때만 위상정렬 수행 가능정렬 필요충분조건
    소스 존재진입차수 0인 정점 1개 이상 존재시작점 보장
    싱크 존재진출차수 0인 정점 1개 이상 존재종료점 보장
    부분순서 표현정점 간 선후관계만 정의·전체순서 아님의존성 표현
  • Kahn's Algorithm 절차:
    단계설명핵심
    1. 진입차수 계산모든 정점의 진입차수 산출초기화
    2. 큐 초기화진입차수 0인 정점 큐에 삽입시작점 수집
    3. 정점 제거큐에서 정점 꺼내 결과에 추가순서 확정
    4. 차수 감소인접 정점 진입차수 1 감소의존성 해소
    5. 반복차수 0 되면 큐 삽입, 큐 빌 때까지 반복종료조건
    6. 사이클 검사처리 정점 수 < V면 사이클 존재사이클 검출
  • 적용: 작업 스케줄링(PERT/CPM), 컴파일러 의존성 분석, 데이터 파이프라인, Git 커밋 그래프, Makefile
  • 인사이트: DAG는 위상정렬의 필요충분조건이며 Kahn's Algorithm으로 위상정렬과 사이클 검출을 동시에 수행 가능해, 작업 스케줄링·빌드 시스템·데이터 파이프라인 등 의존성 기반 시스템 설계의 핵심 자료구조로 활용됨
