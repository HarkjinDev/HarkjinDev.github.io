3. 방향성 비순환 그래프 (Directed Acyclic Graph, DAG)

  • 정의: 순환이 없는 방향 그래프로 작업 의존성 표현에 사용되는 자료 구조
  • 키워드: 방향 그래프, 비순환, 위상정렬, 진입 차수(In-degree), 의존성 분석
  • 분류: 비선형 자료구조
  • 주요 특징:
    특징설명
    비순환성임의 정점에서 출발하여 자신으로 돌아오는 경로 없음
    위상정렬 가능DAG인 경우에만 위상정렬 수행 가능
    소스(Source) 존재진입 차수 0인 정점이 반드시 하나 이상 존재
  • 적용: 작업 스케줄링(PERT/CPM), 컴파일러 의존성 분석, 데이터 파이프라인, Git 커밋 그래프, Makefile
  • 인사이트: DAG는 위상정렬의 필요충분조건이며, 진입 차수 0인 노드를 큐에 넣고 BFS로 처리하는 Kahn’s Algorithm이 위상정렬 구현의 핵심. 사이클 감지에도 활용 가능.
