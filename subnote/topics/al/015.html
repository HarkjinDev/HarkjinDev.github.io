[하위] 4-2. 우선순위 큐 (Priority Queue)

  • 정의: 우선순위가 높은 요소가 먼저 처리되는 추상 자료형
  • 키워드: 이진 힙(Binary Heap), 최대 힙, 최소 힙, heapify(Sift-Down), 다익스트라, 허프만 코딩
  • 분류: 선형 자료구조
  • heapify(힙화): 일반적인 배열이나 이진 트리를 힙(Heap)의 속성(부모가 자식보다 크거나 작은 조건)을 만족하도록 재구조화하는 핵심 연산
  • 구현 방식 비교:
    구현 방식삽입최우선 삭제특징
    정렬 배열O(n)O(1)삽입 시 정렬 유지 필요
    비정렬 배열O(1)O(n)삭제 시 전체 탐색 필요
    이진 힙O(log n)O(log n)최적 균형, 표준 구현
    피보나치 힙O(1) 분할상환O(log n) 분할상환다익스트라 최적화
  • 적용: 다익스트라 최단 경로, 허프만 인코딩, OS 프로세스 스케줄링, 이벤트 시뮬레이션
  • 인사이트: 우선순위 큐의 표준 구현은 이진 힙이며, 피보나치 힙 적용 시 다익스트라 알고리즘의 전체 복잡도가 O(V² + E)에서 O(V log V + E)로 개선됨을 서술하면 고득점 포인트
