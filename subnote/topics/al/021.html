[하위] 3-1. 위상정렬 (Topological Sort)

  • 정의: DAG에서 선행 관계를 유지하며 노드를 선형 순서로 나열하는 알고리즘
  • 키워드: DAG, 진입 차수, Kahn’s Algorithm, DFS 후위 순회, 선행 제약(Precedence), 비유일성
  • 분류: 비선형 자료구조
  • 알고리즘 비교:
    방법절차복잡도
    Kahn’s (BFS 기반, 직관적)진입 차수 0 노드 큐 삽입 → Dequeue → 인접 노드 차수 감소 → 반복O(V + E)
    DFS 기반(역방향 출력)모든 노드 DFS → 완료 시 스택 push → 역순 출력O(V + E)
  • 특징:
    • 위상정렬 결과는 유일하지 않을 수 있음 (진입 차수 0 노드가 복수인 경우)
    • 사이클 존재 시 위상정렬 불가능: Kahn’s에서 전체 노드가 처리되지 않으면 사이클 감지
  • 인사이트: 위상정렬은 사이클 감지 알고리즘으로도 활용되며, Kahn’s는 BFS 기반으로 직관적이고, DFS 기반은 역방향 출력이라는 차이를 명확히 서술. 결과 비유일성도 반드시 언급
