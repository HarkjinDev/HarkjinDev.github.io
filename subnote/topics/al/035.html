6. 트리정렬 (Tree Sort)

  • 정의: 이진 탐색 트리에 삽입 후 중위 순회로 정렬된 결과를 얻는 알고리즘
  • 키워드: BST, 중위 순회(In-order Traversal), O(n log n) 평균, 동적 정렬
  • 분류: 정렬 알고리즘
  • 특징: 안정 정렬, 동적으로 요소를 추가하면서 정렬 상태 유지 가능
  • 메커니즘:
  • 복잡도:
    케이스시간 복잡도조건
    평균O(n log n)균형 BST 구성 시
    최악O(n²)편향 BST (정렬된 입력 삽입 시)
    공간O(n)트리 노드 저장
  • 인사이트: 트리정렬은 동적으로 요소를 추가하면서 정렬 상태를 유지해야 할 때 유용하며, 편향 트리 문제는 AVL 또는 레드-블랙 트리 적용으로 O(n log n) 보장 가능. 힙 정렬과 달리 트리 노드별 메모리가 필요하다는 단점도 언급
