2. 리스트 (List)

  • 정의: 포인터로 노드를 연결하여 동적으로 크기가 변하는 선형 자료 구조
  • 키워드: 노드(Node), 포인터, 단순 연결, 이중 연결, 원형 연결, 동적 메모리 할당
  • 분류: 선형 자료구조
  • 유형 비교:
    유형포인터 구성특징
    단순 연결 리스트next 포인터단방향 순회, 구현 단순
    이중 연결 리스트prev + next 포인터양방향 순회, 삭제 O(1)
    원형 연결 리스트마지막→첫 노드 연결순환 탐색, 큐·덱 구현
  • 배열 vs 연결 리스트:
    구분배열연결 리스트
    탐색O(1) (인덱스)O(n) (순차)
    삽입·삭제O(n) (이동 필요)O(1) (위치 알 때)
    메모리연속 할당동적 분산 할당
  • 인사이트: 연결 리스트는 삽입·삭제 O(1)이나 탐색 O(n)으로 배열과 상호 보완적 관계이며, 데이터 변경이 잦은 경우 연결 리스트, 탐색이 잦은 경우 배열을 선택함을 서술
