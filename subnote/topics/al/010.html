[하위] 2-1. 연결 리스트 (Linked List)

정의: 노드와 포인터로 연결되어 동적 삽입·삭제가 가능한 선형 자료구조
키워드: 노드, Head, next 포인터, 동적 메모리, ordered_insert, 정렬 삽입
분류: 선형 자료구조
ordered_insert 절차 (정렬 삽입):
단계
설명
1. 위치 탐색
Head부터 순회하며 새 값이 현재 노드보다 작은 위치 탐색
2. 이전 노드 저장
삽입 위치 바로 앞 노드의 포인터 저장
3. 새 노드 생성
새 노드의 next를 현재 노드에 연결
4. 포인터 갱신
이전 노드의 next를 새 노드로 갱신
인사이트: ordered_insert는 정렬 상태를 유지하며 삽입하는 알고리즘으로 O(n)이며, 이중 연결 리스트로 구현 시 이전 노드 탐색 없이 삭제 O(1) 보장. 대용량 삽입이 잦을 경우 BST나 스킵 리스트(Skip List)로 O(log n) 개선 가능
