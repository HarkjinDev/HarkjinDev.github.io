[하위] 3-1. 다익스트라 (Dijkstra's Algorithm)

  • 정의: 하나의 출발 정점에서 가중치가 양수인 그래프의 모든 정점까지 최단 경로를 탐욕적으로 탐색하는 최단 경로 알고리즘
  • 키워드: 최단 경로, 양수 가중치, 우선순위 큐(Min-Heap), 탐욕 알고리즘, 거리 테이블, 완화(Relaxation), 단일 출발점
  • 분류: 그래프 탐색
  • 특성: 우선순위 큐(Min-Heap), 시간복잡도: O((V + E) log V), 양수 가중치 조건, 단일 출발
  • 동작 원리:
    단계설명핵심
    1. 초기화출발 정점 거리 0, 나머지 정점 거리 ∞ 설정거리 초기화
    2. 정점 선택미방문 정점 중 현재 거리가 최소인 정점 선택탐욕적 선택
    3. 완화 (Relaxation)선택 정점 경유 시 인접 정점 거리가 단축되면 갱신최단 거리 갱신
    4. 방문 처리선택 정점을 방문 완료로 처리 후 반복확정 처리
    5. 종료모든 정점 방문 완료 시 종료종료 조건
  • 활용 분야: 내비게이션(출발지~목적지 최단 경로 탐색), 네트워크 라우팅(OSPF 프로토콜)
  • 인사이트: 다익스트라는 BFS의 가중치 확장 버전으로, 탐욕적 선택의 정당성이 양수 가중치 조건 하에서 보장됨. 음수 가중치 처리 불가라는 한계를 벨만-포드와의 비교로 명확히 이해하는 것이 시험 핵심 포인트이며, 실무에서는 OSPF·내비게이션 등 핵심 라우팅 알고리즘의 기반으로 광범위하게 활용됨
