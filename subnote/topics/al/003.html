[하위] 1-1. O-Notation (빅오 표기법)

정의: 알고리즘의 최악 시간 복잡도를 점근적 상한으로 표기하는 수학적 표기법
키워드: 점근적 상한, 최고차항, 상수 계수 제거, g(n), c·g(n), n₀
분류: 알고리즘 기초
수학적 정의: f(n) = O(g(n))이면, 모든 n ≥ n₀에 대해 f(n) ≤ c·g(n)을 만족하는 양의 상수 c와 n₀가 존재함
계산 규칙:
규칙
내용
최고차항 선택
낮은 차수 항과 상수 계수 제거 (예: 3n² + 2n → O(n²))
덧셈 규칙
O(f + g) = O(max(f, g))
곱셈 규칙
O(f · g) = O(f) · O(g)
