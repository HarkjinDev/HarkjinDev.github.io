[하위] 1-1. O-Notation (빅오 표기법)

  • 정의: 알고리즘의 최악 시간 복잡도를 점근적 상한으로 표기하는 수학적 표기법
  • 키워드: 점근적 상한, 최고차항, 상수 계수 제거, g(n), c·g(n), n₀
  • 분류: 알고리즘 기초
  • 수학적 정의: f(n) = O(g(n))이면, 모든 n ≥ n₀에 대해 f(n) ≤ c·g(n)을 만족하는 양의 상수 c와 n₀가 존재함
  • 계산 규칙:
    규칙내용
    최고차항 선택낮은 차수 항과 상수 계수 제거 (예: 3n² + 2n → O(n²))
    덧셈 규칙O(f + g) = O(max(f, g))
    곱셈 규칙O(f · g) = O(f) · O(g)
  • 연산시간 차이 (빠른 순):
    연산시간복잡도이름대표 예시
    빠름O(1)상수배열 인덱스 접근
    ↑O(log n)로그이진 탐색
    ↑O(n)선형선형 탐색
    ↑O(n log n)선형로그병합 정렬, 퀵 정렬 평균
    ↑O(n²)이차버블/삽입/선택 정렬
    느림O(2ⁿ)지수부분집합 나열
  • 인사이트: 상수 계수와 하위 차수 항을 무시하고 최고차항만 표기하며, Big-O(상한), Big-Ω(하한), Big-Θ(정확한 경계)의 개념적 차이와 함께 실제 알고리즘에 적용하여 서술하면 고득점 포인트
