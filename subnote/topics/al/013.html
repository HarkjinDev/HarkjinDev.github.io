4. 큐 (Queue)

정의: 선입선출(FIFO) 원칙으로 한쪽에서 삽입, 반대쪽에서 삭제하는 자료 구조
키워드: FIFO, Enqueue, Dequeue, Front, Rear, 원형큐, 우선순위 큐, BFS, 앞 공간 낭비
분류: 선형 자료구조
주요 연산:
연산
설명
시간 복잡도
Enqueue
Rear에 데이터 삽입
O(1)
Dequeue
Front에서 데이터 삭제
O(1)
Peek/Front
Front 데이터 조회 (삭제 없음)
O(1)
isEmpty
공백 여부 확인
O(1)
적용: 프로세스 스케줄링, BFS 탐색, 프린터 스풀링, 네트워크 패킷 처리
인사이트: 일반 배열 기반 큐는 Dequeue 후 앞 공간 낭비 문제가 발생하므로 원형큐(Circular Queue)로 해결하고, 우선순위 큐는 이진 힙으로 구현해 O(log n) 연산을 달성함을 언급
