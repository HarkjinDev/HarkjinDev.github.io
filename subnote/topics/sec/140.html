7. PET (Privacy Enhancing Technology)

  • 정의: 개인정보 보호를 기술적으로 강화하는 프라이버시 보호 강화 기술의 총칭
  • 키워드: 차분 프라이버시, 합성 데이터, 영지식 증명, 동형암호화, 연합학습
  • 분류: 개인정보보호
  • PET 기술 분류: <동차합영/연다신>
    기술설명활용
    동형암호(HE)암호화 상태에서 연산클라우드 안전 처리
    차분 프라이버시(DP)통계에 노이즈 추가집계 데이터 공개
    합성데이터원본 특성 모방 가상 데이터AI 학습 데이터
    영지식 증명(ZKP)정보 공개 없이 사실 증명블록체인, 인증
    연합학습(FL)데이터 이동 없이 분산 학습의료·금융 AI
    다자간 연산(MPC)각자 데이터 비공개 연산공동 분석
    신뢰실행환경(TEE)하드웨어 격리 실행안전한 데이터 처리
  • 인사이트: PET는 데이터 활용과 프라이버시 보호의 딜레마를 기술적으로 해결하는 수단으로, 의료·금융 분야의 데이터 공유 활성화의 핵심 기술임
