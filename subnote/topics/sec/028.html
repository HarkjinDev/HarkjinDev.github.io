16. 합성데이터 (Synthetic Data)

  • 정의: 실제 데이터의 통계적 특성을 모방하여 AI로 생성한 가상 데이터
  • 키워드: 데이터 증강, Model Collapse, 품질 검증, RLHF 학습 데이터, GAN, 데이터 프라이버시, 분포 보존, 편향 증폭, 유용성 검증, 안전성 검증
  • 분류: 암호학
  • 유형:
    구분특징내용
    완전 합성데이터 가상 생성 데이터실제 데이터 없이 모두 가상 생성
    부분 합성데이터일부 변수만을 대체일부 속성만 선택하여 대체
    민감/기밀정보 대체민감 정보만 선택적으로 대체
    복합 합성 데이터 변수 대체 후 추가 변수 도출합성 및 실제 데이터 혼합 생성
  • 주요 생성 방식:
    항목설명핵심
    GAN 기반생성자·판별자 경쟁 학습으로 실제 분포와 유사한 데이터 생성분포 모방
    VAE 기반잠재 공간 샘플링으로 다양한 변형 데이터 생성변형 생성
    LLM 기반대형 언어 모델로 텍스트·대화·지시문 형태의 합성 데이터 생성텍스트 합성
    규칙 기반도메인 규칙·통계 모델로 정형 데이터 생성규칙 생성
  • 주요 활용:
    항목설명핵심
    데이터 증강학습 데이터 부족 문제를 합성 데이터로 보완데이터 확장
    프라이버시 보호개인정보 포함 실데이터 대신 합성 데이터로 대체민감 정보 대체
    RLHF 학습 데이터인간 선호도 데이터 부족 시 합성 피드백 데이터 활용정렬 학습 보완
    희귀 케이스 생성실데이터에서 수집 어려운 엣지 케이스·이상치 생성희귀 사례 확보
    도메인 특화의료·금융 등 민감 도메인의 학습 데이터 확보규제 도메인 대응
  • 유용성 검증방안
    구분검증지표설명
    비정형 합성데이터모델 성능 검증원본·합성 모델 성능 차이 임계값 비교
    Visual Turing Test (VTT)무작위 샘플 중 원본 구분 정답률 측정
    이미지 품질 검증원본·합성 이미지 데이터 분포 거리 비교
    정형 합성데이터2차원 관계 유사성컬럼 간 상관계수 행렬 차이 표준편차 계산
    구별 불가능성ML 모델로 원본·합성 구분 확률 유사성 계산
  • 안전성 검증방안
    구분검증지표설명
    비정형 합성데이터생성과정비가역 알고리즘 사용·메타데이터 처리 확인
    구조적 유사성원본·합성 이미지 구조적 과도 유사 측정
    지각적 유사성원본·합성 이미지 지각적 과도 유사 측정
    주관적 검증전문가·관계자 직접 정성 평가 검수
    식별위험도합성 레코드의 원본 내 동일 존재 확률 측정
    정형 합성데이터연결위험도합성 데이터로 원본 개인 민감정보 유추 위험
    추론위험도합성 데이터로 개인 역추론 가능 위험 측정
  • 인사이트: 합성 데이터는 데이터 부족·프라이버시 규제 문제를 해결하는 실용적 대안으로 부상하고 있으나, Model Collapse가 보여주듯 실제 데이터와의 혼합 없이 순수 합성 데이터만으로 학습을 반복하면 모델 품질이 점진적으로 저하되므로 품질 검증과 실데이터 비율 유지가 합성 데이터 전략의 핵심 설계 요소임
