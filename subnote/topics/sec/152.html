17. 잊힐 권리 (Right to be Forgotten)

  • 정의: 정보주체가 자신에 관한 개인정보의 삭제·처리 중단을 요청할 수 있는 권리로, 디지털 환경에서 과거 정보로 인한 피해를 방지하기 위한 개인정보 자기결정권의 핵심 권리
  • 키워드: 개인정보 자기결정권, GDPR 제17조, 삭제권, Machine Unlearning, 잊힐 권리, 표현의 자유, 공익, AI 학습데이터
  • 분류: 개인정보보호
  • 등장 배경:
    구분내용핵심
    판례2014년 EU 사법재판소 Google Spain 판결잊힐 권리 최초 인정
    법제화GDPR 제17조 삭제권으로 명문화 (2018)EU 법적 근거 확립
    디지털 환경인터넷 정보의 영구성·검색 용이성으로 피해 증가잊혀지지 않는 정보
    AI 확산AI 학습 데이터 내 개인정보 삭제 불가 문제 부각Machine Unlearning 필요
  • 법적 근거:
    국가·지역법령조항핵심
    EUGDPR제17조 삭제권 (Right to Erasure)가장 포괄적 규정
    한국개인정보 보호법제36조 정정·삭제 요구권국내 법적 근거
    한국정보통신망법제44조의2 임시조치게시물 삭제 요청
    미국CCPA (캘리포니아)제1798.105조 삭제권주법 수준 보장
  • 잊힐 권리 vs 표현의 자유 충돌:
    항목잊힐 권리표현의 자유
    보호 가치개인 프라이버시·인격권알 권리·정보 접근권
    충돌 상황과거 범죄·실수 정보 삭제 요청공인·공익 정보 공개 필요
    판단 기준정보의 민감성·시간 경과공익성·정보 현재 관련성
    해결 방식이익 형량(Balancing Test)사안별 판단
    판례Google Spain 사건공인은 제한적 적용
  • AI 시대 잊힐 권리 이슈:
    이슈설명핵심
    파라미터 암기LLM 가중치에 개인정보 고착·선택적 삭제 불가Machine Unlearning 필요
    Machine Unlearning특정 데이터 영향을 모델에서 제거하는 기술기술적 구현 어려움
    RAG·Vector DB검색 증강 생성 시 개인정보 포함 벡터 삭제벡터 단위 삭제 필요
    생성형 AI 출력AI가 삭제 요청된 개인정보를 재생성 가능완전 삭제 불가 문제
    연합학습분산 학습 환경에서 특정 데이터 영향 제거 난이도기술적 한계
  • 인사이트: 잊힐 권리는 디지털 영구성에 대응하는 핵심 개인정보 권리로, AI 시대에는 LLM 파라미터 암기·RAG 벡터 저장으로 기술적 완전 삭제가 어려워 Machine Unlearning 기술 발전과 법·기술 간 간극 해소가 핵심 과제로 부상함
