80. CDR (Content Disarm & Reconstruction)

  • 정의: 파일 내 잠재적 위협 요소를 제거하고 안전한 형태로 재구성하는 보안 기술
  • 키워드: 파일 무해화, 악성코드 제거, 원본 재구성
  • 분류: 특수 보안
  • 동작 원리: 파일 수신 → 구조 분해(파싱) → 안전한 요소만 추출 → 재조립(무해화 파일 생성)
  • 인사이트: CDR은 알려지지 않은 제로데이 악성코드도 구조적으로 제거할 수 있어 시그니처 기반 백신과 상호 보완적이며, 금융·공공 기관의 외부 파일 수신 보안에 핵심임
