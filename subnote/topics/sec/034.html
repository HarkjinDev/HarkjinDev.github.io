3. TPM (Trusted Platform Module)

정의: 암호키를 안전하게 생성·저장하고 플랫폼 부팅 무결성을 하드웨어 수준에서 검증하는 보안 칩
키워드: 보안 칩, PCR, 키 저장, 부팅 무결성, BitLocker, 원격 증명, 봉인(Sealing), HSM, TPM 2.0
분류: 인증/신원관리

TPM 2.0 vs 1.2: TPM 2.0은 다양한 알고리즘 지원, 계층적 키 관리, Windows 11 필수 요건
활용 분야: 디스크 암호화(BitLocker·FileVault TPM 기반 키 보호), Zero Trust, 기밀 컴퓨팅(TEE 환경의 원격 증명), IoT 보안
인사이트: TPM은 소프트웨어 기반 키 관리의 구조적 취약점을 하드웨어로 해결하는 엔드포인트 보안의 핵심 신뢰 앵커(Trust Anchor)로, Zero Trust 아키텍처에서 디바이스 신뢰 검증의 기반이 되며 기밀 컴퓨팅·원격 증명과 결합하여 클라우드·IoT 환경의 하드웨어 신뢰 체계의 핵심 구성 요소로 역할이 확대되고 있음
