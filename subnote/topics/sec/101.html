74. IoT 보안 (IoT Security)

  • 정의: IoT 기기의 생애주기 전반에 걸친 보안 위협 대응 체계
  • 키워드: 경량 암호, 펌웨어 보안, 7대 원칙
  • 분류: 특수 보안
  • 생애주기별 보안 위협과 해결책:
    단계위협대응 방안
    설계기본 비밀번호·취약한 암호화보안 설계 원칙 적용, 강력한 암호화 설계
    배포물리적 탬퍼링·펌웨어 변조보안 부팅, 코드 서명
    운영패치 미적용·원격 공격자동 업데이트, 모니터링
    폐기데이터 잔존초기화 후 폐기, 데이터 삭제 확인
  • IoT 7대 보안 원칙: 기본 비밀번호 금지, 취약점 신고 정책, 소프트웨어 업데이트, 자격증명 안전 보관, 통신 암호화, 공격 표면 최소화, 무결성 보장
  • 인사이트: IoT 기기의 긴 사용 수명(10년 이상)으로 인해 제조사 지원 종료 후에도 취약한 기기가 운영되는 문제가 있으므로, 수명주기 동안의 보안 지원 의무화가 필요함
