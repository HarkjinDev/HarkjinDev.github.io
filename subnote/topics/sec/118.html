1. 적대적 공격 (Adversarial Attack)

  • 정의: 입력 데이터에 인간이 인지하기 어려운 미세한 교란을 가하여 AI 모델의 오분류를 유도하는 공격 기법
  • 키워드: 적대적 예제, 오분류 유도, 회피 공격, 데이터 오염
  • 분류: AI 보안
  • 학습 과정 기반 적대적 공격 4가지:
    • 데이터 오염 공격(Poisoning Attack): 학습 데이터에 악성 샘플 삽입 → 모델 성능 저하
    • 회피 공격(Evasion Attack): 추론 시 입력 교란으로 오분류 유도
    • 추출 공격(Extraction Attack): 모델의 예측값으로 내부 모델 역설계
    • 추론 공격(Inference Attack): 모델 출력으로 학습 데이터 프라이버시 추론
  • 인사이트: 자율주행·의료 AI에서 적대적 공격은 인명 피해로 직결될 수 있어, AI 시스템 인증에 적대적 견고성 검증이 필수 요건으로 논의되고 있음
