4. 파일 카빙 (File Carving)

  • 정의: 파일 시스템 메타데이터(FAT·MFT) 없이 데이터 패턴·시그니처로 삭제·손상된 파일을 복원하는 기법
  • 키워드: Header·Footer 시그니처, 파일 구조 검증, 단편화, 포맷 복원, 슬랙 카빙, MIME 타입
  • 분류: 디지털 포렌식
  • 4가지 카빙 기법:
    항목설명핵심
    Header/Footer 카빙파일 시작(헤더)·끝(푸터) 시그니처로 파일 범위 특정 후 복원시그니처 기반
    Header/Ram Slack 카빙헤더 + 램 슬랙 경계로 파일 크기 추정하여 복원슬랙 경계 활용
    Header/File Size 카빙헤더 내 내장된 파일 크기 정보를 활용하여 복원크기 정보 활용
    파일 구조 검증ASCII·MIME 타입·내부 구조 검증으로 파일 유효성 확인구조 기반 검증
  • 주요 기술:
    구분구성요소핵심 기술
    시그니처 탐지파일 헤더 DBJPEG(FFD8FF)·PDF(%PDF)·ZIP(PK) 등 파일 유형별 헤더 시그니처 매핑
    Magic Number파일 유형 식별을 위한 고유 바이트 패턴 탐지
    복원 처리연속 카빙단편화 없는 연속 저장 파일 헤더~푸터 직접 복원
    단편화 카빙비연속 저장 파일의 파편을 재조합하여 복원
    검증MIME 검증복원된 파일의 실제 유형·무결성 검증
  • 한계: 단편화 복원 한계, 덮어쓰기 복원 불가, SSD 복원 한계
  • 인사이트: 파일 카빙은 파일 시스템이 손상·포맷된 환경에서도 데이터 복원이 가능한 강력한 포렌식 기법이나, SSD·암호화·단편화가 복원 성공률을 크게 낮추는 변수로 작용하므로 시그니처 기반 카빙과 파일 구조 검증을 병행하여 복원 데이터의 유효성을 반드시 확인해야 함
