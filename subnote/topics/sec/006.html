[하위] 2-1. 접근제어 모델 (Access Control Model)

정의: 접근 권한 부여 원칙을 수학적으로 정의한 보안 모델
키워드: BLP, BIBA, Clark-Wilson, 만리장성
분류: 접근제어
주요 모델 비교:
모델
보호 목표
핵심 규칙
특성
BLP (Bell-LaPadula)
기밀성
No Read Up(NRU), No Write Down(NWD)
등급(class(s)), 객체(class(O)), 단순 보안 속성(주체는 자신보다 높은 등급의 객체를 읽을 수 없음), 스타 보안 속성(주체는 자신보다 낮은 등급의 객체에 정보를 쓸 수 없음)
BIBA
무결성
No Read Down(NRD), No Write Up(NWU)
단순 무결성 속성(주체는 자신보다 낮은 등급의 객체를 읽을 수 없음) 성형 무결성 속성(주체는 자신보다 높은 등급의 객체에 정보를 쓸 수 없음)
Clark-Wilson
무결성 (상업용)
직무분리, 변조방지
삼중항(사용자, 프로그램-TP, 데이터-CDI) 관계, CDI(제한된 데이터), UDI(제한되지 않은 데이터), IVP(무결성 검증 절차), TP(변조 절차)
Chiness Wall Mode)
이해충돌 방지
이해 출돌의 회피, 직무 분리, 내부 규칙 설계
접근 권한, 출동 집합, 액세스 제어 규칙, 안전성 검사
인사이트: BLP는 기밀성, BIBA는 무결성에 집중하며 서로 상충할 수 있음. 상업 환경에서는 Clark-Wilson이 적합하고, 군사 환경에서는 BLP가 사용됨. 만리장성은 금융권(투자, 금융 등)에서 활용.
