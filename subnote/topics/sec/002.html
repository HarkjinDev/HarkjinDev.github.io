0. 정보보안 개요 (Information Security Overview)

정의: 정보의 기밀성·무결성·가용성을 보존하고, 인증·책임성·부인방지·신뢰성과 관계를 가지는 것
키워드: CIA Triad, 보안 3원칙, 보안 서비스, 보안 3A
분류: 정보보안 개요
보안 vs 보호 비교:
보안 (Security): 가치있는 유무형 자산의 도난·손실·유출로부터 보호하는 것
보호 (Protection): 보안보다 광의적 의미로 사용, 전체 시스템의 안정성을 확보하는 것
보안 핵심 프레임워크 비교 (CIA Triad / 보안 요구사항 / 보안 3A):
