78. 클라우드 보안 (Cloud Security)

  • 정의: 클라우드 환경에서 데이터와 서비스를 보호하기 위한 보안 활동
  • 키워드: 데이터 보호, 접근제어, 가상화 보안, 공유 책임
  • 분류: 특수 보안
  • 공유 책임 모델 (Shared Responsibility):
영역IaaSPaaSSaaS
데이터고객고객고객
애플리케이션고객고객CSP
런타임고객CSPCSP
인프라CSPCSPCSP
  • 도입 시 보안 요소: 데이터 암호화, 접근제어(IAM), 가상화 보안, 로그 관리, 규정 준수
  • 인사이트: 클라우드 보안 사고의 대부분은 CSP 인프라가 아닌 고객의 잘못된 설정(Misconfiguration)에서 발생하므로, CSPM(Cloud Security Posture Management) 도입이 필수임
