6. 클라우드 포렌식 (Cloud Forensic)

  • 정의: 클라우드 환경에서의 디지털 포렌식 수사 기법
  • 키워드: 클라우드 시그니처, 증거 수집 절차, CSP 협력 필수, 국제 데이터 이전 규정
  • 분류: 디지털 포렌식
  • 도전 과제: 데이터 위치 불명확(다중 관할), 멀티 테넌시 환경, CSP 협력 필요, 증거 휘발성 높음
  • 조사 절차: CSP 협력 요청 → API 기반 로그 수집 → 타임스탬프 검증 → 가상 이미지 확보 → 분석
  • 인사이트: 클라우드 포렌식은 CSP와의 협력이 필수이며, 국제 데이터 이전 규정(GDPR, 개인정보보호법)이 증거 수집을 복잡하게 만드는 요인임
