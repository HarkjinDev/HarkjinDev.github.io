5. 안티포렌식 (Anti-Forensics)

  • 정의: 디지털 포렌식 수사를 방해하기 위해 증거를 인멸·은닉·변조하는 기술 및 행위
  • 키워드: Clear·Purge, 디가우징, 와이핑, 스테가노그래피, ADS, 타임스탬프 변조, 암호화, 로그 삭제
  • 분류: 디지털 포렌식
  • 데이터 삭제 수준 분류:
    항목설명핵심
    Clear일반적인 삭제·포맷, 복원 도구로 복구 가능복원 가능
    Purge덮어쓰기·디가우징으로 복원 불가 수준 삭제복원 불가
    Destroy물리적 파괴·소각으로 완전 폐기물리적 파괴
  • 기술 분류:
    구분항목핵심
    데이터 삭제덮어쓰기 (Wiping)복원 불가 삭제
    디가우징 (Degaussing)자기 소거
    물리적 파괴물리적 폐기
    SSD TRIM 활용SSD 자동 소거
    데이터 은닉스테가노그래피파일 내 은닉
    대체 데이터 스트림 (ADS)NTFS 스트림 은닉
    슬랙 영역 은닉미사용 영역 활용
    데이터 변조타임스탬프 변조시간 정보 조작
    메타데이터 삭제메타정보 제거
    로그 삭제·변조행위 기록 삭제
    암호화전체 디스크 암호화내용 접근 차단
    볼륨 은닉존재 자체 은닉
  • 안티포렌식 탐지 기법:
    항목설명핵심
    와이핑 도구 흔적Eraser·BCWipe 등 삭제 도구 실행 흔적 탐지도구 사용 증거
    타임스탬프 불일치파일 생성·수정·MFT 시간 불일치로 변조 탐지시간 불일치 탐지
    ADS 탐지NTFS 숨겨진 스트림 스캔으로 은닉 파일 탐지스트림 스캔
    스테가노그래피 탐지통계 분석·파일 크기 이상으로 은닉 데이터 탐지통계적 탐지
    로그 공백 탐지로그 삭제·공백 구간으로 행위 은닉 시도 탐지공백 구간 분석
  • 인사이트: 안티포렌식의 존재 자체(와이핑 도구 흔적·타임스탬프 불일치·ADS 발견)가 역설적으로 범죄 의도의 증거로 활용될 수 있으며, 완벽한 증거 은닉은 사실상 불가능하므로 다중 아티팩트 교차 분석과 안티포렌식 탐지 기법의 병행이 현대 디지털 포렌식의 핵심 접근법임
