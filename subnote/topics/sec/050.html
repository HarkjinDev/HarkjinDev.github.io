10. EDR (Endpoint Detection and Response, 엔드포인트 탐지 및 대응)

  • 정의: 엔드포인트(PC·서버·모바일 등)에서 발생하는 악성 행위를 실시간으로 수집·분석·탐지하고 자동으로 대응·복구하는 차세대 엔드포인트 보안 솔루션
  • 키워드: 행위 기반 탐지, IOC(침해지표), IOA(공격지표), 위협 헌팅, MITRE ATT&CK, 포렌식, 격리, XDR 연계, 제로데이 탐지, 자동 대응, 전체 행위
  • 분류: 네트워크 보안
  • 구성/기술요소:
    구분구성요소핵심 기술
    데이터 수집에이전트경량 에이전트 설치, 프로세스·파일·네트워크·레지스트리 행위 수집
    이벤트 로깅시스템 이벤트·API 호출·커널 레벨 활동 실시간 기록
    위협 탐지행위 기반 탐지ML 이상 탐지, IOA 룰 엔진, 베이스라인 편차 분석
    IOC 매칭알려진 악성 해시·IP·도메인 시그니처 비교 탐지
    메모리 분석파일리스 악성코드·인젝션·셸코드 탐지
    MITRE ATT&CK 매핑탐지 이벤트를 ATT&CK 전술·기법으로 자동 분류
    대응·복구자동 격리감염 엔드포인트 네트워크 차단·프로세스 종료
    롤백·복구랜섬웨어 피해 파일 복원, 감염 전 상태 복구
    조사·분석위협 헌팅IOC·IOA 기반 잠재 위협 능동 탐색
    포렌식·타임라인공격 경로·침해 원인 재구성, 증거 보존
  • 탐지·대응 프로세스: <수탐조대복>
    단계설명핵심
    수집에이전트가 엔드포인트 행위 데이터 실시간 수집·전송데이터 수집
    탐지ML·룰·IOC·IOA 기반으로 위협 식별 및 알림위협 식별
    조사공격 경로·영향 범위·침해 타임라인 분석원인 분석
    대응격리·차단·프로세스 종료 자동 또는 수동 실행피해 차단
    복구롤백·파일 복원·정상화 후 재발 방지 조치정상 복구
  • 기존 AV/EPP와의 비교:
    항목전통 AV/EPPEDR
    탐지 방식시그니처 기반행위·이상 기반
    제로데이 대응❌ 취약✅ 가능
    가시성파일 탐지 수준전체 행위 기록
    대응 방식수동자동 격리·대응
    위협 헌팅❌ 불가✅ 가능
  • 인사이트: EDR은 시그니처 기반 AV의 한계를 행위 분석·AI 탐지로 극복한 현재 엔드포인트 보안의 사실상 표준으로, 단독 운용을 넘어 네트워크·클라우드·이메일 보안을 통합하는 XDR로 확장되면서 SOC의 위협 탐지·대응 역량의 핵심 인프라로 자리잡고 있음. 또한, 국정원 국가보안관제 체계에서 EDR을 통한 보안관제체계(EMS) 도입 추진 중.
