2. 네트워크 스캐닝 (Network Scanning)

  • 정의: 네트워크의 호스트, 포트, 취약점을 탐색하는 정보 수집 기법
  • 키워드: 포트 스캔, 취약점 탐지, Nmap, 정보수집
  • 분류: 네트워크 보안
  • 스캔 유형:
    • 포트 스캔: TCP SYN 스캔, TCP Connect 스캔, UDP 스캔
    • 취약점 스캔: Nessus, OpenVAS
    • OS 핑거프린팅: TTL, TCP 윈도우 크기로 OS 추정
  • 인사이트: 네트워크 스캐닝은 공격자와 방어자 모두 활용하는 양날의 검으로, 정기적인 자체 스캔을 통한 노출 자산 파악이 보안 관리의 기본임
