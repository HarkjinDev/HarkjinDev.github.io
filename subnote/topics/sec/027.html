15. 차분 프라이버시 (Differential Privacy)

  • 정의: 통계 질의 결과에 수학적 노이즈를 추가하여 개인을 식별할 수 없도록 보호하는 프라이버시 보존 기법
  • 키워드: 노이즈 추가, ε-DP, 프라이버시 예산, 라플라스 메커니즘, 가우시안 메커니즘, DP-SGD, 민감도, 전역·지역 DP
  • 분류: 암호학
  • 핵심 개념:
    항목설명핵심
    ε (프라이버시 예산)허용 가능한 프라이버시 손실 수준 (작을수록 강한 보호)보호 수준 조절
    민감도 (Sensitivity)특정 개인의 데이터 유무에 따른 질의 결과 최대 변화량노이즈 크기 기준
    노이즈 추가민감도·ε 기반으로 계산된 노이즈를 질의 결과에 추가개인 식별 방지
    프라이버시-유용성 트레이드오프ε 작을수록 강한 보호·낮은 데이터 유용성, 클수록 반대균형 설계 필요
  • 주요 메커니즘:
    항목설명핵심
    라플라스 메커니즘라플라스 분포 기반 노이즈 추가, 수치형 데이터에 적용수치 데이터
    가우시안 메커니즘가우시안 분포 기반 노이즈 추가, (ε·δ)-DP 만족근사 DP
    지수 메커니즘범주형·비수치 데이터의 최적 출력 선택에 적용범주 데이터
    무작위 응답응답 전 확률적으로 참·거짓 반전하여 개인 응답 보호설문 데이터
  • 전역 vs 지역 차분 프라이버시:
    항목전역 DP (Global DP)지역 DP (Local DP)
    노이즈 추가 위치중앙 서버에서 집계 후 추가개인 기기에서 전송 전 추가
    신뢰 주체중앙 서버 신뢰 필요서버 신뢰 불필요
    데이터 유용성높음낮음
    대표 사례인구조사·DB 통계애플 iOS·구글 RAPPOR
  • AI·ML 적용 (DP-SGD):
    항목설명핵심
    DP-SGD경사하강법 적용 시 기울기에 노이즈 추가하여 학습 데이터 보호ML 학습 프라이버시
    기울기 클리핑개별 기울기 크기를 제한하여 민감도 통제민감도 제어
    프라이버시 예산 추적학습 전 과정의 누적 ε 관리예산 소진 관리
  • 인사이트: 차분 프라이버시는 집계 통계 공개 시 개인 정보 유출을 수학적으로 보장하는 유일한 기법으로, DP-SGD를 통해 AI 모델 학습 데이터 보호로 적용 범위가 확대되고 있으며 ε 설정이 프라이버시·유용성 균형의 핵심 설계 변수로 실무 적용 시 도메인별 적정 예산 수준 결정이 가장 중요한 과제임
