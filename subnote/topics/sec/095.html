7. 스마트폰 포렌식 (Smartphone Forensic)

  • 정의: 스마트폰에서 디지털 증거를 수집·분석하는 포렌식 기술
  • 키워드: 논리적/물리적 추출, 데이터 유형, 절차
  • 분류: 디지털 포렌식
  • 데이터 추출 방법:
    • 논리적 추출: 백업, ADB(Android Debug Bridge)로 파일시스템 접근 → 비교적 쉬움
    • 물리적 추출: JTAG, 칩오프(Chip-off)로 스토리지 직접 덤프 → 암호화 우회 가능
    • 클라우드 추출: 계정 백업(iCloud, Google Drive)에서 데이터 획득
  • 인사이트: iOS는 강력한 암호화와 Secure Enclave로 물리적 추출이 어렵고, Android는 제조사별로 상이하여 포렌식 도구(Cellebrite, Oxygen Forensics)의 지속적 업데이트가 중요함
