2. 디지털 포렌식 아티팩트 (Artifact)

  • 정의: 디지털 포렌식에서 분석 대상이 되는 시스템 활동 흔적 데이터
  • 키워드: 레지스트리, 이벤트 로그, Prefetch, MFT, DNS 캐시, PCAP, 브라우저 기록, 메타데이터, 슬랙 공간
  • 분류: 디지털 포렌식
  • 주요 아티팩트 유형:
    구분항목핵심
    윈도우레지스트리설정·실행 흔적
    이벤트 로그행위 기록
    Prefetch실행 이력
    LNK 파일파일 접근 이력
    NTFS $MFT파일 시스템 이력
    점프 리스트최근 사용 이력
    네트워크방화벽 로그네트워크 접근
    DNS 캐시도메인 접근 이력
    PCAP패킷 분석
    라우터·스위치 로그내부 이동 추적
    브라우저방문 기록웹 접근 이력
    다운로드 기록다운로드 추적
    쿠키·세션인증 정보
    모바일통화·SMS 기록통신 이력
    앱 데이터앱 사용 흔적
    위치 정보위치 추적
    사진·미디어 메타데이터미디어 흔적
    메모리프로세스 목록실행 프로세스
    네트워크 연결활성 연결
    암호화 키키·자격증명
  • 인사이트: 아티팩트는 삭제 후에도 MFT·슬랙 공간·볼륨 섀도·메모리에 잔존하므로 철저한 안티포렌식 없이는 완전 은닉이 어려우며, 공격자의 행위 재구성은 단일 아티팩트가 아닌 윈도우·네트워크·브라우저·메모리 아티팩트의 교차 분석을 통해 타임라인을 구성하는 것이 핵심임
