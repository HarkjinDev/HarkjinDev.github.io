2. 무자각 지속 인증 (Implicit Continuous Authentication)

  • 정의: 사용자의 행위 및 환경정보를 활용하여 사용자의 명시적 개입 없이 지속적으로 신원을 검증하는 신뢰기반의 인증기술
  • 키워드: 행위 기반, 생체, 명시적 미고지, 지속적 검증, 무의식, 제로트러스트
  • 분류: 인증/신원관리
  • 특징: 로그인 후에도 지속적 신원 검증, 사용자 개입 없음, AI/ML 기반 이상 탐지
  • 인증 절차:
    인증절차기능설명
    1. 관찰사용자 인증 데이터 수집사용자 특화 속성 정보 수집
    무자각 모니터링상호작용 무자각 모니터링
    2. 프로파일 생성·갱신신규 사용자 프로파일 생성신규 사용자 프로파일 생성
    기존 사용자 프로파일 갱신수집 데이터 기반 프로파일 갱신
    3. 비교패턴 매칭프로파일 데이터 패턴 비교
    머신러닝 기반 본인 여부 분석ML 기반 본인 여부 분석
    4. 결정인증 승인결과에 따른 승인 거부 결정
    재인증 요청정책 기반 재인증 및 조치 수행
  • 장점: 세션 하이재킹 탐지, 비밀번호 없는 사용자 경험
  • 단점: 프라이버시 침해 우려, 오탐(False Positive) 가능성
  • 인사이트: 무자각 지속 인증은 제로 트러스트 아키텍처의 "지속 검증(Continuous Verification)" 원칙을 구현하는 핵심 기술임
