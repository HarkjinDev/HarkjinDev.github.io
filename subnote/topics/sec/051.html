11. XDR (Extended Detection and Response, 확장 탐지 및 대응)

  • 정의: 엔드포인트(EDR)를 넘어 네트워크·클라우드·이메일·아이덴티티 등 다양한 보안 레이어의 데이터를 통합·상관분석하여 위협을 탐지·대응하는 통합 보안 플랫폼
  • 키워드: 통합 보안(보안레이어 통합), 사일로 구조 해결, 상관분석, SIEM 연계, EDR 확장, 네이티브 XDR, 하이브리드 XDR, 위협 인텔리전스, SOC 자동화
  • 분류: 네트워크 보안
  • 구성/기술요소:
    구분구성요소핵심 기술
    데이터 수집엔드포인트행위 분석, 프로세스 모니터링, 자동 격리
    네트워크NDR, 패킷 분석, 이상 탐지
    클라우드CWPP, CSPM, API 모니터링
    이메일샌드박스, URL 분석, 발신자 검증
    아이덴티티IAM, UEBA, MFA 연계
    위협 탐지위협 인텔리전스TI 피드, IOC 매칭, MITRE ATT&CK
    자동 상관분석ML 기반 탐지, 경보 노이즈 감소
    공격 경로 재구성Kill Chain 시각화, 침해 타임라인
    대응·복구SOAR 연계플레이북 기반 자동 대응·오케스트레이션
    자동 격리·차단엔드포인트 격리, 계정 차단, 트래픽 차단
  • XDR 유형:
    항목설명핵심
    네이티브 XDR단일 벤더의 통합 보안 제품군으로 구성단일 벤더 통합
    하이브리드 XDR서드파티 보안 솔루션을 오픈 API로 통합멀티 벤더 연계
  • EDR·SIEM과의 비교:
    항목EDRSIEMXDR
    데이터 범위엔드포인트전체 로그 수집보안 레이어 통합
    상관분석❌ 단일 레이어△ 룰 기반✅ 자동 상관분석
    대응 자동화△ 엔드포인트 한정❌ 대응 미포함✅ 통합 자동 대응
    운영 복잡도낮음높음중간
    위협 가시성엔드포인트로그 중심전체 공격 경로
  • 인사이트: XDR은 보안 솔루션의 사일로(Silo) 구조로 인한 탐지 공백·경보 과부하 문제를 통합 상관분석으로 해결하는 차세대 SOC 아키텍처로, 단일 벤더 의존성과 멀티 벤더 통합 복잡성 간의 선택이 도입 전략의 핵심 의사결정 포인트이며 SIEM·SOAR와의 역할 경계 재정립이 진행 중임
