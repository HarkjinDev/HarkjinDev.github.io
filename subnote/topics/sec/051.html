9. SOAR (Security Orchestration, Automation and Response)

  • 정의: 보안 도구 간 오케스트레이션·자동화·사고대응을 통합하여 보안 운영의 효율성을 높이고 위협 대응 시간을 단축하는 보안 플랫폼
  • 키워드: SOA, SIRP, TIP, 가시성, 플레이북, 워크플로우, 사고대응, Reporting, 경보 피로
  • 분류: 네트워크 보안
  • 아키텍처 (SOA + SIRP + TIP):
    구분구성요소핵심 기술
    SOA보안 오케스트레이션이기종 보안 도구 API 연동·워크플로우 자동화
    플레이북 엔진사고 유형별 대응 절차 자동 실행·분기 처리
    SIRP사고 관리보안 이벤트 접수·분류·추적·에스컬레이션
    케이스 관리사고별 조사 이력·증거·협업 내용 통합 관리
    TIP위협 인텔리전스외부 TI 피드 수집·정규화·IOC 자동 배포
    컨텍스트 강화경보에 위협 정보·공격자 프로파일 자동 연계
  • 인사이트: SOAR는 경보 폭증·인력 부족으로 한계에 달한 SOC 운영을 플레이북 기반 자동화로 혁신하는 핵심 플랫폼으로, SIEM이 "무엇을 탐지했는가"를 담당한다면 SOAR는 "어떻게 대응할 것인가"를 자동화하는 역할을 하며 두 플랫폼의 통합 운용이 현대 SOC의 표준 아키텍처로 자리잡고 있음
