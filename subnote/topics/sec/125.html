7. MCP 보안 (Model Context Protocol Security)

  • 정의: MCP기반 AI 서비스 구축 시 발생 가능한 보안 취약점과 대응 방안
  • 키워드: 프롬프트 인젝션, 도구 오염 공격, CVE-2025-6514(mcp-remote 취약점)
  • 분류: AI 보안
  • 주요 취약점:
    • 프롬프트 인젝션: 외부 도구/리소스를 통한 간접 인젝션
    • 도구 오염 공격(Tool Poisoning): 악의적 MCP 서버가 안전한 도구처럼 위장
    • 권한 상승: MCP 통해 과도한 권한 획득
    • CVE-2025-6514: mcp-remote의 원격 코드 실행 취약점
  • 인사이트: MCP 서버 출처 검증과 샌드박스 실행이 핵심 보안 원칙임
