5. DDoS (Distributed Denial of Service)

정의: 다수의 좀비 PC·서버(봇넷)를 이용하여 비정상적 대규모 트래픽을 유발해 서비스를 마비시키는 분산 서비스 거부 공격
키워드: 봇넷, 대역폭 소진, 자원 소진, 스크러빙 센터, CDN, 앰플리피케이션, Anycast, L7 공격, ISP 필터링
분류: 네트워크 보안
공격 유형:
구분
구성요소
핵심 기술
볼류메트릭 공격
UDP Flood
대량 UDP 패킷으로 대역폭 소진
ICMP Flood
Ping 패킷 대량 전송으로 네트워크 마비
DNS 앰플리피케이션
소량 요청으로 대량 응답 유발, 증폭 공격
NTP 앰플리피케이션
NTP 프로토콜 악용 트래픽 수십 배 증폭
프로토콜 공격
SYN Flood
TCP 연결 미완료로 서버 자원(소켓) 소진
Ping of Death
비정상 대형 패킷으로 시스템 충돌 유발
Smurf Attack
브로드캐스트 ICMP로 다수 응답 유발
애플리케이션 공격
HTTP Flood
대량 HTTP 요청으로 웹 서버 자원 소진
Slowloris
연결을 천천히 유지하여 웹 서버 스레드 고갈
SSL 재협상 공격
반복 SSL 핸드셰이크로 서버 연산 자원 소진
대응 방안:
구분
구성요소
핵심 기술
트래픽 정제
스크러빙 센터
정상·비정상 트래픽 분리 후 정상만 전달
Anycast 라우팅
공격 트래픽을 전 세계 분산 노드로 흡수
트래픽 분산
CDN 활용
전 세계 엣지 서버로 트래픽 분산·부하 감소
로드 밸런서
서버 간 트래픽 분배로 단일 장애점 제거
필터링
ISP 업스트림 필터링
공격 트래픽을 ISP 레벨에서 조기 차단
BCP38
출발지 IP 위조(스푸핑) 방지 필터링 표준
탐지·완화
이상 트래픽 탐지
ML 기반 정상 트래픽 패턴 대비 이상 탐지
Rate Limiting
출발지 IP별 요청 속도 임계치 제한
