1. 네트워크 보안 장비

  • 정의: 네트워크 트래픽을 실시간으로 감시하고 미리 정의된 보안 정책(Policy)에 따라 트래픽을 필터링, 탐지, 차단, 암호화하는 통제 지점(Control Point)의 역할을 수행하는 하드웨어 및 소프트웨어 솔루션
  • 키워드: NAC, FW, IDS, IPS
  • 분류: 네트워크 보안
  • 보안장비 종류:
    장비개념핵심
    NAC단말 보안 검증·접근 제어가시성 확보, 단말 무결성 체크, 비인가 단말 차단·격리
    FW (Firewall)규칙 기반 트래픽 차단IP/Port 기반 제어, 패킷 필터링, Stateful Inspection
    IDS침입 징후 탐지·알림사후 대응, 수동적(Passive) 모니터링, 오용·이상 탐지
    IPS공격 실시간 탐지·차단능동적(Active) 방어, 인라인(In-line) 배치, HIPS/NIPS
    WAF웹 공격 특화 방어L7 제어, 페이로드 검사, SQL Injection·XSS 방어
