3. 개인정보 비식별조치 (De-identification)

  • 정의: 개인을 식별할 수 없도록 개인정보를 처리하는 비식별 기법
  • 키워드: 가명화, 총계처리, 데이터 삭제, 범주화
  • 분류: 개인정보보호
  • 비식별 기법: <가총삭범마>
    기법설명예시
    가명처리식별자를 가명으로 대체홍길동 → 홍○○
    총계처리개별값 대신 통계값 사용나이 → 평균 35세
    데이터 삭제식별 가능 항목 제거이름, 주소 필드 삭제
    범주화값을 범위로 변환35세 → 30대
    데이터 마스킹부분 값 대체010--5678
  • 인사이트: 비식별조치의 충분성은 재식별 가능성 검토(전문가 판단, k-익명성 등)로 확인해야 하며, 완전한 비식별화는 데이터 유용성을 크게 감소시킬 수 있음
