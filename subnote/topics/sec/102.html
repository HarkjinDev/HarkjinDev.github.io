2. 블록체인 합의 알고리즘 (Consensus Algorithm)

  • 정의: 분산 노드 간 데이터 일치를 위해 합의를 도출하는 알고리즘
  • 키워드: 이중지불 문제, 비잔틴 문제, PoW, PoS, PBFT
  • 분류: 특수 보안
  • 주요 합의 알고리즘 비교:
    알고리즘방식장점단점적용 사례
    PoW (작업 증명)연산 경쟁으로 블록 생성권 획득보안성 강함·검증 단순에너지 소비 과다·느린 처리Bitcoin
    PoS (지분 증명)지분 보유량 기반 검증자 선출에너지 효율·확장성부의 집중·초기 분배 문제Ethereum 2.0
    DPoS (위임 지분 증명)지분 보유자가 대표자 선출·위임빠른 처리 속도대표자 담합·중앙화 우려EOS
    PBFT (실용 비잔틴)3단계 메시지 교환으로 합의즉각적 최종성·높은 신뢰노드 수 증가 시 성능 저하Hyperledger
    PoA (권위 증명)신뢰된 검증 기관이 블록 생성고성능·낮은 에너지탈중앙성 낮음·신뢰 의존기업형 블록체인
  • 인사이트: 합의 알고리즘 선택은 보안성-성능-탈중앙성의 트릴레마(Trilemma)에서 어떤 가치를 우선시하느냐에 따라 결정됨
