6. Diffusion Model (확산 모델)

정의: 데이터에 단계적으로 노이즈를 추가하는 순방향 과정(Forward)과, 노이즈에서 원본 데이터를 복원하는 역방향 과정(Reverse)을 학습하여 고품질 데이터를 생성하는 생성 모델
키워드: 노이즈 추가(순방향), 노이즈 제거(역방향), 마르코프 체인(단계별 상태 전이 원리), U-Net(노이즈 예측), 잠재 확산(Latent Diffusion), 스코어 매칭(확률 밀도 기반), DDPM, DDIM, LDM, CFG, 다양한 조건부 생성, 느린 추론
분류: 비지도학습

인사이트: Diffusion 모델은 학습 안정성과 생성 품질을 동시에 달성하며 GAN을 제치고 생성 AI의 주류 아키텍처로 부상함. LDM이 잠재 공간에서 확산 과정을 수행하는 구조로 연산 효율을 크게 개선하였고, 텍스트-이미지 생성을 넘어 영상·3D·과학 도메인까지 확장되며 생성 AI 전반의 핵심 기반 기술로 자리잡고 있음
