6. Diffusion Model (확산 모델)

  • 정의: 데이터에 단계적으로 노이즈를 추가하는 순방향 과정(Forward)과, 노이즈에서 원본 데이터를 복원하는 역방향 과정(Reverse)을 학습하여 고품질 데이터를 생성하는 생성 모델
  • 키워드: 노이즈 추가(순방향), 노이즈 제거(역방향), 마르코프 체인(단계별 상태 전이 원리), U-Net(노이즈 예측), 잠재 확산(Latent Diffusion), 스코어 매칭(확률 밀도 기반), DDPM, DDIM, LDM, CFG, 다양한 조건부 생성, 느린 추론
  • 분류: 비지도학습
  • 동작 원리:
    항목설명핵심
    순방향 (Forward)원본 데이터에 가우시안 노이즈를 T 스텝에 걸쳐 점진적 추가노이즈 확산
    역방향 (Reverse)노이즈에서 노이즈 제거 과정을 학습해 원본 복원노이즈 제거
    학습 목표타임스텝 t에서 추가된 노이즈를 U-Net으로 예측노이즈 예측
  • 주요 유형: DDPM, DDIM, LDM, CFG
  • 장단점:
    항목설명핵심
    장점GAN 대비 학습 안정적, 모드 붕괴 없음안정적 학습
    텍스트·이미지 등 다양한 조건부 생성 가능조건부 제어
    이미지·오디오·영상·3D 등 멀티 도메인 적용범용성
    단점수백~수천 스텝 샘플링으로 생성 속도 느림느린 추론
    높은 추론 연산 비용으로 실시간 적용 제약비용 높음
  • 활용 분야:
    • 이미지 생성·편집: Stable Diffusion, DALL·E, Midjourney
    • 영상 생성: Sora, Runway
    • 오디오·음악 생성: AudioLDM, MusicGen
    • 신약 개발: 단백질·분자 구조 생성
  • 인사이트: Diffusion 모델은 학습 안정성과 생성 품질을 동시에 달성하며 GAN을 제치고 생성 AI의 주류 아키텍처로 부상함. LDM이 잠재 공간에서 확산 과정을 수행하는 구조로 연산 효율을 크게 개선하였고, 텍스트-이미지 생성을 넘어 영상·3D·과학 도메인까지 확장되며 생성 AI 전반의 핵심 기반 기술로 자리잡고 있음
