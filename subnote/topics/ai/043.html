9. GNN (Graph Neural Network, 그래프 신경망)

  • 정의: 노드(Node)와 엣지(Edge)로 구성된 그래프 데이터의 복잡한 연결 관계·상호 의존성을 학습하여 분류·예측·관계 파악을 수행하는 신경망 알고리즘
  • 키워드: 노드, 엣지, 인접행렬, Aggregate·Combine·Readout, GCN, GAT, GraphSAGE, 메시지 패싱
  • 분류: 신경망
  • 특성:
    항목설명핵심
    그래프 데이터노드·엣지로 구성된 비유클리드 데이터관계 구조 표현
    인접행렬 (Adjacency Matrix)노드 간 연결 정보를 행렬로 표현연결 구조
    노드 특징행렬각 노드의 속성값을 행렬로 표현노드 속성
    메시지 패싱이웃 노드 정보를 취합·갱신하는 반복 과정정보 전파
  • 태스크 유형:
    유형설명핵심
    노드 레벨각 노드에 대한 분류·회귀 문제 해결노드 단위 예측
    그래프 레벨전체 그래프에 대한 분류·회귀 문제 해결그래프 단위 예측
    엣지 레벨노드 간 연결 여부·관계 파악링크 예측
  • GNN 학습 절차:
    단계설명핵심
    1. 변환 (Transform)그래프를 인접행렬·노드 특징행렬로 변환입력 구조화
    2. 취합 (Aggregate)타깃 노드에 인접한 이웃 노드의 은닉 변수 정보 취합이웃 정보 수집
    3. 업데이트 (Update/Combine)타깃 노드와 이웃 노드 정보 결합하여 노드 갱신노드 표현 갱신
    4. 생성 (Readout)모든 노드 은닉 변수 결합하여 그래프 단위 표현 생성그래프 표현 생성
  • 주요 GNN 모델:
    구분모델설명핵심
    합성곱 기반GCN (Graph Convolutional Network)스펙트럼 기반 그래프 합성곱 연산이웃 정보 집계
    GraphSAGE이웃 샘플링 기반 귀납적 표현 학습대규모 그래프
    어텐션 기반GAT (Graph Attention Network)이웃 노드별 어텐션 가중치 차등 적용중요 이웃 집중
    생성 기반GraphVAE변분 오토인코더 기반 그래프 생성그래프 생성
    시계열 기반Spatial-Temporal GNN공간·시간 상관관계 동시 학습시공간 예측
  • 인사이트: GNN은 소셜 네트워크·분자·지식 그래프 등 비유클리드 구조 데이터를 효과적으로 학습하는 핵심 딥러닝 패러다임으로, Aggregate→Combine→Readout의 메시지 패싱 구조가 핵심임. 대규모 그래프의 확장성·과평활화(Over-smoothing) 문제가 실무 적용의 핵심 과제이며, 시공간 GNN과 LLM의 결합이 차세대 그래프 AI의 주요 연구 방향으로 부상하고 있음
