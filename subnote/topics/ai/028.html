[하위] 3-2. LDA (Linear Discriminant Analysis, 선형 판별 분석)

정의: 클래스 내 분산은 최소화하고 클래스 간 분산은 최대화하는 축을 찾아 데이터를 저차원으로 투영하는 지도 학습 기반 차원 축소 기법
키워드: 클래스 분별력, 판별 함수, 클래스 내 산포행렬(SW,분산), 클래스 간 산포 행렬(SB, 평균), 다중 클래스 분류
분류: 지도학습
핵심 구성 요소:
클래스 내 산포 행렬 (Within-class Scatter, SW): 같은 클래스 내 데이터들의 분산을 측정
클래스 간 산포 행렬 (Between-class Scatter, SB): 서로 다른 클래스 평균 간의 거리를 측정
판별 함수: SW에 대한 SB의 비율을 최대화하는 방향으로 축을 설정
PCA와의 비교:
PCA: 라벨 불필요(비지도), 전체 데이터의 분산 최대화 방향으로 축 생성
LDA: 라벨 필요(지도), 클래스 분별력 최대화 방향으로 축 생성, 분류 성능에 특화
인사이트: 분류 문제가 명확한 경우 PCA보다 클래스 구분에 유리하지만, 각 클래스가 동일한 공분산을 가진다는 정규 분포 가정이 필요하므로 데이터 분포 확인이 선행되어야 함
