2. 머신러닝 (Machine Learning)

  • 정의: 데이터를 통해 패턴과 규칙을 컴퓨터가 스스로 학습하여 미래 결과를 예측하거나 판단하게 하는 인공지능의 하위 분야
  • 키워드: 학습 데이터, 특징 추출(Feature Extraction), 모델링, 예측 모델, 귀납적 사고, 지도/비지도/강화
  • 분류: AI 개론
  • 귀납적사고(머신러닝의 근간):
    • 개별적인 관찰 사례와 데이터들을 종합하여 일반적인 규칙이나 가설을 도출해내는 사고 방식
    • 관찰, 패턴 인식, 일반화(Generalization), 가설 설정
    • 연역적 사고(규칙→결론) vs 귀납적 사고(사례→규칙)
  • 학습 방식별 분류:
    • 지도학습 / 비지도학습 / 강화학습
  • 핵심 프로세스:
    • 데이터 수집 → 전처리 → 모델 선택 → 학습 → 평가 → 배포
  • 인사이트: 양질의 데이터 확보가 모델 성능을 결정(Garbage In, Garbage Out), 피처 엔지니어링의 중요성
