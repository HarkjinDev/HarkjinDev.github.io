1. SVM (Support Vector Machine)

  • 정의: 데이터를 분류할 때 서로 다른 클래스 간의 거리(Margin)를 최대화하는 최적의 초평면(Hyperplane)을 찾는 지도학습 기반 분류 알고리즘
  • 키워드: 초평면, 마진 최대화, 서포트 벡터, 커널 트릭, 고차원 매핑
  • 분류: 지도학습
  • 주요 특징: 과적합 회피, 통계적 학습, 차원의 저주 회피
  • 핵심 구성 요소:
    • 초평면 (Hyperplane): 클래스를 구분하는 결정 경계로, N차원 데이터에서 N-1차원의 경계면
    • 서포트 벡터 (Support Vector): 결정 경계 형성에 직접 영향을 주는 경계선 근처의 데이터 포인트
    • 커널 트릭 (Kernel Trick): 저차원 데이터를 고차원으로 매핑하여 선형 분리가 불가능한 데이터도 분류 가능하게 하는 기법
  • 커널 기법:
    커널 기법설명핵심
    선형 커널 (Linear)데이터를 변환 없이 선형 분리함기본 커널, 연산 속도 빠름
    다항식 커널 (Polynomial)데이터를 다항식 차원으로 변환함복잡한 비선형 경계 학습
    RBF 커널 (Gaussian)무한 차원 공간으로 매핑함가장 보편적, 가우스 분포 기반
    시그모이드 커널 (Sigmoid)신경망 활성화 함수와 유사함로지스틱 회귀와 유사한 성질
  • 인사이트: 데이터 차원이 커도 효과적이며 과적합에 강한 특성이 있어 소규모 고품질 데이터 학습에 유리하나, 대용량 데이터에서는 연산 비용이 크게 증가하는 한계가 있음
