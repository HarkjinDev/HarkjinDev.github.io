2. 워드 임베딩 (Word Embedding)

  • 정의: 단어를 고차원 벡터 공간의 수치(실수)로 변환하여 단어 간의 의미적 유사성을 보존하는 기술
  • 키워드: 분산 가설, Word2Vec, CBOW, Skip-gram, GloVe, FastText, ELMo, BERT, 코사인 유사도, 단어 유추, 상관계수, t-SNE
  • 분류: 자연어 처리 (NLP)
  • 분산 가설(Distributional Hypothesis): 비슷한 문맥에서 나타나는 단어들은 비슷한 의미를 가진다는 가정(유사 문맥 = 유사 의미)
  • 주요 모델:
    분류모델설명핵심
    추론 기반 (정적)Word2Vec신경망 기반 단어 벡터화단어 간 산술 연산
    추론 기반 (정적)CBOW주변 단어로 중심 단어 예측빠른 학습 속도
    추론 기반 (정적)Skip-gram중심 단어로 주변 단어 예측저빈도 단어 학습 유리
    통계 기반 (정적)GloVe말뭉치 전체 통계 정보 반영전역적 의미 관계 보존
    서브워드 (정적)FastText단어를 부분 단위로 분리 학습미등록 단어(OOV) 대응
    문맥 기반 (동적)ELMo양방향 LSTM으로 문맥 파악단어 중의성 해결 가능
    문맥 기반 (동적)BERT트랜스포머 양방향 인코딩현재 NLP 핵심 표준 기술
  • 워드 임베딩 성능 지표:
    항목설명핵심
    코사인 유사도벡터 간의 각도로 유사도 측정1에 가까울수록 의미적 유사함
    단어 유추 (Analogy)단어 간 관계를 벡터 연산함의미적·문법적 추론 능력 검증
    상관계수 (Correlation)인간 평가와 수치 상관성 확인전문가 판단(WS-353)과의 일치도
    클러스터 순도 (Purity)군집화 후 동일 범주 포함 비율단어 범주화의 정확성 평가
    시각화 평가 (t-SNE)고차원 벡터를 2차원 투영함군집 형성 여부 시각적 확인
  • 인사이트: 단어를 단순 숫자로 바꾸는 단계를 넘어, 단어 간의 '거리'를 통해 의미적 관계를 산출함. 이는 단순 통계 기반에서 딥러닝 기반 언어 모델로 넘어가는 핵심 가교 역할을 함
