10. 강화학습 (Reinforcement Learning, RL)

  • 정의: 에이전트가 환경과 상호작용하며 보상을 최대화하는 행동 정책을 시행착오를 통해 스스로 학습하는 머신러닝 패러다임
  • 키워드: 에이전트, 환경, 상태, 행동, 보상, 정책, 가치 함수, Q-러닝, PPO, 마르코프 결정 과정(MDP)
  • 분류: 생성형 AI / LLM
  • 핵심 구성 요소:
    항목설명핵심
    에이전트 (Agent)환경과 상호작용하며 행동을 결정하는 주체학습 주체
    환경 (Environment)에이전트가 행동하고 보상을 받는 외부 세계상호작용 대상
    상태 (State)현재 환경의 상황을 나타내는 정보현재 상황
    행동 (Action)에이전트가 상태에서 선택할 수 있는 행위선택지
    보상 (Reward)행동의 결과로 환경이 에이전트에게 제공하는 피드백학습 신호
    정책 (Policy)상태에서 행동을 선택하는 에이전트의 전략행동 전략
    가치 함수 (Value)특정 상태·행동의 장기적 누적 보상 기댓값장기 보상 평가
  • 마르코프 결정 과정 (MDP):
    항목설명핵심
    마르코프 속성현재 상태만으로 다음 상태가 결정됨 (과거 독립)현재 상태 의존
    구성(S, A, P, R, γ) — 상태·행동·전이확률·보상·할인율MDP 5요소
    할인율 (γ)미래 보상을 현재 가치로 환산하는 계수 (0~1)미래 보상 가중
  • 주요 알고리즘 분류:
    구분알고리즘핵심
    가치 기반Q-러닝, DQNQ값으로 최적 행동 선택
    정책 기반REINFORCE, PPO정책 직접 최적화
    액터-크리틱A3C, SAC가치·정책 동시 학습
    모델 기반AlphaZero, Dreamer환경 모델 학습 후 계획
  • 주요 기술:
    항목설명핵심
    Q-러닝상태-행동 쌍의 Q값을 학습하여 최적 정책 도출가치 기반 학습
    DQNQ-러닝에 딥러닝을 결합, 고차원 상태 공간 처리딥 Q-러닝
    PPO정책 업데이트 폭을 제한하여 안정적으로 학습하는 알고리즘안정적 정책 학습
    탐색-활용 균형새로운 행동 탐색(Exploration)과 기존 최적 활용(Exploitation)의 균형ε-greedy 전략
    경험 재생 (Replay Buffer)과거 경험을 저장 후 무작위 샘플링하여 학습 안정화상관성 제거
    보상 설계 (Reward Shaping)학습 효율을 높이기 위한 중간 보상 설계학습 가속화
  • 지도학습·비지도학습과 비교:
    항목강화학습지도학습비지도학습
    학습 신호보상 (지연·희소)레이블 (즉각)없음
    데이터환경 상호작용레이블 데이터비레이블 데이터
    목표누적 보상 최대화정답 예측패턴·구조 발견
    적용 문제순차적 의사결정분류·회귀클러스터링
  • 인사이트: 강화학습은 명시적 정답 없이 환경과의 상호작용만으로 초인적 성능을 달성한 AlphaGo·ChatGPT(RLHF)의 핵심 기반으로, 단순 게임을 넘어 LLM 정렬·로보틱스·Physical AI의 핵심 학습 패러다임으로 확장되고 있으며 "보상 설계의 질"이 곧 에이전트 행동의 질을 결정하는 핵심 변수임
