10. 추론 모델 (Reasoning Model)

  • 정의: 복잡한 문제를 단계적·논리적 사고과정으로 분해해 추론하는 LLM 유형
  • 키워드: 딥시크, CoT, Test-time Compute, System2 Thinking, 강화학습기반 추론
  • 분류: AI 응용 / 트렌드
  • 구성요소·기술요소:
    구분구성요소핵심 기술
    학습강화학습(RLVR)정답 검증가능 보상으로 추론경로 학습
    Process Reward Model(PRM)추론 각 단계별 정답여부 보상
    Outcome Reward Model(ORM)최종 결과만 평가하는 보상모델
    Distillation대형 추론모델의 사고패턴 경량모델 전이
    탐색Chain-of-Thought(CoT)단계별 중간추론 생성
    Tree-of-Thought(ToT)다중 추론경로 트리형 탐색
    Monte Carlo Tree Search(MCTS)무작위 시뮬레이션 기반 탐색트리 확장
    Best-of-N SamplingN개 후보 중 최적답 선택
    추론심화Test-time Compute추론시점에 연산량을 늘려 정확도향상
    System 2 Thinking빠른직관 아닌 느리고 신중한 추론
    자기검증(Self-verification)생성한 답을 스스로 재검토·수정
