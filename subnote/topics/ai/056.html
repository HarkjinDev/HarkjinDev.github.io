[하위] 2-3. LLM Reasoning 기법

  • LLM Reasoning 기법:
    기법설명대표 모델/기법
    사고사슬 (CoT)중간 추론 단계별 서술GPT-4, PaLM
    디컴포지션 (Decomposition)문제를 하위로 분해Least-to-Most, Self-Ask
    메타-리즌 (Meta-Reasoning)추론 과정 자기검토·수정Self-Reflection, ReAct
    귀납적 추론 (Inductive Reasoning)데이터→패턴→일반화In-Context Learning
    상호 추론 (Mutual Reasoning)두 모델 상호 검증rStar(Microsoft)
    트리탐색 (Tree of Thought: ToT)트리구조 다경로 탐색GPT-4 + ToT
    자기일관성 (Self-Consistency)다수 경로 중 최다 답 선택CoT-SC
    연역적 추론 (Deductive Reasoning)규칙→전제→결론 도출LogiCoT
    유추 추론 (Analogical Reasoning)유사 사례 기반 추론Structure Mapping
    프로그램합성 (PAL)코드로 추론 과정 표현Program-Aided Language
    ReAct추론+행동 반복 루프LangChain, ReAct
