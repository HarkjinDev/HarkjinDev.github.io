4. TF-IDF (Term Frequency-Inverse Document Frequency)

정의: 특정 문서 내에서 단어의 빈도(TF)와 해당 단어가 전체 문서군에서 나타나는 빈도의 역수(IDF)를 곱하여 단어의 중요도를 평가하는 통계적 가중치 모델
키워드: 단어 빈도(TF), 역문서 빈도(IDF), 텍스트 마이닝, 정보 검색, 가중치 산출, 희소 행렬, 핵심어 추출, 코사인 유사도
분류: 자연어처리(NLP)
활용 원리: 특정 문서에는 많이 나오지만 전체 문서에는 흔치 않은 단어일수록 높은 점수를 부여하여 '핵심어'를 추출
핵심 공식: (로그를 취하는 이유: 문서 수가 커질수록 IDF 값이 폭발적으로 증가하는 것을 방지)
요소
공식
설명
TF
TF(t,d)=단어 t의 등장 횟수문서 d의 전체 단어 수TF(t, d) = \frac{\text{단어 } t\text{의 등장 횟수}}{\text{문서 } d\text{의 전체 단어 수}}TF(t,d)=문서 d의 전체 단어 수단어 t의 등장 횟수​​
특정 문서 내에서 단어 출현 비율
IDF
IDF(t)=log⁡(전체 문서 수단어 t가 포함된 문서 수)IDF(t) = \log\left(\frac{\text{전체 문서 수}}{\text{단어 } t\text{가 포함된 문서 수}}\right)IDF(t)=log(단어 t가 포함된 문서 수전체 문서 수​)​
단어의 희소성을 반영 (흔한 단어일수록 낮은 값)
TF-IDF
TF-IDF(t,d)=TF(t,d)×IDF(t)TF\text{-}IDF(t, d) = TF(t, d) \times IDF(t)TF-IDF(t,d)=TF(t,d)×IDF(t)​
단어의 최종 중요도를 나타내는 가중치
