3. 과소적합 (Underfitting)

  • 정의: 머신러닝 모델이 학습 데이터의 패턴을 충분히 학습하지 못하여 학습 데이터와 새로운 데이터 모두에서 낮은 성능을 보이는 현상
  • 키워드: 높은 편향(High Bias), 낮은 분산, 모델 복잡도 부족, 과적합 대비, 편향-분산 트레이드오프
  • 분류: 학습/최적화
  • 발생 원인:
    항목설명핵심
    모델 복잡도 부족선형 모델로 비선형 패턴 학습 시도단순 모델 한계
    학습 데이터 부족충분한 학습 데이터 미확보로 패턴 미학습데이터 부족
    학습 횟수 부족에포크 수 부족으로 수렴 전 학습 종료조기 학습 종료
    과도한 정규화L1·L2 정규화 강도 과다로 모델 표현력 억제규제 과잉
    특징 부족유의미한 입력 특징 미포함·중요 변수 누락피처 부족
    과도한 드롭아웃높은 드롭아웃 비율로 학습 능력 과도 억제드롭아웃 과다
  • 대응 방안:
    구분항목설명핵심
    모델 개선모델 복잡도 증가더 깊은 신경망·고차 다항 모델 사용표현력 향상
    앙상블 방법여러 모델 결합으로 표현력·성능 향상모델 결합
    데이터 개선학습 데이터 증가더 많은 학습 데이터 확보·수집데이터 확충
    피처 엔지니어링유의미한 특징 추가·교차 특징 생성특징 강화
    학습 개선에포크 수 증가충분한 학습 횟수 보장·조기 종료 완화충분한 학습
    정규화 완화과도한 L1·L2 정규화 강도 감소규제 완화
    학습률 조정적절한 학습률 설정으로 수렴 촉진학습률 최적화
    드롭아웃 감소드롭아웃 비율 낮춰 학습 능력 향상드롭아웃 완화
  • 편향-분산 트레이드오프:
    항목설명핵심
    편향 (Bias)모델의 예측값과 실제값의 체계적 차이학습 능력 부족
    분산 (Variance)데이터 변화에 따른 모델 예측의 변동성일반화 능력
    트레이드오프모델 복잡도 증가 시 편향 감소·분산 증가균형 설계 핵심
    최적점편향·분산 합이 최소가 되는 모델 복잡도일반화 최적
  • 인사이트: 과소적합은 모델이 데이터의 구조 자체를 이해하지 못한 상태로, 과적합과 달리 학습 데이터에서도 성능이 낮다는 점이 진단의 핵심 단서임. 모델 복잡도·데이터·학습 횟수·정규화 강도의 4가지 축에서 원인을 파악하고 조정하는 것이 대응의 핵심이며, 편향-분산 트레이드오프를 고려한 최적 모델 복잡도 탐색이 궁극적 해결책임
