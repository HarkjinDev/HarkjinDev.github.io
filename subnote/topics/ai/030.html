4. AutoEncoder (AE, 오토인코더)

  • 정의: 입력 데이터를 저차원 잠재 벡터(Latent Vector)로 압축하는 인코더와, 이를 다시 원본으로 복원하는 디코더로 구성된 비지도학습 기반 신경망
  • 키워드: 인코더, 디코더, 잠재 공간(Latent Space), 잠재 벡터(z), 병목 레이어, 차원 축소, 재구성 오차, 표현 학습, VAE, 이상 탐지
  • 분류: 비지도학습
  • 구조:
    항목설명핵심
    인코더입력을 압축해 잠재 벡터 z 생성특징 추출
    병목 레이어핵심 특징만 남긴 압축 표현 저장정보 압축
    디코더잠재 벡터 z로 원본 데이터 복원데이터 복원
    학습 목표입력·복원 출력 간 재구성 오차 최소화Loss 최소화
  • VAE (Variational AutoEncoder):
    항목설명핵심
    핵심 차이z를 평균·분산의 확률 분포로 표현분포 기반
    샘플링분포에서 z 샘플링 후 디코더 전달다양성 확보
    손실 함수재구성 오차 + KL Divergence 적용정규화
    의의AE에 생성 능력 부여한 확장 모델생성 모델화
  • 장단점:
    항목설명핵심
    장점레이블 없이 핵심 표현 학습 가능비지도 학습
    이상 탐지·노이즈 제거 등 다양한 활용범용성
    VAE 확장으로 데이터 생성 능력 보유생성 확장
    단점GAN·Diffusion 대비 생성 품질 낮음품질 한계
    잠재 공간 해석 어렵고 병목 크기 의존해석 어려움
  • 활용 분야: 이상 탐지, 차원 축소, 노이즈 제거, 생성 모델
  • 인사이트: AE는 비지도학습으로 데이터의 핵심 표현을 학습하는 기반 구조로, VAE로의 확장을 통해 생성 모델의 영역으로 발전함. 특히 Latent Diffusion Model(LDM)이 AE의 잠재 공간 위에서 확산 과정을 수행하는 구조를 채택하면서, AE는 현대 생성 AI 아키텍처의 핵심 구성 요소로 재조명되고 있음
