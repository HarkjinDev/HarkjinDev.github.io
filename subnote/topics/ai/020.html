4. 로지스틱 회귀분석 (Logistic Regression)

정의: 선형 회귀 방식을 기반으로 하되, 결과를 시그모이드 함수에 통과시켜 특정 클래스에 속할 확률(0~1)로 변환하는 이진 분류 알고리즘
키워드: 선형 결합, 시그모이드 함수, 확률 기반 분류, 로짓 함수, 이진 분류, 오즈비
분류: 지도학습
핵심 원리:
오즈비(Odds Ratio)에 로그를 취한 로짓(Logit) 변환을 사용
선형 결합 → 시그모이드 함수 → 최종 분류(이진)
인사이트: 신경망의 기본 단위인 뉴런의 연산 구조와 유사하며, 의료/금융 분야의 위험도 예측 등에 널리 활용됨
