6. 전이학습 (Transfer Learning)

  • 정의: 특정 도메인·태스크에서 사전 학습된 모델의 지식(가중치)을 유사한 다른 도메인·태스크에 재활용하여 적은 데이터와 비용으로 높은 성능을 달성하는 머신러닝 학습 패러다임
  • 키워드: 사전학습 모델, 파인튜닝, 도메인 적응, 태스트 전이, 특징 추출, 가중치 재사용, 파운데이션 모델, 소스 도메인, 타겟 도메인, 귀납적 편향
  • 분류: 생성형 AI / LLM
  • 전이학습의 유형:
    유형설명적용 조건
    도메인 적응 (Domain Adaptation)동일 태스크, 다른 도메인 간 전이태스크 동일·데이터 분포 상이
    태스크 전이 (Task Transfer)동일 도메인, 다른 태스크 간 전이도메인 유사·목적 상이
    귀납적 전이 (Inductive TL)소스·타겟 태스크가 다름, 타겟 레이블 존재일반적인 파인튜닝 시나리오
    트랜스덕티브 전이 (Transductive TL)태스크 동일, 도메인 상이, 타겟 레이블 없음도메인 적응 시나리오
    비지도 전이 (Unsupervised TL)소스·타겟 모두 레이블 없음표현 학습·클러스터링
  • 주요 적용 전략:
    전략설명적합한 상황
    특징 추출 사전학습 레이어를 동결(Freeze)하고 최종 레이어만 새로 학습데이터 매우 부족, 도메인 유사
    전체 파인튜닝 전체 레이어의 가중치를 타겟 데이터로 재학습데이터 충분, 도메인 상이
    부분 파인튜닝 상위 N개 레이어만 해동(Unfreeze)하여 학습중간 규모 데이터, 절충 전략
    PEFT 소수의 추가 파라미터만 학습 (LoRA, Adapter 등)LLM 경량 파인튜닝에 특화
  • 도메인별 대표 사전학습 모델:
    도메인대표 모델사전학습 데이터
    이미지ResNet, ViT, CLIPImageNet, JFT
    자연어 (NLP)BERT, GPT, RoBERTaWikipedia, BookCorpus
    멀티모달CLIP, DALL·E, Flamingo이미지-텍스트 쌍
    의료 영상BioViL, MedSAM의료 이미지 데이터셋
    코드CodeBERT, StarCoderGitHub 코드 저장소
    음성Wav2Vec, Whisper다국어 음성 데이터
  • 인사이트: 전이학습은 현대 AI 개발의 표준 패러다임으로, "처음부터 학습"은 소수의 대형 기관만 수행하고 대부분은 사전학습 모델을 기반으로 파인튜닝하는 구조가 산업 전반에 정착됨. LoRA·QLoRA 등 PEFT 기법의 발전으로 LLM 전이학습의 진입 장벽이 크게 낮아졌으며, 소비자급 GPU 환경에서도 수십억 파라미터 모델의 도메인 특화가 가능해짐
