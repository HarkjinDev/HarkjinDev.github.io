4. TF-IDF (Term Frequency-Inverse Document Frequency)

  • 정의: 특정 문서 내에서 단어의 빈도(TF)와 해당 단어가 전체 문서군에서 나타나는 빈도의 역수(IDF)를 곱하여 단어의 중요도를 평가하는 통계적 가중치 모델
  • 키워드: 단어 빈도(TF), 역문서 빈도(IDF), 텍스트 마이닝, 정보 검색, 가중치 산출, 희소 행렬, 핵심어 추출, 코사인 유사도
  • 분류: 자연어처리(NLP)
  • 활용 원리: 특정 문서에는 많이 나오지만 전체 문서에는 흔치 않은 단어일수록 높은 점수를 부여하여 '핵심어'를 추출
  • 핵심 공식: (로그를 취하는 이유: 문서 수가 커질수록 IDF 값이 폭발적으로 증가하는 것을 방지)
    요소공식설명
    TF TF(t, d) = \frac{\text{단어 } t\text{의 등장 횟수}}{\text{문서 } d\text{의 전체 단어 수}}특정 문서 내에서 단어 출현 비율
    IDF IDF(t) = \log\left(\frac{\text{전체 문서 수}}{\text{단어 } t\text{가 포함된 문서 수}}\right)단어의 희소성을 반영 (흔한 단어일수록 낮은 값)
    TF-IDFTF\text{-}IDF(t, d) = TF(t, d) \times IDF(t)단어의 최종 중요도를 나타내는 가중치
  • 절차:
    프로세스설명핵심
    1. 전처리토큰화, 불용어 제거, 어간 추출 수행데이터 정제 및 정규화
    2. TF 계산개별 문서 내 단어 출현 빈도 산출문서 내 단어 중요도 측정
    3. IDF 계산전체 문서 기준 역문서 빈도 산출단어의 희소성 및 변별력 파악
    4. TF-IDF 행렬 생성문서와 단어 간 희소 행렬 구성텍스트의 수치 벡터화
    5. 가중치 적용핵심어 추출 및 문서 유사도 계산검색 랭킹 및 정보 검색 활용
  • 인사이트: 단순 빈도 기반의 한계를 극복하여 검색 엔진 최적화 및 키워드 추출에 여전히 널리 사용됨
