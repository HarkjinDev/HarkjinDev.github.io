2. 과적합 (Overfitting)

  • 정의: 모델이 학습 데이터에 너무 과하게 맞춰져서, 실제 새로운 데이터(Test Data)에 대해서는 예측 성능이 떨어지는 현상
  • 키워드: 분산-편향 트레이드오프(Bias-Variance Tradeoff), 일반화 오류, 드롭아웃, 규제화, 교차검증, 조기종료
  • 분류: 학습/최적화
  • 원인: 학습 데이터 부족, 모델 복잡도 과다, 너무 많은 학습 횟수
  • 해결 방법: <드규교조>
    방법설명핵심
    드롭아웃 (Dropout)학습 시 임의의 노드를 비활성화함노드 의존도 감소
    L1/L2 규제화(Regularization)가중치에 페널티를 부여해 복잡도 제한가중치 크기 제어
    교차 검증데이터를 여러 개로 나누어 반복 검증일반화 성능 확인
    조기 종료(Early Stopping)성능 개선이 멈추면 학습을 조기 중단최적 학습 지점 포착
  • 인사이트: 과적합 방지를 위한 데이터 증강(Data Augmentation) 및 정규화(Normalization) 전략
