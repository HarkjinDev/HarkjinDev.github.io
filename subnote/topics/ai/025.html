[하위] 2-2. DBSCAN (Density-Based Spatial Clustering)

정의: 일정 반경 내의 데이터 밀도를 기준으로 군집을 형성하며 밀도가 낮은 지역의 데이터는 노이즈로 처리하는 알고리즘
키워드: Epsilon, MinPts, Core Point, Noise Point, 밀도 기반, 노이즈(이상치 제거)
분류: 비지도학습
핵심 구성 요소:
구성 요소
설명
핵심
Epsilon (ϵ\epsilonϵ)
탐색 반경 거리 기준
군집 크기 결정 파라미터
MinPts
군집 형성 최소 포인트 수
밀도 판단 임계값
Core Point
반경 내 MinPts 이상 존재
군집 생성의 시작점
Border Point
핵심점 반경 내 일반 점
군집 가장자리 데이터
Noise Point
어떤 군집에도 미포함 점
이상치(Outlier) 식별
주요 특징 및 장점:
자율성: 군집의 개수(K)를 사전에 정할 필요 없이 자동으로 결정함
유연성: 기하학적인 모양(Non-spherical)의 군집도 효과적으로 탐색 가능
강건성: 이상치(Outlier)를 노이즈로 명확히 분류하여 제거하는 능력이 탁월함
인사이트: 데이터셋의 밀도가 불균일한 경우 성능이 저하될 수 있으므로, 하이퍼파라미터 최적화나 OPTICS 등 확장 모델과의 연계 검토가 필요함
