8. 시계열 딥러닝 (Time-series Deep Learning)

  • 정의: 시간 순서로 수집된 연속 데이터의 패턴·추세·계절성을 딥러닝 모델로 학습하여 예측·분류·이상 탐지를 수행하는 AI 기술
  • 키워드: 시계열, LSTM, Transformer, TCN, 자기상관, 트렌드·계절성, 이상 탐지, 예측, 시간 의존성
  • 분류: 신경망
  • 시계열 데이터 특성:
    항목설명핵심
    추세 (Trend)장기적 증가·감소 방향성장기 방향성
    계절성 (Seasonality)주기적으로 반복되는 패턴주기적 반복
    불규칙성 (Noise)예측 불가한 랜덤 변동랜덤 변동
    자기상관 (Autocorrelation)과거값이 현재값에 영향시간 의존성
    정상성 (Stationarity)통계적 특성이 시간에 무관학습 안정성
  • 주요 딥러닝 모델:
    구분모델설명핵심
    RNN 계열LSTM장기 의존성 학습·게이트 구조장기 패턴 학습
    GRULSTM 경량화·게이트 단순화경량 순환 모델
    Bidirectional LSTM과거·미래 양방향 시계열 학습양방향 학습
    합성곱 계열TCN (Temporal CNN)인과적 확장 합성곱 시계열 처리병렬 처리 우수
    WaveNet확장 합성곱 기반 음성·신호 생성음성·신호 처리
    트랜스포머Transformer자기 어텐션 기반 장기 의존성 포착전역 의존성
    Informer희소 어텐션·장문 시계열 효율 처리장문 예측 효율
    PatchTST시계열 패치 단위 Transformer 학습패치 기반 학습
    TimesNet1D 시계열을 2D 변환 후 CNN 처리2D 변환 처리
    하이브리드N-BEATS순수 딥러닝 기반 시계열 분해 예측분해 기반 예측
    TFT (Temporal Fusion Transformer)어텐션 + 게이트 기반 다변량 예측다변량 통합 예측
  • 핵심 기법:
    항목설명핵심
    슬라이딩 윈도우고정 크기 윈도우로 입력 시퀀스 생성시퀀스 구성
    시계열 분해추세·계절성·잔차 분리 후 학습성분 분리 학습
    차분 (Differencing)비정상 시계열을 정상 시계열로 변환정상성 확보
    다변량 입력여러 변수 동시 입력·상관관계 학습다변량 처리
    교사 강요 (Teacher Forcing)훈련 시 실제값을 다음 입력으로 사용학습 안정화
    어텐션 메커니즘중요 시점에 가중치 집중하여 예측중요 시점 포착
  • 태스크 유형:
    유형설명핵심
    예측 (Forecasting)미래 시점 값 예측 (단기·장기)미래 예측
    분류 (Classification)시계열 패턴별 클래스 분류패턴 분류
    이상 탐지 (Anomaly Detection)정상 패턴 대비 이상 시점 감지이상 감지
    세그멘테이션시계열 구간별 의미 단위 분할구간 분할
    결측값 보간누락된 시점 값 추정·복원결측 복원
  • 인사이트: 시계열 딥러닝은 LSTM 중심에서 Transformer 계열로 빠르게 전환되고 있으며, PatchTST·TimesNet 등 시계열 특화 Transformer가 기존 LSTM을 성능면에서 앞서고 있음. 단순 정확도보다 불확실성 정량화·해석 가능성·비정상 시계열 대응이 실무 적용의 핵심 과제이며, 다변량·장기 예측·이상 탐지를 통합하는 범용 시계열 Foundation Model 개발이 차세대 연구 방향으로 부상하고 있음
