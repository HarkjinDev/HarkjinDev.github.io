8. PEFT (Parameter-Efficient Fine-Tuning, 파라미터 효율적 파인튜닝)

  • 정의: 소수의 추가 파라미터 또는 일부 파라미터만 선택적으로 학습하여 적은 자원으로 특정 태스크에 효율적으로 적응시키는 파인튜닝 기법
  • 키워드: LoRA, QLoRA, Adapter, Prefix Tuning, 파라미터 동결(Freeze), 경량 파인튜닝, 저랭크 근사, 양자화, 메모리 효율, 멀티태스크 적응
  • 분류: 생성형 AI / LLM
  • 등장배경: 전체 파인튜닝 한계, 메모리 한계, 과적합 위험, 멀티태스크 비효율
  • 주요 기법 상세:
    기법핵심 원리학습 파라미터 비율특징
    LoRA가중치 행렬을 저랭크 행렬(A×B)로 근사하여 추가 학습~0.1~1%추론 시 원본 가중치에 병합 가능, 지연 없음
    QLoRALoRA + 4bit NormalFloat 양자화로 메모리 절감~0.1~1%소비자급 GPU(24GB)에서 65B 모델 파인튜닝 가능
    AdapterTransformer 레이어 사이에 소형 병목 MLP 모듈 삽입~1~4%모듈 교체로 멀티태스크 전환 용이
    Prefix Tuning입력 앞에 학습 가능한 가상 토큰 벡터(Prefix) 추가~0.1%모델 가중치 전혀 변경 안 함
    Prompt Tuning소프트 프롬프트 임베딩 벡터만 학습매우 소량모델 규모 클수록 효과적
    IA³Attention·FFN의 활성화 값에 학습 가능한 스케일 벡터 곱~0.01%추론 비용 증가 없음, 극소 파라미터
  • 인사이트: PEFT는 LLM 파인튜닝의 민주화를 실현한 핵심 기술로, 대형 AI 연구소만 가능했던 모델 특화 작업을 개인·스타트업 수준에서도 가능하게 만든 패러다임 전환점.멀티태스크 환경에서 PEFT의 진가가 발휘됨 — 태스크별 경량 어댑터(수 MB)만 교체하면 되므로, 수백억 파라미터 모델을 태스크마다 별도 유지하는 비효율을 근본적으로 해소.
