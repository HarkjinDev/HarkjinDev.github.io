3. 딥러닝 (Deep Learning)

  • 정의: 인간의 뇌 구조를 모방한 인공신경망(ANN)을 다층(Deep)으로 쌓아 데이터로부터 고차원적인 특징을 자동 학습하는 기술
  • 키워드: 다층 신경망(DNN), 가중치(Weight), 순전파, 역전파, 손실계산, 활성화 함수, 손실 함수, 옵티마이저, 배치 정규화, 드롭아웃, CNN, RNN/LSTM, Transformer, GAN, 과접합, 기울기 소실, 블랙박스, 데이터 의존성
  • 분류: AI 개론
  • 동작 원리: 입력 → 순전파(Forward) → 역전파(Backprop) → 출력
    항목설명핵심
    순전파 (Forward)입력 데이터가 각 레이어를 거쳐 출력값 생성예측 수행
    손실 계산예측값과 실제값의 차이를 손실 함수로 측정오차 산출
    역전파 (Backprop)손실을 역방향으로 전파하여 기울기 계산기울기 산출
    가중치 최적화옵티마이저(SGD·Adam 등)로 가중치 업데이트학습 수렴
  • 주요 구성 요소:
    항목설명핵심
    활성화 함수비선형성 부여 (ReLU·Sigmoid·Softmax 등)비선형 변환
    손실 함수예측 오차 측정 (MSE·Cross Entropy 등)오차 측정
    옵티마이저가중치 업데이트 방향·크기 결정 (Adam·SGD)학습 최적화
    배치 정규화레이어 입력 분포를 정규화하여 학습 안정화학습 안정
    드롭아웃학습 중 뉴런 일부를 무작위 비활성화하여 과적합 방지정규화
  • 주요 아키텍처:
    항목설명핵심
    CNN합성곱 연산으로 이미지·공간 특징 추출 특화이미지 처리
    RNN·LSTM순환 구조로 시퀀스·시계열 데이터 처리 특화순서 학습
    Transformer어텐션 메커니즘 기반 병렬 처리 구조현재 주류
    GAN생성자·판별자 경쟁 학습으로 데이터 생성생성 모델
  • 주요 문제점 및 대응:
    항목설명핵심
    과적합학습 데이터에 과도 최적화되어 일반화 실패드롭아웃·정규화
    기울기 소실역전파 시 기울기가 소멸되어 학습 불가ReLU·잔차연결
    블랙박스내부 추론 과정의 해석 불가XAI 연계
    데이터 의존성대규모 고품질 데이터 없이 성능 확보 어려움전이학습 활용
  • 인사이트: 특징 자동 추출(Representation Learning)로 기존 머신러닝의 Feature Engineering 한계를 극복하며 AI 성능의 패러다임 전환을 이끔. CNN→RNN→Transformer로 이어지는 아키텍처 진화는 각각 공간·시간·전역 의존성 문제를 해결해 왔으며, 현재 Transformer가 거의 모든 도메인의 표준 구조로 자리잡음
