7. 양자 머신러닝 (QML, Quantum Machine Learning)

  • 정의: 양자 컴퓨팅의 중첩·얽힘·양자 간섭 특성을 활용하여 머신러닝 알고리즘의 학습·최적화·추론 속도를 기하급수적으로 향상시키는 양자-고전 융합 컴퓨팅 기술
  • 키워드: 양자 중첩, 양자 얽힘, 큐비트, VQC, 양자 어드밴티지, PQC, 양자 커널, NISQ, HHL 알고리즘
  • 분류: AI 응용 / 트렌드
  • 양자 컴퓨팅 핵심 원리:
    항목설명핵심
    중첩 (Superposition)큐비트가 0과 1을 동시에 존재하는 상태병렬 계산 가능
    얽힘 (Entanglement)두 큐비트가 거리에 상관없이 상태를 공유초고속 상관 계산
    간섭 (Interference)정답 경로 강화·오답 경로 소거로 최적해 탐색확률 조작
    측정 (Measurement)중첩 상태를 0 또는 1로 확정하는 관측 행위결과 추출
  • 핵심 기술:
    구분항목설명핵심
    양자 모델VQC (변분 양자 회로)파라미터화된 양자 회로로 ML 모델 구현양자 신경망
    QNN (양자 신경망)양자 회로 기반 신경망 레이어 구성양자 딥러닝
    양자 커널 방법양자 특징 공간에서 SVM 커널 계산 가속양자 SVM
    양자 알고리즘HHL 알고리즘선형방정식 지수적 가속 풀이선형계산 가속
    양자 PCA고차원 데이터 주성분 분석 양자 가속차원 축소 가속
    QSVM양자 커널로 분류 문제 고속 처리분류 가속
    양자 볼츠만 머신양자 터널링 기반 확률 분포 학습생성 모델
    하이브리드하이브리드 양자-고전양자 회로 + 고전 최적화 반복 결합현실적 구현
    NISQ 알고리즘현재 노이즈 있는 중간 규모 양자 장치 활용현재 적용 가능
  • 인사이트: QML은 양자 컴퓨팅과 ML의 융합으로 특정 계산 문제에서 고전 ML의 한계를 초월하는 잠재력을 지니나, NISQ 시대의 오류율·큐비트 수 한계로 실질적 양자 어드밴티지 입증은 아직 초기 단계임. 하이브리드 양자-고전 접근이 현재의 현실적 구현 경로이며, 오류 정정 기술 발전과 큐비트 확장이 QML 실용화의 핵심 선결 과제로 2030년대 본격 부상이 전망됨
