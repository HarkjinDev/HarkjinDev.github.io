3. 앙상블 (Ensemble) 기법

정의: 여러 개의 단일 모델(약한 학습기)을 결합하여 더 강력하고 안정적인 예측 모델을 만드는 기법
키워드: 부트스트랩, 보팅, 배깅, 부스팅, 스태킹, 오차 보정, 편향-분산 트레이드오프, 블랙박스
분류: 지도학습
주요 기법:
항목
설명
핵심
보팅 (Voting)
여러 모델의 예측값을 투표로 결정
하드/소프트 보팅 방식
배깅 (Bagging)
병렬로 복제 데이터 학습
분산 감소, 랜덤 포레스트
부스팅 (Boosting)
오답에 가중치를 부여해 순차 학습
편향 감소, 높은 정확도
스태킹 (Stacking)
예측 결과들을 다시 학습 데이터로 활용
메타 모델 기반 결합
장단점:
구분
내용
장점
단일 모델 대비 높은 정확도, 과적합 방지, 이상치에 강인함
단점
학습 시간 증가, 모델 해석의 어려움(Black Box), 높은 연산 비용
인사이트: 배깅은 분산(Variance)을 줄여 과적합을 방지하고, 부스팅은 편향(Bias)을 줄여 정확도를 높이는 데 유리함. 성능 향상만큼 모델이 복잡해지므로 하이퍼파라미터 튜닝을 통한 최적화가 필수적임.
