1. 트랜스포머 (Transformer)

정의: 오직 Attention 메커니즘만을 사용하여 입력 데이터 전체의 관계를 한 번에 파악하고 병렬 처리를 가능하게 한 딥러닝 아키텍처
키워드: Self-Attention, Multi-head Attention, Positional Encoding, Scaled Dot-Product
분류: 생성형 AI/LLM
주요 특징: 병렬 처리 가능, 장거리 의존성 해결(Long-term Dependency)
구성요소:
구성 요소
설명
핵심
Self-Attention
입력 문장 내 단어 간의 관계를 직접 계산
문맥적 의미 파악
Multi-Head Attention
여러 개의 Attention을 병렬로 수행
다양한 관점의 정보 추출
Positional Encoding
순서 정보가 없는 모델에 위치 정보를 부여
단어의 순차적 위치 학습
Layer Normalization
각 층의 출력을 정규화하여 학습 안정화
Gradient Vanishing 방지
Residual Connection
입력값을 출력값에 더해 정보 손실 최소화
층이 깊어져도 학습 원활

분류
모델 사례
특성 및 용도
Encoder-Only
BERT, RoBERTa

Decoder-Only
GPT, Llama, Claude
문장 생성, 대화형 AI (현재 LLM 주류)

인사이트: 현재 모든 LLM(GPT, Claude, Gemini)의 근간이 되는 아키텍처이며, 비전(ViT) 및 오디오 분야로도 확장됨
