11. RLHF / RLAIF

  • 정의: 인간(RLHF) 또는 AI(RLAIF)의 피드백을 기반으로 강화학습하여 LLM의 응답 품질·안전성·유용성을 인간 선호에 맞게 정렬하는 기법
  • 키워드: 보상 모델, PPO, DPO, 인간 선호 정렬, AI 안전성, Constitutional AI, SFT, 선호도 데이터, Alignment
  • 분류: 생성형 AI / LLM
  • RLHF vs RLAIF 비교:
    항목RLHFRLAIF
    피드백 주체인간 어노테이터AI 모델
    비용·속도높은 비용·느림낮은 비용·빠름
    품질높은 신뢰도모델 편향 위험
    확장성인력 의존으로 한계대규모 확장 가능
    대표 사례GPT-4, ClaudeClaude (Constitutional AI)
  • RLHF 학습 파이프라인:
    단계설명핵심
    1. SFT고품질 지시문-응답 쌍으로 기본 모델 파인튜닝기본 정렬
    2. 선호도 데이터 수집동일 질문에 대한 응답 쌍을 인간이 비교·순위화선호 레이블링
    3. 보상 모델 학습선호도 데이터로 응답 품질을 점수화하는 모델 학습품질 평가기
    4. PPO 강화학습보상 모델 피드백으로 LLM 정책을 반복 최적화정책 최적화
  • 주요 기술:
    항목설명핵심
    보상 모델 (RM)응답의 유용성·안전성·정확성을 수치로 평가하는 모델품질 점수화
    PPO보상 신호로 LLM 정책을 안정적으로 최적화하는 강화학습 알고리즘정책 학습
    DPO보상 모델 없이 선호도 데이터로 직접 정렬하는 경량화 대안단순화 정렬
    Constitutional AIAI 스스로 원칙에 따라 응답을 비판·수정하는 RLAIF 방식AI 자기 정렬
    KL 발산 제약정렬 과정에서 원본 모델과 너무 멀어지지 않도록 제약과도 최적화 방지
  • 인사이트: RLHF는 LLM을 단순 언어 모델에서 인간 친화적 어시스턴트로 전환한 핵심 기술이나, 인간 피드백의 비용·편향 한계를 극복하기 위해 RLAIF·DPO로 빠르게 진화하고 있으며, "어떤 가치를 정렬할 것인가"라는 AI 안전성·윤리 설계가 기술만큼 중요한 핵심 과제로 부상하고 있음
