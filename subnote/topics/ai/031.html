5. GAN (Generative Adversarial Network, 생성적 적대 신경망)

정의: 생성자(Generator)와 판별자(Discriminator)가 서로 경쟁하며 학습하여 실제 데이터와 구별하기 어려운 새로운 데이터를 생성하는 비지도학습 기반 생성 모델
키워드: 생성자, 판별자, 적대적 학습, 잠재 벡터(Latent Vector), 모드 붕괴(Mode Collapse, 특정결과물만 생성하는 현상), 균형점(nash equilibrium), 이미지 생성, 데이터 증강
분류: 비지도학습
동작 원리:
구성 요소
역할
목표
생성자 (Generator)
랜덤 노이즈(z)로부터 가짜 데이터 생성
판별자를 속이는 데이터 생성
판별자 (Discriminator)
입력 데이터가 진짜·가짜인지 구분
진짜·가짜를 정확히 판별
학습 방식
두 모델이 서로 적대적으로 경쟁하며 동시에 성능 향상
균형점(Nash Equilibrium) 도달
주요 유형: DCGAN, cGAN, CycleGAN, StyleGAN, Pix2Pix
DCGAN (Deep Convolutional GAN):
항목
설명
핵심 구조
생성자에 전치 합성곱(Transposed Conv), 판별자에 일반 Conv 사용
주요 개선
Pooling 대신 Strided Conv 사용, 배치 정규화(Batch Norm) 적용
활성화 함수
생성자: ReLU, 판별자: Leaky ReLU 사용
의의
GAN의 불안정한 학습 문제를 CNN 구조로 크게 개선, 이후 GAN 변형 모델의 기반
장단점:
구분
내용
장점
고품질·고해상도 이미지 생성 가능
레이블 없이 비지도학습으로 데이터 분포 학습
데이터 증강, 스타일 변환 등 다양한 응용 가능
단점
모드 붕괴·학습 불안정으로 학습 난이도 높음
생성 품질 평가 기준이 주관적
Diffusion 모델 등장 이후 이미지 생성 분야에서 주도권 이동
인사이트: GAN은 생성 모델 패러다임을 개척한 핵심 기술이며, DCGAN을 시작으로 StyleGAN·CycleGAN 등 다양한 변형 모델로 발전하며 이미지 생성·편집·증강 분야를 혁신함. 최근에는 Diffusion 모델에 주도권을 내줬으나, 실시간 이미지 생성·변환 등 속도가 중요한 영역에서는 여전히 경쟁력을 유지하고 있음
