2. 활성화 함수 (Activation Function)

  • 정의: 가중치가 적용된 입력값의 합이 어느 정도의 신호로 출력될지를 결정하는 함수
  • 키워드: 기울기 소실, 비선형 변환, Sigmoid, Tanh, ReLU, Leaky ReLU, Softmax
  • 분류: 신경망
  • 사용 이유: 신경망에 비선형성을 부여하여 깊은 층에서도 복잡한 연산이 가능하게 함
  • 주요 유형:
    함수명설명 및 특징수식 (Formula)핵심
    Sigmoid출력을 0~1 사이로 제한함\sigma(x) = \frac{1}{1 + e^{-x}}기울기 소실(Vanishing Gradient) 발생
    Tanh출력을 -1~1 사이로 제한함\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}출력의 중심이 0 (Zero-centered)
    ReLU0 이하값은 0, 양수는 그대로 출력f(x) = \max(0, x)연산이 매우 빠르며 가장 대중적임
    Leaky ReLU음수 영역에 작은 기울기를 부여함f(x) = \max(0.01x, x)죽은 ReLU(Dying ReLU) 현상 해결
    Softmax모든 출력 합이 1이 되도록 정규화\sigma(z)_i = \frac{e^{z_i}}{\sum e^{z_j}}다중 클래스 분류의 출력층에 사용
  • 인사이트: 활성화 함수 선택 = 기울기 흐름의 통로 확보. 층이 깊어질수록 ReLU 변형(Leaky ReLU, ELU) 등을 사용하여 학습 안정성을 높이는 전략이 중요함
