1. XAI (eXplainable AI, 설명 가능한 인공지능)

  • 정의: AI가 내린 판단 결과에 대해 '왜 그런 결과가 나왔는지' 인간이 이해할 수 있도록 근거를 제시하는 기술
  • 키워드: 블랙박스 해소, 투명성, 신뢰성, 기여도 분석, 해석 가능성, SHAP, LIME, 대리 모델, 특징 중요도, AI 윤리, 모델 감사(Audit)
  • 분류: AI 응용/트렌드
  • 필요성: AI의 블랙박스(Black-box) 특성 해결, 신뢰성 확보, 책임 소재 파악, 편향 탐지, 규제 요구
  • 해석 가능성의 분류 체계:
    분류 기준유형설명
    설명 범위전역(Global)모델 전체의 동작 방식을 설명
    지역(Local)개별 예측 결과 하나에 대해 설명
    적용 시점사전(Ante-hoc)설계 단계부터 해석 가능한 모델 사용 (화이트박스)
    사후(Post-hoc)학습 완료된 모델에 외부적으로 설명 부여
    모델 의존성모델 특정적특정 모델 구조에 종속된 설명 기법
    모델 비종속적어떤 모델에도 적용 가능한 범용 기법
  • 핵심 기법:
    기법설명특징
    SHAP게임 이론 기반의 Shapley Value를 활용해 기여도 산출이론적 견고함, 전역·지역 설명 가능
    LIME특정 예측 주변에 단순 대리 모델을 만들어 국소 설명모델 비종속적, 빠른 적용 가능
    Attention Map트랜스포머의 어텐션 가중치를 시각화해 토큰 파악NLP 모델의 문맥 파악 구조 확인
    대리 모델블랙박스 모델을 결정 트리 등 해석 가능 모델로 근사전역 설명 가능, 정확도 손실 주의
    Grad-CAMCNN의 경사도(Gradient)를 이용해 기여 영역 시각화이미지 분류 및 객체 탐지 특화
    PDP특정 변수 변화에 따른 전체 예측값의 평균 변화 시각화전역적 피처 영향력 파악 용이
    ICEPDP를 개별 샘플 단위로 세분화하여 선 그래프로 표시샘플 간 이질적 패턴 탐지 유용
  • 평가 기술: <설희누가>
    평가 기술설명핵심
    Descriptive Accuracy (설명의 정확성)중요 피처 제거 시 예측 변화 관찰피처 추출의 정확성 반영
    Sparsity Accuracy (희소성 정확도)가중치가 0인 피처의 비중 측정설명의 희소성 및 간결성
    Cumulative DA (누적 설명 정확도)상위 피처 누적 제거 시 정확도 하락 측정중요 특징의 조기 발견 능력
    XAI 가중치 기반 평가피처 중요도와 정확도 하락의 상관관계 평가기여도 점수의 신뢰성 검증
  • 인사이트: 의료, 금융, 국방 등 중대한 결정이 필요한 분야에서 AI 도입을 위한 필수 요건(AI 윤리와 연계 가능)
