5. RNN (Recurrent Neural Network, 순환 신경망)

  • 정의: 이전 시점의 hidden state를 현재 입력과 함께 처리하여 순서·시간적 의존성을 학습하여, 시퀀스 데이터 처리에 특화된 모델
  • 키워드: 순환 구조, Hidden State, 시퀀스 학습, BPTT, 기울기 소실, 장기 의존성, LSTM, GRU(경량 LSTM), 시계열
  • 분류: 딥러닝 / 시퀀스 모델
  • 동작 원리:
    항목설명핵심
    Hidden State이전 시점 정보를 현재 입력과 결합하여 갱신기억 전달
    가중치 공유모든 시점에서 동일한 가중치를 반복 사용파라미터 효율
    BPTT(Backpropagation Through Time)시간 역방향으로 펼쳐 기울기를 역전파하는 학습 방식시간 역전파
  • 주요 문제점:
    항목설명핵심
    기울기 소실시퀀스가 길어질수록 초기 기울기가 소멸되는 현상장기 학습 불가
    기울기 폭발기울기가 지수적으로 증가하여 학습 불안정 발생학습 불안정
    장기 의존성 한계멀리 떨어진 시점 간의 관계 학습이 어려움단기 기억
  • LSTM (Long Short-Term Memory):
    항목설명핵심
    등장 배경RNN의 기울기 소실·장기 의존성 문제 해결을 위해 고안기울기 소실 해결
    Cell State장기 정보를 별도 경로로 유지하는 핵심 메모리장기 기억
    Forget Gate이전 Cell State에서 불필요한 정보를 제거정보 삭제
    Input Gate현재 입력 중 저장할 정보를 선택정보 선택
    Output GateCell State에서 출력할 정보를 결정정보 출력
    GRULSTM을 단순화한 경량 변형, 게이트를 2개로 축소경량 LSTM
  • 인사이트: RNN은 시퀀스 데이터 처리의 기반 아키텍처로, LSTM·GRU로 발전하며 NLP·시계열 분야를 주도했으나 Transformer의 등장으로 NLP 주류에서 대체됨. 그러나 경량성과 온디바이스 적용 용이성으로 인해 IoT·엣지 환경의 시계열 처리 영역에서는 여전히 실용적 가치를 유지하고 있음
