1. 인공신경망 (ANN)

  • 정의: 생물학적 뉴런의 동작 원리를 모방하여 입력 신호의 가중치 합을 활성화 함수를 통해 출력하는 수학적 모델
  • 키워드: 입력층/은닉층/출력층, 가중합, 활성화 함수, 임계값, 비선형성, XOR 문제, MLP, 역전파, 범용 근사자
  • 분류: 신경망
  • 상위-하위 관계: 퍼센트론(단일 뉴런) → 다층 퍼셉트론(MLP) → ANN → DNN
  • 핵심 구성 요소:
    항목설명핵심
    가중치 (Weight)입력 신호가 결과에 주는 영향력 조절 매개변수영향력 조절
    편향 (Bias)뉴런 활성화 민감도를 조절하는 임계값활성화 조절
    활성화 함수가중합을 비선형 신호로 변환하여 전달비선형 변환
    입력층원시 데이터를 받아 신경망에 전달하는 레이어데이터 입력
    은닉층특징을 추출·변환하는 중간 처리 레이어특징 추출
    출력층최종 예측값을 출력하는 레이어결과 출력
  • 퍼셉트론 (Perceptron):
    항목설명핵심
    개념입력값의 가중합을 임계값과 비교해 0·1 출력하는 최소 신경망 단위단일 뉴런
    단층 퍼셉트론선형 분류만 가능, AND·OR 문제 해결 가능선형 분류
    XOR 문제단층 퍼셉트론으로 XOR 비선형 분류 불가선형 한계
    한계비선형 문제 해결 불가로 1차 AI 겨울의 원인이 됨표현력 한계
  • 다층 퍼셉트론 (MLP, Multi-Layer Perceptron):
    항목설명핵심
    개념퍼셉트론을 입력·은닉·출력층으로 다층 구성한 신경망다층 구조
    XOR 해결은닉층 추가로 비선형 경계 학습 가능해져 XOR 해결비선형 극복
    역전파 결합역전파 알고리즘으로 다층 가중치 학습 가능해짐학습 가능화
    범용 근사자은닉층이 충분하면 어떤 연속 함수도 근사 가능딥러닝 이론 근거
  • XOR 문제: 초기 인공신경망(단층 퍼셉트론)은 데이터를 오직 직선 하나로만 분류할 수 있어, 값이 서로 다를 때만 참(1)이 되는 교차 패턴의 데이터(XOR)를 결코 나눌 수 없는 문제
  • 인사이트: 초기 퍼셉트론의 XOR 문제를 다층 구조(MLP)와 역전파 알고리즘으로 극복하며 딥러닝 시대를 열었으며, 단순한 뉴런 하나의 한계가 오히려 다층 신경망 발전의 동기가 됨. MLP의 범용 근사 정리(Universal Approximation Theorem)는 딥러닝의 이론적 근거가 되며, 층을 깊게 쌓을수록 더 복잡한 함수를 효율적으로 표현할 수 있음이 실증적으로 확인됨
