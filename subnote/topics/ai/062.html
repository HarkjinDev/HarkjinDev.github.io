9. LoRA (Low-Rank Adaptation)

  • 정의: 파라미터의 가중치 중 낮은 순위 가중치만 미세조정하는 개선된 파인튜닝 기법
  • 키워드: 저랭크 근사, 가중치 동결, 랭크(r), 스케일링 계수(α), 가중치 병합, PEFT, 어댑터, 경량 파인튜닝
  • 분류: 생성형 AI / LLM
  • 핵심 원리:
    단계동작 원리설명
    1. Parameter 분해Low Rank Matrix 생성W \approx A \times B, 저차원 행렬 표현
    2. 학습 대상 설정기존 W 고정, Matrix 업데이트원본 파라미터 미 변경
    3. 미세 조정 수행Matrix A/B 학습신규 태스크에 특화 학습
    4. 추론 시 통합기존 W + Matrix A/B원본 유지 + 미세 조정 반영
  • QLoRA (Quantized LoRA):
    항목설명
    핵심 원리LoRA + 4bit NormalFloat(NF4) 양자화를 결합하여 메모리 사용량 대폭 절감
    이중 양자화양자화 상수 자체도 재양자화하여 추가 메모리 절감
    페이지드 옵티마이저GPU 메모리 부족 시 CPU RAM으로 자동 오프로드하여 OOM 방지
    효과소비자급 GPU(24GB)에서 65B 모델 파인튜닝 가능
  • 인사이트: LoRA는 현재 PEFT의 사실상 표준(De-facto Standard)으로, HuggingFace 생태계와의 통합으로 진입 장벽이 크게 낮아짐. QLoRA·DoRA 등 다양한 변형 기법의 기반이 되며, 멀티태스크 환경에서 경량 어댑터 교체 방식의 핵심 기법으로 자리잡음
