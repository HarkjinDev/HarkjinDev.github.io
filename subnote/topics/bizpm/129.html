[하위] 4-1. 고객 여정 맵(Customer Journey Map)

  • 정의: 고객이 제품·서비스를 인지하는 순간부터 구매·사용·이탈까지 모든 접점(Touchpoint)에서의 경험·감정·행동을 시각화하여 고객 중심의 경험을 설계·개선하는 전략 도구
  • 키워드: 접점(Touchpoint), 감정곡선, 페르소나, 고객 감정, Peak Moment, 고통점(Pain Point), MOT(진실의순간), Drop-off point
  • 분류: 마케팅/트렌드
  • 구성요소:
    구분항목설명핵심
    기반페르소나목표 고객의 특성·목표·행동 패턴 정의고객 대표 모델
    시나리오페르소나가 경험하는 구체적 상황 설정맥락 설정
    여정단계 (Stage)고객 여정의 주요 단계 구분여정 구조화
    접점 (Touchpoint)고객과 브랜드가 만나는 모든 채널·순간상호작용 지점
    채널 (Channel)접점이 발생하는 온·오프라인 매체경험 매체
    경험고객 행동각 단계에서 고객이 하는 구체적 행동행동 패턴
    고객 감정각 접점에서 느끼는 감정·만족도 수준감정 곡선
    고통점 (Pain Point)고객이 불편·불만족을 느끼는 지점개선 우선순위
    기회기회 영역Pain Point 해결·경험 향상 가능 지점혁신 포인트
    내부 프로세스고객 경험을 지원하는 내부 활동운영 연계
  • 절차: 페르소나 정의 -> 여정 단계 설정 -> 접점 식별 -> 고객 행동·감정 기록 -> Paint point 도출 -> 기회 영역 정의 -> 실행·검증
  • 감정 곡선 분석:
    항목설명핵심
    Peak Moment고객 만족이 최고조에 달하는 긍정 경험경험 강화 포인트
    MOT (진실의 순간)브랜드 인상이 결정되는 핵심 접점집중 관리
    Pain Point감정 곡선이 하락하는 불만족 구간개선 우선순위
    Drop-off Point고객이 여정을 이탈하는 지점이탈 방지 대상
