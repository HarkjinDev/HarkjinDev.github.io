8. 디지털 플랫폼 정부

정의: 데이터와 플랫폼 기반으로 국민·기업·정부가 함께 참여하여 서비스를 만드는 혁신적 전자정부 모델
키워드: 국민 참여, 마이데이터, 원스톱 서비스, API 연계, 공공데이터 개방
분류: IT 전략/기획
핵심 구성요소:
공공 플랫폼: 마이데이터, 공공 API 허브
국민 참여: 정책 설계 단계부터 국민 공동 창출
데이터 연계: 기관 간 데이터 실시간 연계·공유
원스톱 서비스: 생애주기별 통합 서비스 제공
인사이트: 디지털 플랫폼 정부는 기존 전자정부의 단방향 서비스 제공에서 국민 참여형 공동 서비스 생산으로 패러다임을 전환한 모델임.
