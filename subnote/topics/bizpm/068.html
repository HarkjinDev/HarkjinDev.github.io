[하위] 4-1. CPM (Critical Path Method)

정의: 프로젝트 활동을 네트워크 다이어그램으로 표현하여 주경로(Critical Path)와 여유시간을 분석하는 일정 관리 기법
키워드: 주경로, Float(여유시간), ES, EF, LS, LF, 네트워크 다이어그램
분류: 프로젝트 관리
주경로(Critical Path): Float = 0인 활동들의 연속 → 지연 시 프로젝트 전체 지연
절차:
액티비티 정의 → 액티비티 수행기간 추정 → 네트워크 다이어그램 작성 → Forward 계산(ES, EF) → Backward(LS, LF) → Float 계산 → 주경로 분석 → 프로젝트 수행기간 추정
주경로 도출 방법:
자원 제한 시: CCPM 또는 자원 평준화(Resource Leveling) 적용
인사이트: CPM의 핵심은 주경로에 집중 자원 투입이며, Float > 0인 활동의 자원을 주경로로 재배치하는 Crashing 기법과 연계함.
