[하위] 6-1. 더블 다이아몬드 모델 (Double Diamond Model)

  • 정의: 발산(Diverge)·수렴(Converge)을 두 번 반복하여 올바른 문제 정의와 올바른 해결책을 도출하는 디자인 싱킹 방법론
  • 키워드: 발산·수렴, Discover·Define·Develop·Deliver, 디자인 싱킹, 문제 정의, 공감, 프로토타입
  • 분류: 경영전략 분석
  • 4단계 프로세스:
    다이아몬드단계방향설명핵심
    1차 (문제 공간)1. Discover (발견)발산 ↗사용자 공감·현장 관찰·탐색공감·탐색
    2. Define (정의)수렴 ↘인사이트 분석·핵심 문제 정의문제 재정의
    2차 (해결 공간)3. Develop (개발)발산 ↗다양한 아이디어·프로토타입 실험아이디어 탐색
    4. Deliver (전달)수렴 ↘최적 해결책 검증·출시해결책 구현
