4. CRUD 매트릭스 (CRUD Matrix)

정의: 엔터티 타입과 단위 프로세스 간 CRUD(Create/Read/Update/Delete) 관계를 매핑하여 데이터 모델의 완전성을 검증하는 도구
키워드: 생성(Create), 조회(Read), 갱신(Update), 삭제(Delete), 엔티티 타입, 단위 프로세스, 완전성 검증
분류: 데이터 모델링
구성 요소: <엔단>
엔티티 타입: 프로세스에 영향 주는 데이터 (고객, 제품, 주문, 목록)
단위 프로세스: 엔티티에 영향주는 업무 단위 (고객 등록, 주문 조회)
CRUD: 상태 표시 (생성, 조회, 갱신, 삭제)
활용 목적:
누락 검증: 생성(C)만 있고 삭제(D)가 없는 엔터티 등 업무 논리 오류 탐지
집중도 분석: 특정 프로세스에 CRUD가 집중되는 병목 포인트 식별
인사이트: CRUD 매트릭스는 데이터 모델 검토 시 빠진 업무 로직을 조기에 발견하는 품질 도구로, 설계 단계에서 반드시 수행해야 할 검증 절차임
