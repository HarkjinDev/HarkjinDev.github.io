[하위] 1-1. 동적/정적 SQL

정의:
정적 SQL(Static SQL): 컴파일 시점에 SQL 구문이 확정되어 실행계획을 미리 생성하고 재사용하는 방식 (Pre-compile)
동적 SQL(Dynamic SQL): 실행 시점에 SQL 구문을 문자열로 동적 생성하여 실행하는 방식 (Runtime Parse)
키워드: ISO/IEC 9075, 사전 컴파일, 고정 쿼리, 실행계획 재사용, SQL 인젝션 위험, 바인드 변수
