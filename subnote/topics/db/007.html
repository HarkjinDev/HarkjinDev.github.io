3. 관계형 데이터 모델 (Relational Data Model)

  • 정의: 데이터를 2차원 테이블(릴레이션) 형태로 표현하고, 관계 대수와 관계 해석을 통해 데이터를 조작하는 논리적 데이터 모델
  • 키워드: 릴레이션, 튜플, 속성, 도메인, 키, 정규화, 관계 대수
  • 분류: 데이터 모델링
  • 핵심 구성 요소:
    • 릴레이션(Relation): 행과 열로 구성된 2차원 테이블
    • 튜플(Tuple): 릴레이션의 각 행 (레코드)
    • 속성(Attribute): 릴레이션의 각 열 (컬럼)
    • 도메인(Domain): 속성이 가질 수 있는 원자값의 범위
  • 주요 특성: 튜플의 무순서성, 속성의 원자값, 중복 튜플 불허
  • 인사이트: 관계형 모델의 핵심 장점은 데이터 독립성과 집합 기반 연산이며, ACID 트랜잭션 보장으로 금융·ERP 시스템의 표준으로 자리잡음
