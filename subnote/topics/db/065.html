1. 데이터 거버넌스 (Data Governance)

정의: 조직의 데이터 자산을 정책, 표준, 조직 체계를 통해 체계적으로 관리하여 데이터 품질과 활용성을 보장하는 관리 체계
키워드: 원칙/조직/프로세스, 데이터 표준화, 품질관리, 메타데이터, 성숙도 5단계
분류: 데이터 거버넌스/품질
3대 구성 요소: <원조프>
원칙(Principle): 데이터 관리를 위한 지침과 방향성
조직(Organization): DA, DBA, 데이터 오너 등 역할과 책임 구조
프로세스(Process): 데이터 표준화, 품질관리, 보안 등 실행 절차
6대 관리 영역: 데이터 품질 관리, 메타 데이터 관리, 데이터 주기 관리, 데이터 보안, 데이터 표준화, 데이터 거버넌스 조직
성숙도 5단계: 초기→관리→정의→정량적관리→최적화
인사이트: 데이터 거버넌스는 기술적 과제가 아닌 조직 문화와 경영진 의지가 핵심이며, 데이터 오너십(Data Ownership) 명확화가 성공의 첫 번째 조건임
