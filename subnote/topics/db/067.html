2. 데이터 품질관리 (Data Quality Management)

  • 정의: 데이터의 정확성, 완전성, 일관성, 유효성을 체계적으로 측정하고 개선하는 활동
  • 키워드: 정확성, 완전성, 일관성, 유효성, 유일성, 적시성, ISO-8000, MDM, DQM
  • 분류: 데이터 거버넌스/품질
  • 품질관리 프레임워크:
  • 품질 기준 (6개 차원):
    • 정확성: 실세계 사실과 데이터 값이 일치하는 정도
    • 완전성: 필수 항목 누락 없이 채워진 정도
    • 일관성: 여러 시스템에 걸쳐 동일 데이터가 동일한 값을 가지는 정도
    • 유효성: 정의된 도메인/형식에 맞는 값이 저장된 정도
    • 유일성: 중복 없이 고유한 정도
    • 적시성: 필요한 시점에 최신 데이터가 제공되는 정도
  • 관련 표준: ISO 8000, ISO 25012, ISO 25024, MDM, DQM 프레임워크
  • 인사이트: 데이터 품질관리는 사후 교정보다 데이터 생성 시점의 예방적 관리(Data-at-Source)가 비용 효율적이며, DQ 인증 제도를 통한 공식화가 확산 중
