5. 데이터 표준화 (Data Standardization)

정의: 데이터의 명칭, 정의, 형식, 규칙을 전사적으로 통일하여 일관성을 확보하는 활동입니다.
키워드: 명칭/정의/형식/규칙 4원칙, ISO 8000, DRM, 데이터 사일로, 메타데이터
분류: 데이터 거버넌스/품질
데이터 표준화 4원칙: <명정형규>
명칭 표준화: 데이터 항목의 이름을 전사적으로 통일 (예: 고객ID vs 고객번호 → 단일화)
정의 표준화: 각 데이터 항목의 의미와 범위를 명확하게 정의
형식 표준화: 날짜, 코드, 숫자 등 데이터 형식을 통일 (예: YYYYMMDD)
규칙 표준화: 데이터 생성·변경·삭제에 관한 업무 규칙 통일
관련 표준 및 도구: ISO 8000, DRM(데이터 참조 모델), 메타데이터 관리 시스템
인사이트: 데이터 표준화는 단순한 네이밍 규칙이 아니라 조직 전체의 데이터 소통 언어를 통일하는 작업이며, 표준화 미흡 시 시스템 통합·연계 시 데이터 불일치와 사일로 현상이 반복적으로 발생함
