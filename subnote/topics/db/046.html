4. 다차원 색인구조 (Multidimensional Index Structure)

  • 정의: 2차원 이상의 공간 데이터를 효율적으로 탐색하기 위한 트리 기반 특수 색인 구조
  • 키워드: 차원의 저주, K-D Tree, Quad Tree, R-Tree, MBR, 공간 인덱스
  • 분류: 인덱스/튜닝
  • 색인구조 분류: PAM(다차원 점), SAM(다차원 공간 영역)
  • 주요 유형:
    • K-D Tree: 이진 트리 구조로 다차원 점 데이터 분할 — 정적 데이터에 적합
    • Quad Tree: 공간을 4분할(4개 사분면)하여 재귀 분할하는 구조
    • R-Tree: MBR(Minimum Bounding Rectangle) 기반, 동적 데이터에 강점
    • R+ Tree: R-Tree 변형, 중첩 영역 제거
    • R* Tree: R-Tree 개선, 영역 겹침 최소화
  • 인사이트: 차원이 증가할수록 K-D Tree 성능이 급격히 저하(차원의 저주)하므로, 고차원 공간 데이터에는 R* Tree나 VA-File 같은 대안적 방법이 권장됨
