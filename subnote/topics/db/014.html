1. SQL (Structured Query Language)

정의: 관계형 데이터베이스를 정의·조작·제어하는 비절차적 표준 질의 언어
키워드: DDL, DML, DCL, TCL, 선언형(What), 비절차형, 관계형 DB, ANSI 표준(ISO/IEC 9075:2023)
분류: SQL
SQL 구문 분류:
구분
역할
주요 명령어
DDL (Data Definition Language)
데이터 구조 정의
CREATE, ALTER, DROP, TRUNCATE
DML (Data Manipulation Language)
데이터 조작
SELECT, INSERT, UPDATE, DELETE
DCL (Data Control Language)
권한 제어
GRANT, REVOKE
TCL (Transaction Control Language)
트랜잭션 제어
COMMIT, ROLLBACK, SAVEPOINT
SQL 처리과정: <구실인>
인사이트: SQL은 선언형 언어(What)이며 절차형 언어(How)가 아니므로 실행 계획(Execution Plan)은 DBMS 옵티마이저가 결정함을 답안에 명시 권장, DDL은 자동 COMMIT이 수행되므로 ROLLBACK 불가 특성을 답안에 반드시 언급
