2. 벡터 데이터베이스 (Vector Database)

정의: 고차원 벡터(임베딩)를 저장하고 유사도 기반의 근사 최근접이웃(ANN) 검색을 수행하는 특수 목적 데이터베이스
키워드: 고차원 벡터, 임베딩, ANN, HNSW, IVF, 코사인 유사도, RAG 연동
분류: 특수 DB
메커니즘:
핵심 기술:
HNSW (Hierarchical Navigable Small World): 계층적 그래프 구조로 고속 ANN 검색 수행
IVF (Inverted File Index): 벡터를 클러스터링 후 역인덱스로 검색 범위를 좁혀 빠른 검색 수행
유사도 측정: ENN(Exact), ANN(근사), k-NN(k-최근접)
활용: RAG(검색 증강 생성)의 핵심 컴포넌트, AI 서비스 지식 베이스
인사이트: LLM과 RAG의 확산으로 벡터 DB 수요가 급증하고 있으며, Pinecone·Weaviate·Milvus 등 전용 솔루션과 pgvector(PostgreSQL 확장) 같은 하이브리드 접근도 주목받고 있음
