[하위] 4-2. 델타 레이크(Delta Lake)

  • 정의: 데이터 레이크의 단점을 보완, ACID 보장 및 메타데이터 관리를 통한 데이터 레이크하우스의 핵심 기술
  • Delta Lake 주요 기능:
    구분주요 기능설명
    신뢰성ACID Transaction추가·수정·삭제 시 원자성·일관성·격리성·지속성 보장
    Time Travel특정 시점 데이터 조회·복구·감사·이력 추적 지원
    품질 관리Schema Enforcement정의된 스키마 외 데이터 저장 방지·품질 확보
    Schema Evolution변경되는 데이터 구조 유연하게 반영·운영 편의성 향상
    운영 효율화MERGE·UPDATE·DELETECDC·Upsert 등 데이터 변경 작업 효율적 처리
    Batch & Streaming실시간·배치 데이터를 동일 저장 계층에서 통합 처리
    성능 최적화Compaction다수 작은 파일 통합·저장·조회 성능 향상
    Z-Ordering조회 빈도 높은 컬럼 기준 저장·검색 성능 향상
    활용성 향상Bronze 계층원천 데이터 저장·재처리·이력 추적 기반 제공
    Silver 계층정제·검증된 데이터 저장·분석 신뢰성 확보
    Gold 계층집계·가공 데이터 제공·AI·BI 분석 활용도 극대화
    Time Travel특정 시점 데이터 조회·복구·감사·이력 추적 지원
  • Delta Lake 특징:
    구분특징설명
    LakehouseDL + DW 통합데이터 저장·분석 기능을 단일 플랫폼에서 제공
    데이터 신뢰성정합성 보장·버전 관리분석 가능한 수준의 신뢰성 있는 데이터 제공
    데이터 확장성대용량 분산 처리 지원클라우드·빅데이터 환경에서 유연하게 확장 가능
    실시간 처리성Batch·Streaming 통합 처리실시간·배치 데이터를 동시에 처리 가능
    분석 활용성AI·BI 친화적 구조Bronze·Silver·Gold 계층으로 분석용 데이터 제공
    운영 효율성데이터 파이프라인 단순화데이터 관리·운영 복잡도 감소
