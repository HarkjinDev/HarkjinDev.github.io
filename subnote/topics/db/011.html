7. 암스트롱 공리 (Armstrong's Axioms)

  • 정의: 릴레이션 R에 대해 X, Y, Z라는 속성의 집합이 주어졌을 경우, 여러 가지 함수종속의 성질을 유도해 낼 수 있는 추론 규칙
  • 키워드: 함수적 종속(FD), 반사/확장/이행 공리, 폐포(closure), 정당성(sound), 완전성(complete)
  • 분류: 데이터 모델링
  • 특징: <폐정완>
    특징설명핵심
    폐포 (Closure, F+)FD 집합 F로부터 추론 가능한 모든 FD의 집합모든 FD 추론 집합
    정당성 (Soundness)F로부터 반드시 F+에 속하는 FD만 생성·잘못된 함수 종속 미생성올바른 FD만 생성
    완전성 (Completeness)세 법칙 반복 적용으로 F+의 모든 FD 생성 가능F+ 전체 생성 가능
  • 기본 규칙 (정당성): <반확이/재부이>
    구분원리내용
    반사공리재귀성규칙 (Subset Property): 부분집합의 성질Y ⊆ X이면, X → Y이다.
    확장공리부가성규칙 (Augmentation): 확대성질X → Y이면, XZ → YZ이다.
    이행공리이행성규칙 (Transitivity): 이행성 성질X → Y이고 Y → Z이면, X → Z이다.
  • 부가 규칙 (완전성): <분합의>
    구분원리내용
    분해규칙 (Decomposition)결합된 종속을 개별 종속으로 분리X → YZ이면, X → Y이고 X → Z이다
    합집합규칙 (Union)개별 종속을 하나의 결합 종속으로 통합X → Y이고 X → Z이면, X → YZ이다
    의사이행성규칙 (Pseudotransitivity)중간 속성에 추가 조건을 결합한 이행성 확장X → Y이고 WY → Z이면, WX → Z이다
  • 인사이트: 암스트롱 공리는 정당(sound)하며 완전(complete)한 성질을 이용하여 데이터베이스 정규화에 사용되며, 기본 3규칙(반사·확장·이행)으로부터 분해·합집합·의사이행성 규칙을 모두 유도할 수 있음
