[하위] 1-2. 제4정규형 (4NF)

정의: BCNF를 만족하면서 다치 종속성(MVD, Multi-Valued Dependency)을 제거한 정규화 단계
키워드: 다치종속 제거, MVD, 4NF, BCNF
분류: 정규화
다치 종속(MVD): 하나의 속성 값에 다른 두 속성이 각각 독립적으로 다중 값을 가지는 종속성 (A →→ B)
항목

다치 종속 (MVD)
X →→ Y: X 값에 대해 Y 값의 집합이 독립적으로 결정됨
독립적 다중 값
함수 종속과 차이
함수 종속은 1:1, 다치 종속은 1:多 독립 관계
1:多 독립 종속
발생 조건
하나의 테이블에 독립적인 두 다치 속성이 공존할 때
독립 다치 공존
자명한 MVD
X →→ Y이고 Y ⊆ X 이거나 X∪Y = 전체 속성인 경우
