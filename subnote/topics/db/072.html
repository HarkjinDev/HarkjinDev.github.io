7. 데이터 스페이스(Data Space)

  • 정의: 참여자 간 데이터 주권을 보장하면서 안전하게 데이터를 공유·거래할 수 있는 분산형 데이터 공유 인프라
  • 분류: 데이터 거버넌스/품질
  • 데이터 스페이스 핵심 원칙:
    원칙설명핵심
    데이터 주권데이터 제공자가 사용 조건·목적·범위 직접 통제제공자 통제권 보장
    분산성중앙 저장소 없이 참여자 간 직접 데이터 교환탈중앙화
    신뢰 프레임워크참여자 신원 인증·데이터 계약·감사 체계Trust Framework
    상호운용성표준 기반 이기종 시스템 간 데이터 교환개방 표준
    공정한 가치 배분데이터 제공·활용에 따른 공정한 보상 체계데이터 경제
  • 데이터 스페이스 구성요소:
    구성요소설명핵심
    데이터 제공자 (Provider)데이터를 공유·제공하는 참여자사용 조건 설정
    데이터 소비자 (Consumer)데이터를 활용·분석하는 참여자계약 기반 접근
    커넥터 (Connector)참여자 간 데이터 교환 표준 인터페이스IDS Connector
    메타데이터 브로커데이터 카탈로그·검색·발견 서비스데이터 탐색
    신원 제공자 (Identity Provider)참여자 신원 인증·인가 서비스PKI·DID 기반
    앱 스토어데이터 처리·분석 앱 제공 마켓플레이스서비스 생태계
    클리어링 하우스데이터 거래 로그·감사·분쟁 해결거래 투명성
    데이터 계약 (Data Contract)사용 조건, 과금 모델 등 명시데이터 표준 계약
    표준 프로토콜DCAT, ODRL(정책), GAIA-X표준표준 준수
  • 인사이트: 데이터 스페이스는 중앙집중형 플랫폼의 데이터 독점 한계를 극복하고 데이터 주권·신뢰·상호운용성을 보장하는 분산형 데이터 공유 생태계로, EU Gaia-X·IDSA 표준을 중심으로 산업·공공·의료 등 전 분야로 확산되는 차세대 데이터 거버넌스 패러다임
