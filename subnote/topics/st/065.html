1. 데이터 사이언스 (Data Science)

  • 정의: 데이터 분석을 통해 비즈니스 인사이트를 도출하는 융합 학문
  • 키워드: 수학, 통계, 컴퓨팅, 도메인 지식, 머신러닝
  • 분류: 시각화/빅데이터
  • 요구 역량 (벤다이어그램):
    • 수학/통계: 모델링, 가설검정, 확률론
    • 컴퓨팅: 프로그래밍, 데이터 엔지니어링
    • 도메인 지식: 비즈니스 이해, 문제 정의
    • → 세 영역의 교집합 = 데이터 사이언스
  • 인사이트: 수학·통계·컴퓨팅·도메인 지식의 융합. 단순 분석 기술을 넘어 비즈니스 문제 해결 역량이 핵심 차별점이다.
