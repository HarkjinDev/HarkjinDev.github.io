2. t-검정 (t-test)

  • 정의: t분포를 이용하여 모평균 또는 두 집단의 평균 차이를 검정하는 통계 기법
  • 키워드: t분포, 소표본, 모분산 미지(Unknown), 독립표본, 대응표본
  • 분류: 가설검정
  • t분포:
    • 모집단의 분산을 모를 때, 표본의 크기가 작을 때 (n < 30), 표본 평균을 통해 모집단의 평균을 추정하기 위해 사용하는 확률 분포
    • t = \frac{\bar{X} - \mu}{s / \sqrt{n}}
    • \bar{X}: 표본 평균, \mu: 모집단 평균, s: 표본 표준편차, n: 표본 크기 (df = n-1)
  • 유형: 독두대단 (독립표본·두 집단 / 대응표본·단일 집단)
    • 독립표본 t-검정: 두 독립 집단 평균 비교
    • 대응표본 t-검정: 동일 집단의 사전-사후 비교
    • 단일표본 t-검정: 단일 집단을 모집단 평균과 비교
  • 인사이트: t-검정은 모분산이 알려지지 않은 소표본(n≤30)에서 표준정규분포 대신 t분포 사용. n이 커질수록 t분포는 표준정규분포에 수렴한다.
