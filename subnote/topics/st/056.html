4. 텍스트 마이닝 (Text Mining)

  • 정의: 비정형 텍스트에서 의미 있는 정보와 패턴을 추출하는 분석 기법
  • 키워드: 형태소 분석, TF-IDF, LDA, N-gram, 키워드 추출
  • 분류: 데이터 마이닝
  • 주요 기술:
    • 형태소 분석: 텍스트를 의미 단위로 분해 (KoNLPy)
    • TF-IDF: 단어 중요도 계산 (빈도×문서 내 희소성)
    • 토픽 모델링: LDA(잠재 디리클레 할당)로 주제 추출
    • 감성 분석: 긍정/부정 분류
  • 인사이트: 비정형 텍스트를 정형 데이터로 변환하는 전처리가 핵심. 오피니언 마이닝은 텍스트 마이닝의 감성 분석 특화 분야다.
