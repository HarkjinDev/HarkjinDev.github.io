6. 음성 데이터 마이닝 (Voice Data Mining)

정의: 음성 데이터에서 패턴, 감성, 의미를 추출하는 분석 기법
키워드: STT, 피치, 에너지, 성조, 어조, 음성 감성 분석
분류: 데이터 마이닝
주요 기술:
STT(Speech-to-Text): 음성 → 텍스트 변환
음성 특성 분석: 피치, 에너지, 성조, 어조
감성 분석: 음성에서 감정 상태 추출
화자 인식: 화자 구분 및 특성 파악
발전 방향: wav2vec, Whisper 기반 딥러닝 고정밀 STT
인사이트: STT를 통한 텍스트 변환 후 텍스트 마이닝과 결합하는 하이브리드 방식이 활용된다. 콜센터 품질 관리, 보안 인증에 적용된다.
