6. 편향(Bias)

  • 정의: 데이터 수집·표본추출·학습·분석 과정에서 특정 방향으로 데이터 또는 분석 결과가 치우치는 현상
  • 키워드: 선택편향, 생존자편향, 확증편향, 알고리즘 편향, 이상치, SMOTE, 공정성, XAI
  • 분류: 통계 기초
  • 편향 유형 (발생단계별):
    발생단계편향 유형설명핵심
    수집·표본선택 편향특정 집단만 표본화→일반화 실패온라인 설문만 수집
    생존자 편향성공 사례만 관찰·실패 사례 누락폐업 기업 제외 분석
    측정 편향측정 도구·기준 일관성 부족평가자별 라벨링 기준 상이
    라벨링확증 편향가설 지지 데이터만 선별 라벨링주관적 라벨링
    역사적 편향과거 차별적 데이터 반영AI 채용 여성 불이익
    알고리즘알고리즘 편향모델 설계·목적함수에서 발생다수 집단 최적화·소수 차별
    인지인지 편향분석가 선입견 반영앵커링·가용성 편향
  • 편향 특징:
    특징설명핵심
    발생 범위데이터 수집~모델 배포 이후 전 과정전 수명주기
    오류 형태모델의 체계적 오류·일반화 실패랜덤 오류와 구별
    데이터 차원전체 분포(Distribution) 단위로 치우침분포 수준 왜곡
    AI 위험데이터 편향→모델 편향→사회적 차별로 증폭차별 증폭
  • 데이터 분석 결과에 미치는 영향:
    영향 영역이상치 영향편향 영향
    통계량평균·분산·표준편차 등 기술통계 왜곡추정량의 체계적 오차(Estimator Bias)
    모델 학습회귀계수 왜곡·손실함수 발산특정 집단 과적합·일반화 성능 저하
    예측 정확도MAE·RMSE 등 오차 지표 악화전체 정확도 양호·부분집단 성능 격차
    일반화테스트셋 성능 저하·재현성 부족Out-of-Distribution 환경 급격한 성능 저하
    의사결정잘못된 인사이트로 전략적 오류잘못된 정책·차별적 의사결정(AI 채용 차별)
    사회·법적분석 신뢰도·재현성 문제GDPR·EU AI Act 위반·차별·인권 침해 소송
  • 편향 탐지 기법:
    기법설명핵심
    왜도(Skewness) 측정데이터 분포의 비대칭도 확인분포 치우침 정량화
    카이제곱 검정모집단 분포와 표본 분포 정합성 검정범주형 편향 탐지
    Residual Analysis잔차 분포의 무작위성 검증체계적 오류 식별
    공정성 지표집단별 정확도·FPR·FNR 비교Demographic Parity
  • 편향 처리 방안:
    구분기법설명핵심
    데이터 수준재표본추출 (Resampling)언더샘플링·오버샘플링(SMOTE) 적용클래스 불균형 해소
    가중치 부여취약 집단 데이터에 가중치 부여소수집단 반영
    모델 수준규제화 (Regularization)L1/L2 규제로 모델 편향 통제과적합·편향 억제
    공정성 제약집단 간 예측 균등성 목적함수 추가Fairness Constraint
    사후 수준임계값 조정집단별 분류 임계값 개별 최적화Post-processing
    XAI 검증설명가능 AI로 편향 원인 사후 분석SHAP·LIME 활용
  • 이상치 vs 편향 비교:
    항목이상치 (Outlier)편향 (Bias)
    정의대다수와 현저히 다른 개별 데이터 포인트전체 분포가 체계적으로 치우친 현상
    발생 단위개별 데이터 포인트전체 분포(Distribution)
    오류 성격무작위적 오류 가능체계적·방향성 있는 오류
    탐지 방법IQR·Z-score·박스플롯왜도·카이제곱·잔차 분석
    처리 방법제거·대체·변환재표본추출·가중치·규제화
    AI 영향모델 학습 불안정차별적 의사결정·일반화 실패
  • 인사이트: 편향은 데이터 수집부터 모델 배포까지 전 과정에서 발생하며, AI 시대에는 데이터 편향→모델 편향→사회적 차별로 증폭되므로 탐지·처리·공정성 검증을 수명주기 전반에 걸쳐 통합 관리해야 함
