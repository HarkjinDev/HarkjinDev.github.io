4. 이상치 (Outlier)

정의: 데이터의 일반적 분포에서 크게 벗어난 비정상적 관측값
키워드: 이상치, 전역/맥락/군집 이상치, IQR, Z-score
분류: 통계 기초
주요 발생 원인: 데이터 오류, 측정 단위 불일치, 샘플링 오류, 비정상적 이벤트, 극단적인 행동
유형 (3가지):
전역 이상치(Global): 전체 데이터 관점에서 극단적으로 벗어난 값
맥락 이상치(Contextual): 특정 맥락에서만 비정상 (시계열 계절성 등)
군집 이상치(Collective): 개별로는 정상이나 집합 패턴이 비정상
탐지 방법:
구분
탐지 기법
설명
통계 기반
Z-score
평균과 표준편차를 기준으로 이상값 여부 판단 (±3σ\pm3\sigma±3σ 이상)
IQR (사분위수 범위)
Q1,Q3Q1, Q3Q1,Q3를 기준으로 IQR 범위를 벗어난 값을 이상치로 간주
시각화 기반
Boxplot
사분위수와 수염 범위를 시각화하여 이상값을 직관적으로 탐지
ML 기반
KNN
인접한 KKK개의 거리 평균이 긴 점을 이상치로 탐색
DBSCAN
군집 내 밀도가 낮거나 소속되지 않은 데이터를 이상값으로 식별
