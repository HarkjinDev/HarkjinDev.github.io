1. 확률분포(Probability distribution)

  • 정의: 확률변수가 취할 수 있는 모든 값과 그 값이 나타날 확률을 대응시킨 함수 관계
  • 키워드: 확률변수(Random Variable), 확률함수(PMF/PDF), 기댓값, 분산, 중심극한정리(CLT)
  • 분류: 확률분포
  • 통계적 추론: 얻은 데이터로부터 그 발생원이 어떤 확률분포를 취하는지 추정하는 것
  • 확률변수 (Random Variable):
    • 표본공간의 각 결과(사건)를 하나의 실수 값으로 매핑해주는 함수
    • 설명: 주사위를 던졌을 때 나오는 '눈'이나 사람의 '키'처럼, 어떤 실험의 결과를 숫자로 바꾸어 놓은 것
    • 표기: 보통 대문자 X로 표기하며, 그 변수가 가질 수 있는 구체적인 값은 소문자 x로 씀
  • 확률질량함수 (PMF, Probability Mass Function):
    • 이산확률변수 X가 특정 값 x를 가질 확률을 직접적으로 나타내는 함수
    • 특징:
      • 지점 확률: 함수값 자체가 해당 지점의 확률(예: 주사위 1이 나올 확률은 1/6)
      • 전체 합: 모든 확률을 더하면 반드시 1이 됨(\sum P(x) = 1)
  • 확률밀도함수 (PDF, Probability Density Function):
    • 연속확률변수 X가 어떤 구간에 속할 확률을 나타내기 위해 사용하는 함수
    • 특징:
      • 구간 확률: 특정 지점의 확률은 0이며, 일정 구간의 면적(적분)이 곧 확률이 됨
      • 전체 면적: 함수 아래의 전체 면적을 합치면 1이 됨 (\int f(x)dx = 1)
  • 확률분포 종류: <이질연밀 / 베이기다포 / 정지균티카프>
    • 이산확률분포: 확률변수가 취할 수 있는 값의 수가 유한한 분포
      분포특징 / 확률질량함수(PMF)기댓값(E(X)) / 분산(Var(X))
      베르누이 분포 X \sim Ber(p)- 성공, 실패 두 가지 결과 - f(x) = p^x(1-p)^{1-x}- E(X) = p - Var(X) = p(1-p)
      이항 분포 X \sim B(n, p)- 독립적 n번 반복 시 성공 횟수 - f(x) = \binom{n}{x} p^x(1-p)^{n-x}- E(X) = np - Var(X) = np(1-p)
      기하 분포 X \sim Geo(p)- 첫 번째 성공이 일어날 때까지의 시행 횟수 - f(x) = (1-p)^{k-1}p- E(X) = \frac{1}{p} - Var(X) = \frac{1-p}{p^2}
      다항 분포 X \sim Mult(n, p_1, \dots, p_k)- 결과가 3가지 이상인 독립 시행을 n번 반복 시 각 결과의 발생 횟수 - f(x_1, \dots, x_k) = \frac{n!}{x_1! \cdots x_k!} p_1^{x_1} \cdots p_k^{x_k}- E(X_i) = np_i - Var(X_i) = np_i(1 - p_i)
      포아송 분포 X \sim Pois(\lambda)- 일정한 시간/공간 내에서 발생하는 사건 개수 - f(x) = \frac{\lambda^x e^{-\lambda}}{x!}- E(X) = \lambda - Var(X) = \lambda
    • 연속확률분포: 어떤 범위에서 연속적인 값을 갖는 분포
      분포특징 / 확률밀도함수(PDF)기댓값(E(X)) / 분산(Var(X))
      정규 분포 X \sim N(\mu, \sigma^2)- 평균을 중심으로 좌우 대칭인 종 모양 - f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}(\frac{x-\mu}{\sigma})^2}- E(X) = \mu - Var(X) = \sigma^2
      지수 분포 X \sim Exp(\lambda)- 사건이 발생할 때까지의 대기 시간 - f(x) = \lambda e^{-\lambda x} \quad (x \ge 0)- E(X) = \frac{1}{\lambda} - Var(X) = \frac{1}{\lambda^2}
      균등 분포 X \sim U(a, b)- 모든 구간에서 확률밀도가 일정한 분포 - f(x) = \frac{1}{b-a} \quad (a \le x \le b)- E(X) = \frac{a+b}{2} - Var(X) = \frac{(b-a)^2}{12}
      t-분포 X \sim t(k)- 모집단의 표준편차를 모를 때 표본평균 검정 - 표본 크기가 작을 때 정규분포 대용- E(X) = 0 - Var(X) = \frac{k}{k-2} \quad (k > 2)
      카이제곱 분포 X \sim \chi^2(k)- 모집단의 분산 추정 및 독립성/동질성 검정 - 표준정규분포 변수들의 제곱 합- E(X) = k - Var(X) = 2k
      F-분포 X \sim F(d_1, d_2)- 두 집단의 분산 동일성 검정 (분산 분석) - 두 카이제곱 분포 변수의 비율- E(X) = \frac{d_2}{d_2-2} \quad (d_2 > 2) - Var(X) = \text{식 생략 (복잡)}
  • 인사이트: 데이터 본연의 특성(Domain)에 적합한 분포 모델링을 통해 분석 오류(Bias)를 최소화하고, 중심극한정리를 활용한 표본 기반의 모수 추론으로 데이터 기반 의사결정의 통계적 신뢰성을 확보해야 함.
