3. 선형회귀분석 (Linear Regression Analysis)

  • 정의: 하나 이상의 독립변수(X)와 연속형 종속변수(Y) 간의 선형 관계를 모델링하여 미래 값을 예측하고 변수 간 영향력을 분석하는 통계적 기법
  • 키워드: 회귀계수, 최소제곱법(OLS), 잔차, 결정계수(R²), 다중공선성, 단순회귀, 다중회귀, 정규성, 등분산성, F-검정
  • 분류: 회귀분석
  • 회귀계수: 회귀식(f(x))의 형태를 결정하는 파라미터(a, b)
  • 선형회귀: 회귀분석에서 사용하는 회귀식이 ‘파라미터에 관한’ 1차식이 될때 선형회귀로 분류
  • 결정계수(R²): 회귀식이 잘 들어맞는지 평가하는 지표(0~1)
    • 조정 결정계수(R’²): 설명변수 개수 k에 따라 조정한 결정계수
    • R^2 = SSR / SST = 1 - (SSE/SST)
    • 결정계수 수식
      단계계산방법설명핵심
      총 변동 계산SST: \sum(Y_i - \bar{Y})^2Y 평균과 관측값 차이 제곱합전체 변동량
      회귀에 의한 변동 계산SSR: \sum(\hat{Y}_i - \bar{Y})^2예측값과 평균의 차이 제곱합모델 설명 변동
      잔차 변동 계산SSE: \sum(Y_i - \hat{Y}_i)^2실제값과 예측값 차이 제곱합설명 못한 변동
      결정계수 계산R^2 = SSR / SST회귀변동 / 총변동설명력 비율
      결정계수 해석결정계수 값 크기 평가모델 설명력 수준 평가1에 가까울수록 ↑
  • 4가지 가정: <선독정등>
    항목설명핵심
    선형성X와 Y 간의 관계가 선형임선형 관계
    독립성관측값·잔차가 서로 독립Durbin-Watson 검정
    정규성잔차가 정규분포를 따름Q-Q plot
    등분산성X값에 관계없이 잔차의 분산이 일정잔차 플롯, 브루쉬-페이건 검정(BP검정)
  • 회귀 모형: (Y = a + bx + ε)
    항목설명핵심
    단순선형회귀Y = β₀ + β₁X + ε (독립변수 1개)1개 변수
    다중선형회귀Y = β₀ + β₁X₁ + β₂X₂ + … + ε (독립변수 2개 이상)다변수
    β₀ (절편)X=0일 때 Y의 기댓값기준값
    β₁ (회귀계수)X 1단위 증가 시 Y의 평균 변화량영향력 크기
    ε (오차항)모델로 설명되지 않는 무작위 잔차설명 불가 변동
  • 최소제곱법 (least squares, OLS):
    항목설명핵심
    목적실제값과 예측값 차이(잔차)의 제곱합을 최소화잔차 최소화
    추정잔차 제곱합(RSS)을 최소화하는 β 계수 산출계수 추정
    가정오차항이 평균 0, 등분산, 정규분포, 독립임을 가정가우스-마르코프
  • 회귀분석 유형:
    구분유형설명핵심
    선형단순 선형회귀독립1·종속1 관계 분석1:1 선형
    다중 선형회귀독립N·종속1 관계 분석N:1 선형
    비선형다항 회귀비선형 관계 다항식 모델링차수 확장
    곡선 회귀지수·로그 곡선 모델링비선형 함수
    로지스틱단순 로지스틱 회귀이항형 종속변수 예측이진 분류
    다중 로지스틱 회귀2개 이상 범주 예측다범주 분류
    정규화리지 (Ridge)L2 규제로 계수 조정다중공선성 해결
    라쏘 (Lasso)L1 규제로 계수 0화변수 선택
    엘라스틱넷 (ElasticNet)L1+L2 규제 동시 적용Ridge+Lasso
  • 인사이트: 선형회귀는 단순하지만 회귀계수의 명확한 해석 가능성 덕분에 의료·경제·사회과학 등 인과 분석이 중요한 분야에서 딥러닝 시대에도 여전히 핵심 기준선(Baseline) 모델로 활용됨. 모형 성능(R²)보다 가정 충족 여부(정규성·등분산성·다중공선성)를 먼저 검토하는 것이 선형회귀 적용의 올바른 순서이며, 가정 위반 시 추정치 자체의 신뢰성이 무너짐
