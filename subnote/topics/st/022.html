4. 불편추정량 (Unbiased Estimator)

정의: 추정량의 기대값이 모수와 일치하여 체계적 편향이 없는 추정량
키워드: E(θ̂)=θ, 편향 없음, 표본평균, 표본분산(n-1)
분류: 추정
불편추정량: 기댓값과 모수가 동일한 추정량
구분
설명
표본평균의 추정량
E(Xˉ1)=x1+x2+⋯+xnn=aE(\bar{X}_1) = \frac{x_1 + x_2 + \dots + x_n}{n} = aE(Xˉ1​)=nx1​+x2​+⋯+xn​​=a • 표본평균은 모수와 동일한 특성으로 인해 불편추정량임
표준분산의 추정량
V(Xˉ1)=(x1−a)2+(x2−a)2+⋯+(xn−a)2n−1V(\bar{X}_1) = \frac{(x_1 - a)^2 + (x_2 - a)^2 + \dots + (x_n - a)^2}{n-1}V(Xˉ1​)=n−1(x1​−a)2+(x2​−a)2+⋯+(xn​−a)2​ • 표준분산은 분모를 n으로 나누게 되면, 모수의 표준분산과 달라지게 되어, 불편추정량 생성을 위해 n−1n-1n−1로 나누어야 함
불편추정량의 자유도:
특징:
표본평균 X̄: E(X̄) = μ → 불편추정량
표본분산: s² = Σ(Xi-X̄)² / (n-1) → 불편성 확보
분모를 n으로 나누면 분산을 과소 추정 (편향 발생)
인사이트: 표본분산에서 분모를 n이 아닌 (n-1)로 나누는 이유는 불편추정량 확보 때문. 자유도(degree of freedom) 1 소모가 핀시험 답안 포인트.
