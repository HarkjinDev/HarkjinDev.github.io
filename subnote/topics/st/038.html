4. 다중회귀분석 (Multiple Regression Analysis)

  • 정의: 복수의 독립변수로 하나의 종속변수를 예측하는 회귀 분석 기법
  • 키워드: 독립변수 2개 이상, 종속변수 1개, R², 조정 R², 다중공선성, 유의성
  • 분류: 회귀분석
  • 수식: Y = β0 + β1X1 + β2X2 + ... + βpXp + ε
  • 특징:
    • R²(결정계수): 모형의 설명력 (0~1)
    • 조정 R²: 독립변수 증가 패널티 보정
    • 편회귀계수: 다중회귀분석의 기울기(β1, β2 등)
    • 회귀평면: 설명 변수가 2개일 때 그래프를 그리면 회귀모형은 직선이 아닌 평면
    • 다중공선성 주의 (VIF≥10 확인)
    • 다중회귀 결과: F검정(전체 모형 유의성)
  • 인사이트: 조정 R²(Adjusted R²)를 사용해야 독립변수 수 증가에 의한 결정계수 과대 추정 방지. 다중공선성(VIF≥10) 확인 필수.
