1. 상관분석 (Correlation Analysis)

  • 정의: 두 변수 간 선형적 관계의 방향과 강도를 수치로 측정하는 통계 분석 기법
  • 키워드: 피어슨, 스피어만, 켄달, 상관계수, 공분산, 산점도, 다중공선성
  • 분류: 회귀분석
  • 분산: 확률 변수가 기댓값으로부터 얼마나 떨어진 곳에 분포하는지를 가늠하는 값
    • V(X) = E(X²) - {E(X)}² → 제곱의 평균 - 평균의 제곱
  • 공분산: 2개의 확률변수의 상관 정도를 나타내는 값
    • σxy = (1/N)Σ(xi-μx)(yi-μy) → 2변수 상관 정도, 편차곱 평균
  • 상관계수: 두 변수 사이의 통계적 선형 관계의 강도와 방향을 -1에서 +1 사이의 수치로 나타낸 지표
    • 방향: 양의 상관(+) / 음의 상관(-) / 무상관(0)
  • 상관계수 종류:
    구분항목설명핵심
    모수피어슨(Pearson)연속형·정규분포 가정선형관계 측정
    비모수스피어만(Spearman)순위 기반·비정규순위 상관
    켄달(Kendall)순위 쌍 일치도소표본 적합
    범주파이계수(φ)2×2 이진변수이진 상관
    크래머(Cramér's V)다범주 명목변수범주형 상관
    편편상관(Partial)제3변수 통제 후순수 관계 추출
  • 상관계수 해석:
    구간해석핵심
    ±0.9 이상매우 강한 상관거의 선형
    ±0.7~0.9강한 상관유의미
    ±0.4~0.7중간 상관참고 수준
    ±0.2~0.4약한 상관미약
    ±0.2 미만무상관에 가까움관계 없음
    부호양(+): 동반 상승, 음(-): 역방향방향성
  • 피어슨 상관계수 구성:
    항목설명핵심
    공식r = Cov(X,Y) / σX·σY공분산 정규화
    범위-1 ≤ r ≤ +1방향+강도
    가정정규분포·등분산·선형모수 기반
    한계비선형 관계 미탐지인과≠상관
  • 인사이트: 상관관계 ≠ 인과관계. 상관계수는 선형 강도만 측정하므로, 산점도·편상관·비모수 기법을 병행해 허위상관과 인과관계 오류를 차단해야 함
