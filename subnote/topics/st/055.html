2. 데이터 어노테이션 (Data Annotation)

  • 정의: AI 모델 학습에 필요한 원시 데이터(이미지·텍스트·음성 등)에 레이블·태그·메타데이터를 부착하여 모델이 학습 가능한 형태로 가공하는 작업
  • 키워드: 레이블링, 태깅, 바운딩 박스, 시맨틱 세그멘테이션, 크라우드소싱, 품질 관리, RLHF, 휴먼 인 더 루프, 데이터 품질
  • 분류: 데이터 마이닝
  • 핵심 기법:
    기법특징설명
    바운딩 박스빠른 데이터 가공객체 테두리 사각형 틀 캡처
    폴리곤물체 크기 정밀 인식객체 외곽선 정밀 지점 표시
    폴리라인선형 형태 추적직선 및 곡선 다중 점 추적
    포인트작업이 간단함단일 픽셀 지정 및 개수 산출
    큐보이드3D 입체 정보 식별3D 육면체 형태 입체 박스
    시맨틱 세그멘테이션모든 픽셀 분류모든 픽셀 단위 클래스 분류
  • 라벨링과의 비교:
    항목어노테이션 (Annotation)레이블링 (Labeling)
    개념 범위레이블링을 포함하는 상위 개념어노테이션의 하위 개념
    작업 복잡도위치·관계·맥락 등 복합 정보 부착단순 클래스·태그 부여
    출력 형태바운딩 박스·세그멘테이션·키포인트 등카테고리·숫자·텍스트 태그
    데이터 유형이미지·영상·음성·텍스트 전반주로 텍스트·정형 데이터
    작업 난이도높음 (전문 지식 필요)낮음 (단순 분류 가능)
    비용높음낮음
    활용 예시객체 탐지·세그멘테이션·NER스팸 여부·감성 긍부정 분류
    주요 도구LabelImg, CVAT, Scale AIExcel, Label Studio
  • 인사이트: "AI의 성능은 데이터 품질에 달려있다"는 원칙 하에, 어노테이션은 AI 개발의 가장 핵심적인 선행 작업으로 자리잡음. 최근 LLM 기반 AI 보조 어노테이션이 확산되며 비용·속도 문제가 완화되고 있으나, 전문 도메인과 RLHF용 선호도 어노테이션에서는 여전히 고품질 인간 판단이 대체 불가한 핵심 요소로 남아 있음
