5. 오피니언 마이닝 (Opinion Mining)

정의: 텍스트에서 긍정/부정 등 주관적 의견과 감성을 추출하는 분석 기법
키워드: 감성 스코어링, 긍정/부정/중립, SNS, 여론 분석
분류: 데이터 마이닝
수행 절차:
① 텍스트 수집 (SNS, 리뷰, 뉴스)
② 전처리 (토크나이징, 불용어 제거)
③ 감성 사전 구축 또는 ML 모델 학습
④ 감성 분류 (긍정/부정/중립)
⑤ 인사이트 도출 및 시각화
비교 (텍스트 마이닝 vs 오피니언 마이닝):
구분
텍스트 마이닝
오피니언 마이닝
목적
정보/패턴 추출
감성·의견 분류
분석 대상
일반 비정형 텍스트
리뷰·SNS·댓글
출력
키워드·토픽
긍/부/중립 스코어
인사이트: 오피니언 마이닝은 텍스트 마이닝의 하위 분야로, 마케팅 전략·정책 평가·여론 모니터링에 활용된다.
