3. 연관 규칙 분석 (Association Rule Analysis)

정의: 항목 간 동시 발생 패턴을 지지도·신뢰도·향상도로 발견하는 분석 기법
키워드: 지지도, 신뢰도, 향상도, Apriori, 빈발 항목집합
분류: 데이터 마이닝
Apriori 알고리즘: 모든 항목 집합에 대한 지지도를 계산하는 대신, 최소 지지도 이상의 빈발항목집합만을 찾아내서 연관규칙을 계산하는 기법
연관규칙 탐색 원칙:
최소 지지도(minimum support) : 분석자가 정하는 최소한의 지지도
최소 신뢰도(minimum confidence) : 분석자가 정하는 최소한의 신뢰도
연관 분석(Association Analysis) 수행 단계
단계 1) 빈발항목집합의 생성 (Frequent Item Set Generation): 최소지지도(minimum support)를 만족하는 모든 빈발항목을 찾는 과정
단계 2) 연관규칙의 생성 (Rule Generation): 단계 1에서 찾은 빈발항목집합을 바탕으로, 최소신뢰도(minimum confidence) 기준을 통과하는 모든 연관 규칙을 도출

인사이트: 향상도(Lift)>1이면 A와 B가 독립보다 더 강하게 연관됨을 의미. Apriori 알고리즘은 최소 지지도·신뢰도 이상인 빈발 항목집합을 효율적으로 발견한다.
