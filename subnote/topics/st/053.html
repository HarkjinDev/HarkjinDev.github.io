[하위] 1-3. 확증적 데이터 분석(CDA, Confirmatory Data Analysis)

  • 정의: 가설을 설정한 후, 수집한 데이터로 가설을 평가하고 추정하는 전통적인 분석 기법
  • 절차: <가수통증>
    절차설명사례
    가설설정분석 목표별 가설 수립CCTV 범죄예방 효과
    데이터수집가설 관련 데이터 확보CCTV 및 범죄 기록
    통계분석변수 간 상관관계 분석설치 전후 상관 분석
    가설검증가설의 채택 또는 기각범죄 감소로 가설 채택
  • 비교표:
    비교 항목탐색적 데이터 분석 (EDA)확증적 데이터 분석 (CDA)
    목적수집된 데이터 기반 인사이트 도출도출된 가설에 대한 검증
    핵심 요소저항성, 잔차해석중심극한의 정리, P-Value
    기법히스토그램, 줄기잎그림, 상자수염그림, 산점도t-test, F-test, 분산분석(ANOVA), 상관분석, 회귀분석
    활용모형정립, 데이터마이닝, 가설도출가설검증, 유의성 검증
