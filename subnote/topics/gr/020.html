4. ISO/IEC 27017 (클라우드 보안 통제)

정의: ISO 27002를 기반으로 클라우드 서비스 보안 통제를 추가한 국제표준
키워드: 클라우드 보안 가이드, 추가 통제항목
분류: 보안관리체계
추가 통제항목: 클라우드 자산 소유권, 가상 환경 분리, CSP 역할과 책임 명확화
관련 표준: ISO 27018(클라우드 개인정보보호), ISO 27701(프라이버시 관리)
인사이트: 클라우드 서비스 이용 기업은 ISO 27017을 통해 CSP의 보안 책임 범위를 명확히 하고 공유 책임 모델의 자사 영역을 체계적으로 관리할 수 있음
