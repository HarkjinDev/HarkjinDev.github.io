8. 생성형 AI 서비스 이용자를 위한 개인정보 보호 가이드(26. 5.)

  • 정의: 생성형 AI 서비스 이용 전 과정에서 개인정보 수집·학습·보관·이전·파기 위험을 식별하고 대응하는 가이드
  • 키워드: 개인정보 생애주기, 가드레일, RAG, Machine Unlearning, PbD, 옵트아웃, 다크패턴, C2PA
  • 개인정보 처리 생애주기:
    구분생애주기설명핵심
    데이터 구축원천 데이터 수집공개·라이선스 데이터 스크래핑소스 검증 필요
    사전 학습원천데이터→Foundation Model 주입대규모 학습
    사후 학습SFT·RLHF로 모델 특화·품질 향상도메인 특화
    가드레일 적용배포 전 개인정보 출력 방어우회공격 차단
    서비스 이용프롬프트 입력질의어·파일·이미지 입력 단계민감정보 노출
    실시간 추론·참조RAG·Vector DB 지식 참조권한 상승 위험
    기록 저장/삭제대화이력 보관·옵트아웃다크패턴 주의
    외부 서비스 연동에이전틱 AI·서드파티 API 연동과도권한 위험
  • 데이터 구축 단계 주요 위험:
    생애주기주요 위험설명핵심
    원천 데이터 수집무분별한 수집아동정보·사생활 필터링 부재검증 없는 수집
    거부 의사 무시robots.txt 무시 강행 스크래법적 리스크
    사전/사후 학습파라미터 암기식별정보 가중치 고착·삭제 불가Unlearning 불가
    인간 편향 유입SFT·RLHF 검수자에 민감정보 노출편향 주입
    가드레일 적용우회 공격 취약Jailbreak·역프롬프트로 필터 무력화민감정보 출력
    콘텐츠 생성불투명성 위험워터마크·출처 미비로 투명성 결여합성이력 미식별
  • 서비스 이용 단계 주요 위험:
    생애주기주요 위험설명핵심
    프롬프트 입력데이터 무단 재활용입력데이터 학습용 오남용재활용 동의 없음
    재식별 위험누적 질의 결합 시 재식별 가능연계 식별 위험
    실시간 추론RAG 권한 상승Vector DB ACL 미흡·타인정보 조회접근제어 부재
    기록 저장/삭제다크패턴UI로 옵트아웃 권리행사 방해기만적 UI/UX
    삭제 의미 모호대화삭제≠즉시파기·단계적 관리완전삭제 불가
    부적절한 파기계정 탈취 시 2차 유출 위험무기한 보존
    외부 서비스 연동국외 이전 미식별해외 리전 이전 여부 미인지국외이전 관리
    과도한 API 권한최소권한 원칙 미비 정보 과다유출Agentic AI 위험
    서비스 결과 활용허위정보 위험워터마크 미적용 산출물 외부 유출가짜뉴스 2차피해
  • 정보주체(이용자) 관점 점검사항:
    구분점검사항설명핵심
    이용 전학습정책 사전 식별입력데이터 재활용·통제권 수준 확인학습 재활용 여부
    보관/파기 주기 검토대화이력 보관기간·범위 사전 식별처리방침 확인
    크롤러 수집 차단소유 웹페이지 robots.txt 설정 강제수집 거부 선언
    국외 이전 여부 점검처리방침·이메일로 이전국가 확인 후 입력해외이전 사전인지
    이용 중프롬프트 입력 통제사생활·중요 데이터 입력 배제민감정보 미입력
    Opt-out 명시적 활성화채팅기록 모델학습 활용동의 비활성화학습 제공 차단
    임시 대화 모드 활용기록 미저장 임시세션으로 흔적 최소화로컬·클라우드 최소화
    연동 권한 최소화연동 서비스 과도한 API 권한 확인최소권한 원칙
    워터마크 삽입 확인AI 저작물 표기·메타데이터 식별정보 확인C2PA 기준
    이용 후증빙 자산 즉각 확보개인정보 유출·왜곡 인지 시 캡처 확보증거 보전
    권리 구제 신청KISA 개인정보침해신고센터 신속 신청사후 구제
  • 개인정보처리자(서비스 제공자) 대응방안:
    구분대응방안설명핵심
    이용 전적법 처리 근거 수립개보법 제15조 정당한 이익 요건 검토목적·필요성 심사
    UI/UX 개선학습활용·국외이전·보존주기 투명 고지처리방침 명확화
    robots.txt 준수수집 엔진 단계 거부의사 실시간 준수스크래핑 배제
    이용 중Privacy by Design 구축프롬프트·학습 정제 단계 실시간 마스킹설계부터 PbD
    출력 가드레일 배치LLM↔UI 사이 가드레일 레이어 필수 배치환각·유출 실시간 차단
    옵트아웃 기능 제공원클릭 옵트아웃·대화보존 주기 직접 설정권리행사 보장
    안전한 연계TLS 암호화·OAuth 최소권한통신구간 보호
    투명성 확보 기술워터마크·C2PA 표식 필수 삽입생성물 출처 증명
    이용 후프라이버시 대시보드처리현황 실시간 조회·정정·삭제 창구 마련정보주체 권리
    권리구제 담당부서 운영침해 인지 시 즉각 대응 프로세스 가동처리방침 공개
  • 인사이트: 생성형 AI 개인정보 위험은 학습·서비스 전 생애주기에 분산되므로, 이용자는 Opt-out·임시모드·최소입력으로, 제공자는 PbD·가드레일·C2PA로 이중 방어체계를 구축해야 함
