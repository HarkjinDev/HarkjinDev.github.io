[하위] 1-2. 웹소켓 (WebSocket)

  • 정의: HTTP 프로토콜을 업그레이드하여 클라이언트-서버 간 단일 TCP 연결로 전이중 실시간 양방향 통신을 지원하는 프로토콜
  • 키워드: RFC 6455, HTTP Upgrade, ws/wss, 전이중, Upgrade 헤더, 실시간
  • 분류: 전송 프로토콜
  • 연결 수립 절차:
    1. 클라이언트 → HTTP GET + Upgrade: websocket 요청
    2. 서버 → 101 Switching Protocols 응
    3. 이후 TCP 연결 유지, 프레임 단위 양방향 통신
  • HTTP Polling vs WebSocket 비교:
    구분HTTP PollingWebSocket
    연결 방식반복 요청-응답단일 연결 유지
    지연높음낮음
    오버헤드HTTP 헤더 매번최초 1회만
    용도단순 갱신채팅, 주식, 게임
  • 인사이트: 웹소켓은 HTTP와 호환되는 포트(80/443)를 사용하여 방화벽 우회가 용이하며, wss://는 TLS 기반 보안 웹소켓으로 도청 방지.
