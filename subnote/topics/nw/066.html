4. 5G 네트워크 슬라이싱 (Network Slicing)

  • 정의: 물리적망을 다수의 독립된 가상망으로 논리적으로 분리해 서비스별 QoS를 제공하는 기술
  • 키워드: NSSF, NSSAI, AMF, SST, SD, 자원 격리
  • 분류: 이동통신/무선
  • 네트워크 슬라이싱 구성요소
    구분구성요소설명
    S/W 설정·제어NSSAI슬라이스 요청 파라미터
    NSSF슬라이스 선택 함수
    AMF슬라이스 정보 처리
    SST서비스 제공 슬라이스 유형
    SDSST 식별 추가정보
    인프라 구성VNF가상 H/W 인프라
    SDNSW 제어 네트워크
    MANOVNF 인프라 관리
  • 인사이트: 네트워크 슬라이싱은 하나의 인프라로 완전히 다른 SLA를 동시에 만족시키는 5G의 핵심 차별점. 슬라이스 간 자원 격리(security isolation)가 보안의 핵심 과제.
