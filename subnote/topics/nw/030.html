2. EGP (Exterior Gateway Protocol)

  • 정의: 서로 다른 자율 시스템(AS) 간 라우팅 정보를 교환하여 인터넷 전체의 경로를 관리하는 외부 게이트웨이 프로토콜
  • 키워드: AS 간 라우팅, BGP-4, 정책 라우팅, AS 번호, AS-PATH, 피어링, eBGP, iBGP, NLRI
  • 분류: 라우팅 / 주소체계
  • IGP vs EGP 비교:
    항목IGPEGP
    적용 범위AS 내부 라우팅AS 간 라우팅
    목적최적 경로 탐색정책 기반 경로 제어
    대표 프로토콜OSPF, RIP, EIGRPBGP-4
    경로 선택 기준메트릭(홉·대역폭)정책·속성 기반
  • BGP 동작 방식:
    항목설명핵심
    eBGP서로 다른 AS 간 BGP 세션외부 AS 연결
    iBGP동일 AS 내부 BGP 세션내부 경로 전파
    피어링 (Peering)BGP 라우터 간 TCP 179 포트로 세션 수립세션 기반 연결
    NLRI도달 가능한 네트워크 정보를 포함한 경로 광고경로 정보 교환
  • BGP 경로 속성:
    속성설명핵심
    AS-PATH경로가 통과한 AS 번호 목록, 루프 방지에 활용루프 방지
    NEXT-HOP다음 라우터의 IP 주소다음 홉 지정
    LOCAL-PREFAS 내부에서 선호 경로 결정 (높을수록 우선)내부 선호도
    MED외부 AS에 선호 진입 경로 전달 (낮을수록 우선)외부 선호도
    COMMUNITY경로 태깅으로 그룹 정책 적용정책 그룹화
    ORIGIN경로 정보 출처 (IGP·EGP·Incomplete)경로 출처
  • 활용 분야: ISP 간 연결, 멀티호밍, IX (인터넷 교환 포인트)
  • 인사이트: BGP-4는 현재 인터넷의 유일한 EGP로, 전 세계 10만 개 이상의 AS 경로를 관리하는 인터넷의 핵심 라우팅 프로토콜임. 최적 경로가 아닌 정책 기반 경로를 선택하는 특성상 BGP 설정 오류·하이재킹이 대규모 인터넷 장애로 이어질 수 있어 RPKI 기반 경로 검증이 보안 표준으로 부상하고 있음
