2. VXLAN (Virtual eXtensible LAN)

  • 정의: UDP 헤더로 L2 이더넷 프레임을 캡슐화하여 L3 네트워크에서 대규모 가상 L2 네트워크를 구현하는 터널링 기술
  • 키워드: VNI, VTEP, UDP 4789, 오버레이 네트워크, 클라우드, 멀티테넌시
  • 분류: 데이터링크/LAN
  • VLAN vs VXLAN 비교:
    구분VLANVXLAN
    식별자12비트 VLAN ID (최대 4,096)24비트 VNI (최대 16,777,216)
    전송 방식L2 스위칭UDP 캡슐화(L3 전송)
    확장성제한적대규모 클라우드 환경 적합
    표준IEEE 802.1QRFC 7348
  • 동작 구성요소:
    • VTEP(VXLAN Tunnel End Point): 캡슐화/역캡슐화 수행
    • VNI: 가상 네트워크 식별자
  • 인사이트: VXLAN은 클라우드 데이터센터에서 테넌트 격리와 대규모 VM 마이그레이션을 가능하게 하는 핵심 기술로, SDN·NFV와 결합되어 현대 네트워크 가상화의 기반이 됨.
