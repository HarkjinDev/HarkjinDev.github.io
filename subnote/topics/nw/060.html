4. MPLS VPN (Multi-Protocol Label Switching VPN)

  • 정의: 4바이트 레이블 기반 2계층 스위칭으로 고속 전달과 VPN을 구현하는 기술
  • 키워드: Label, LER, LSR, FEC, LSP, LDP, LIB, CoS, Traffic Engineering
  • 분류: 네트워크
  • 특징:
    항목설명핵심
    고속 전송3계층 라우팅을 2계층 스위칭으로 대체2계층 스위칭
    QoS 지원CoS별 차등적 우선순위 서비스 제공CoS 차등 서비스
    VPNLabel로 VPN 구분해 보안 제공Label 기반 VPN
    Traffic Engineering장애 시 대체경로 설정해 망 복구대체경로 복구
  • 구성요소:
    구분구성요소핵심 기술
    네트워크 장비LER (Label Edge Router)목적지 분류·Label 부착
    LSR (Label Switch Router)Label 스위칭으로 패킷 전달
    프로토콜FEC (Forwarding Equivalence Class)동일 경로로 매핑되는 패킷 그룹
    LSP (Label Switched Paths)FEC 기반 사전결정된 전송경로
    LDP (Label Distribution Protocol)LSR간 Label 정보 분배
    LIB (Label Information Base)Label 바인딩 정보 저장 DB
  • 인사이트: Label 기반 2계층 스위칭으로 고속 전달, QoS, VPN, 장애복구를 동시에 지원
