2. VXLAN (Virtual eXtensible LAN)

  • 정의: UDP 헤더로 L2 이더넷 프레임을 캡슐화하여 L3 네트워크에서 대규모 가상 L2 네트워크를 구현하는 터널링 기술
  • 키워드: VNI, VTEP, UDP 4789, 언더레이/오버레이 네트워크, 클라우드, 멀티테넌시
  • 분류: LAN
  • VXLAN vs VLAN 비교:
    항목VLANVXLAN
    식별자VID 12bit (4,094개)VNI 24bit (약 16M개)
    전송 방식L2 스위칭UDP 캡슐화(L3 전송)
    동작 계층L2L2 over L3 오버레이
    확장성제한적대규모 클라우드 환경 적합
    VM 이동성동일 VLAN 내 제한L3 경계 초월 이동 가능
    표준IEEE 802.1QRFC 7348
  • 핵심 구성요소:
    구성요소설명핵심
    VNI (VXLAN Network Identifier)24bit 가상 네트워크 식별자·약 16M 세그먼트테넌트 식별자
    NVE (Network Virtual Interface)VTEP의 논리적 가상 인터페이스오버/언더레이 연결
    VTEP (VXLAN Tunnel End Point)VXLAN 캡슐화·역캡슐화 수행 종단점(장비)캡슐화 주체
    오버레이 (Overlay)VTEP 간 논리적 L2 터널 네트워크가상 L2 망
    언더레이 (Underlay)VXLAN 트래픽을 실어 나르는 물리 IP 네트워크L3 물리망
    VXLAN 헤더8byte·VNI 24bit 포함 UDP 캡슐화 헤더UDP 포트 4789
  • VTEP 동작 방식:
    단계설명핵심
    1. 프레임 수신VM이 이더넷 프레임 송신L2 원본 프레임
    2. VNI 매핑출발지 VTEP이 VM→VNI 매핑 확인테넌트 식별
    3. 캡슐화MAC-in-UDP 캡슐화·목적지 VTEP IP 설정오버레이 터널
    4. IP 전송언더레이 IP 네트워크로 라우팅 전송L3 물리망 경유
    5. 역캡슐화목적지 VTEP이 VXLAN 헤더 제거·원본 프레임 전달원본 복원
  • 인사이트: VXLAN은 클라우드 데이터센터에서 테넌트 격리와 대규모 VM 마이그레이션을 가능하게 하는 핵심 기술로, SDN·NFV와 결합되어 현대 네트워크 가상화의 기반이 됨.
