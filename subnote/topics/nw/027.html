11. HTTP/3

  • 정의: TCP 기반의 HTTP/2 한계를 극복하기 위해 UDP 기반의 QUIC 프로토콜 위에서 동작하는 차세대 HTTP 표준
  • 키워드: RFC 9114, QUIC, UDP, 0-RTT, 멀티플렉싱, HOL Blocking 해소, TLS 1.3 내장, 연결 마이그레이션, 스트림
  • 분류: 네트워크 / 프로토콜

  • 인사이트: HTTP/3는 TCP의 구조적 한계인 HOL Blocking을 UDP 기반 QUIC으로 근본적으로 해결하며, 0-RTT·연결 마이그레이션으로 모바일·불안정 네트워크 환경에서 체감 성능을 크게 향상시킴. TLS 1.3 기본 내장으로 보안이 프로토콜 수준에서 보장되어 웹 표준의 새로운 기준으로 빠르게 자리잡고 있음
