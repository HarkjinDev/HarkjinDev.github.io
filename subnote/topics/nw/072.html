4. AI-RAN (AI-based Radio Access Network)

  • 정의: 무선접속망(RAN)에 AI/ML을 내재화하여 무선 자원 할당, 빔포밍, 간섭 관리, 셀 최적화를 실시간 자율 수행하는 차세대 RAN 기술
  • 키워드: O-RAN, xApp/rApp, Near-RT RIC, Non-RT RIC, 빔관리, 자원 스케줄링, 에너지 효율
  • 분류: 차세대 네트워크
  • AI-RAN/O-RAN 구성요소:
    구분구성요소설명핵심
    AI-RANNon-RT RIC1초 이상 주기 AI·ML 기반 정책·모델 관리장주기 지능 제어
    Near-RT RIC10ms~1초 주기 실시간 RAN 파라미터 최적화단주기 실시간 제어
    xAppNear-RT RIC 위에서 동작하는 서드파티 앱지능형 앱 실행
    rAppNon-RT RIC 위에서 동작하는 서드파티 앱정책 관리 앱
    기지국 기능O-CU (중앙 유닛)RRC·PDCP 프로토콜 처리·중앙 집중 기능상위 계층 처리
    O-DU (분산 유닛)RLC·MAC·하위 PHY 실시간 처리중간 계층 처리
    O-RU (무선 유닛)상위 PHY·RF 안테나 신호 송수신 처리무선 신호 처리
    전송 인터페이스Open Fronthaul (F1)O-DU↔O-RU 간 표준화된 개방형 인터페이스개방형 프론트홀
    E2 인터페이스Near-RT RIC↔O-CU·O-DU 간 제어 인터페이스RIC 제어 인터페이스
    A1 인터페이스Non-RT RIC→Near-RT RIC 정책 전달 인터페이스정책 전달
    O1 인터페이스SMO↔RAN 구성요소 간 관리·운영 인터페이스운영 관리
    관리SMO RAN 전체 구성·운영·관리 총괄통합 관리 플랫폼
    클라우드 인프라O-RAN 기능을 가상화·컨테이너 기반으로 운용클라우드 네이티브
  • AI-RAN 주요 응용: 동적 스펙트럼 관리, 에너지 절감(트래픽 예측 기반 슬립 모드), 안테나 빔 최적화
  • 인사이트: AI-RAN은 O-RAN 오픈 인터페이스를 기반으로 다양한 벤더의 AI 앱을 RAN에 탑재하는 개방형 생태계를 형성. 에너지 효율과 주파수 효율을 동시에 극대화하는 6G RAN의 핵심 기술.
