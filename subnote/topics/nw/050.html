4. DiffServ (Differentiated Services)

  • 정의: IP 패킷의 DSCP 필드를 이용하여 트래픽을 클래스별로 분류하고 차등적 포워딩 처리를 적용하는 QoS 모델
  • 키워드: RFC 2474, DSCP, PHB, EF/AF/BE, 토큰 버킷, 폴리싱, 쉐이핑, 에지 라우터
  • 분류: QoS
  • PHB(Per-Hop Behavior) 종류:
    PHBDSCP 값설명대상 트래픽
    EF(Expedited Forwarding)46최소 지연/손실 보장VoIP, 실시간
    AF(Assured Forwarding)12~444클래스×3드롭비즈니스 앱
    BE(Best Effort)0기본 처리일반 인터넷
  • DiffServ 동작: 에지 라우터에서 패킷 분류·마킹 → 코어 라우터는 DSCP 기반 PHB만 적용
  • DiffServ vs IntServ 비교:
    구분IntServDiffServ
    자원 예약RSVP per-flow없음(클래스 기반)
    확장성낮음(상태 유지)높음
    보장 수준강한 보장상대적 보장
    적용 범위소규모대규모 인터넷
  • 인사이트: DiffServ는 확장성이 높은 QoS 모델로 코어 라우터에서 per-flow 상태 유지가 불필요함. IntServ의 RSVP 시그널링 부하 없이 대규모 망에서 실용적으로 적용 가능.
