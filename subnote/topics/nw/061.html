5. WebRTC (Web Real-Time Communication)

  • 정의: 웹 브라우저에서 별도 플러그인·앱 설치 없이 JavaScript API만으로 음성·영상·데이터 실시간 통신을 지원하는 오픈 표준
  • 키워드: W3C/IETF, ICE, STUN, TURN, SDP, P2P, getUserMedia, DataChannel
  • 분류: 이동통신/무선
  • WebRTC 핵심 컴포넌트:
    컴포넌트역할
    getUserMedia마이크·카메라 접근
    RTCPeerConnectionP2P 미디어 스트림 연결
    RTCDataChannel임의 데이터 P2P 전송
    ICE/STUN/TURNNAT 통과 및 연결 설정
    SDP(Session Description Protocol)미디어 협상
  • NAT 통과: STUN(공인 IP 확인) → 실패 시 TURN(릴레이 서버) 경유
  • 인사이트: WebRTC는 브라우저 기반 화상회의·실시간 협업을 플러그인 없이 구현할 수 있어, Zoom·Google Meet 등의 기반 기술로 활용됨. DTLS-SRTP로 미디어 암호화가 의무화.
