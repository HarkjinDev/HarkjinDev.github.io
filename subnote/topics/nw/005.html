4. ASN.1 (Abstract Syntax Notation One)

정의: 통신 프로토콜에서 데이터 구조를 언어·플랫폼에 독립적으로 정의하는 추상 구문 표기 표준
키워드: ISO/ITU-T, BER/DER/CER/PER, 직렬화, SNMP, X.509, 인코딩 규칙
분류: 네트워크 기초
인코딩 규칙 비교:
규칙
특징
주요 용도
BER(Basic Encoding Rules)
가장 기본, 유연한 인코딩
SNMP, LDAP
DER(Distinguished Encoding Rules)
BER 부분집합, 유일한 인코딩
X.509 인증서
CER(Canonical Encoding Rules)
긴 데이터에 최적화
보안 프로토콜
PER(Packed Encoding Rules)
비트 단위 최적화, 경량
이동통신 프로토콜
활용 분야: X.509 인증서, SNMP MIB, LTE/5G 무선 프로토콜, LDAP
인사이트: ASN.1은 이종 시스템 간 데이터 호환성을 보장하는 핵심 표준으로, 보안 인증서(PKI)와 네트워크 관리(SNMP)에 직결되므로 SEC 도메인과 연계하여 정리할 것.
