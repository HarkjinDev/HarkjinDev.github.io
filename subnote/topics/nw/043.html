1. NFV (Network Function Virtualization)

  • 정의: 방화벽·로드밸런서·IDS 등 전용 하드웨어 장비의 네트워크 기능을 범용 x86 서버에서 소프트웨어(VNF)로 구현하는 가상화 기술
  • 키워드: ETSI, VNF, NFVI, NFV-MANO, 오케스트레이션, 상용 서버, SDN 연계
  • 분류: 네트워크 가상화/SDN
  • NFV 구조:
    구성요소설명
    VNF(Virtual Network Function)소프트웨어로 구현된 네트워크 기능
    NFVI(NFV Infrastructure)컴퓨팅·스토리지·네트워크 자원 풀
    NFV-MANOVNF 생명주기 관리·오케스트레이션
    VNFM개별 VNF 관리
    NFVO(Orchestrator)서비스 체이닝, 자원 조율
  • 기존 방식 vs NFV 비교:
    구분전용 장비NFV
    확장성장비 추가 필요SW 배포로 즉시 확장
    비용고비용 전용 HW범용 서버 활용
    유연성낮음동적 서비스 체이닝
  • 인사이트: NFV와 SDN은 상호 보완 관계로, NFV는 기능 가상화, SDN은 제어 평면 분리를 담당. 5G 코어 네트워크는 NFV/SDN 기반 클라우드 네이티브 아키텍처로 구현.
