13. 신뢰성 테스트 (Reliability Test)

  • 정의: 소프트웨어가 주어진 환경에서 특정 기간 동안 정상적으로 기능을 수행하는 능력(신뢰성)을 측정하고 검증하는 테스트
  • 키워드: MTBF(Mean Time Between Failures), MTTF, MTTR, 가용성, 결함 내성, 신뢰성 성장 모델
  • 분류: 테스트
  • 신뢰성 측정 지표:
    지표공식설명
    MTTF평균 고장 간격첫 고장까지의 평균 시간
    MTTR평균 수리 시간고장 후 복구까지의 평균 시간
    MTBFMTTF + MTTR고장과 고장 사이의 평균 시간
    가용성MTBF / (MTBF + MTTR)정상 운영 비율
  • 인사이트: 신뢰성 테스트는 장시간 부하 환경에서의 안정성 측정이 목적으로, MTBF·MTTR을 SLA 지표와 연계하여 가용성 목표 달성 여부를 정량적으로 평가.
