[하위] 5-1. 사용성 평가 (Usability Evaluation)

  • 정의: 특정 사용자가 특정 맥락에서 목표를 효과적·효율적·만족스럽게 달성할 수 있는 정도를 측정·평가하는 활동
  • 키워드: 닐슨 10대 원칙, 사용성 테스트, 휴리스틱 평가, 전문가 리뷰, 사용자 테스트
  • 분류: 요구공학
  • 사용성 평가 방법 비교:
    방법설명참여자
    휴리스틱 평가 (Heuristic Evaluation)닐슨 10대 원칙 기반 전문가 점검전문가 3~5인
    사용성 테스트 (Usability Test)실제 사용자가 태스크 수행 관찰실사용자
    인지적 워크스루 (Cognitive Walkthrough)학습 용이성 중심 시나리오 평가전문가
    설문/인터뷰주관적 만족도 측정사용자
  • 닐슨의 10대 사용성 원칙 (핵심):
    • 시스템 상태 가시성 / 현실 세계 매칭 / 사용자 통제·자유 / 일관성·표준 / 오류 방지 / 인식 vs 기억 / 유연성·효율성 / 미학적·최소 디자인 / 오류 인식·진단·복구 지원 / 도움말·문서화
  • 인사이트: 전문가 5인의 휴리스틱 평가로 전체 사용성 문제의 약 75%를 발견할 수 있어 비용 대비 효과가 높으며, 사용자 테스트와 병행 시 보완 효과 극대화.
