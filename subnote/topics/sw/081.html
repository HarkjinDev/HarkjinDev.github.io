2. 동적 테스트 (Dynamic Test)

  • 정의: 소프트웨어를 실제 실행하여 결함을 발견하는 테스트
  • 키워드: 실행 기반, 블랙박스·화이트박스, 기능 검증, 테스트 케이스, 동적 분석
  • 분류: 테스트
  • 동적 테스트 vs 정적 테스트 비교:
    구분동적 테스트정적 테스트
    실행 여부코드 실행 필수코드 실행 불필요
    방법블랙박스·화이트박스리뷰·인스펙션·정적 분석
    발견 시점실행 후작성 중·완료 후
    결함 유형런타임 오류 중심코드 결함·논리 오류
  • 테스트 유형:
    항목명세 기반구조 기반경험 기반
    근거요구사항·명세서내부 코드·구조테스터 경험·직관
    대표 기법동등분할·경계값·상태전이구문·결정·조건 커버리지탐색적 테스트·오류 추정
    특징기능 검증 중심코드 실행 경로 검증문서화되지 않은 결함 발견
  • 인사이트: 동적 테스트는 실제 환경에서의 동작 검증이 목적으로, 정적 테스트와 상호 보완적이며 초기 정적 테스트로 기본 결함 제거 후 동적 테스트 효율이 향상됨.
