[하위] 2-1. 응집도 (Cohesion)

정의: 모듈 내부 구성 요소들이 얼마나 밀접하게 관련되어 하나의 목적을 수행하는지를 나타내는 척도
키워드: 7단계, 기능적 응집도, 우연적 응집도, 높을수록 좋음
분류: 설계 원리/패턴
응집도 7단계 (높음 → 낮음): <기순교절시논우>
수준
유형
설명
최고
기능적 (Functional)
단일 기능만 수행 (최이상적)
↑
순차적 (Sequential)
이전 출력이 다음 입력으로 연결
↑
통신적 (Communicational)
동일 데이터 참조·처리
↑
절차적 (Procedural)
순서 관련 여러 기능 포함
↑
시간적 (Temporal)
같은 시점에 실행되는 기능 묶음
↑
논리적 (Logical)
유사 범주의 기능을 선택적 실행
최저
우연적 (Coincidental)
아무 관계 없는 기능의 임의 집합
인사이트: 시험에서는 7단계를 순서와 함께 예시로 구분하는 것이 핵심이며, 기능적 응집도가 단일 책임 원칙(SRP)과 동일한 개념임을 연결하면 고득점.
