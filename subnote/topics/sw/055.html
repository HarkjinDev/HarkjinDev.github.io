4. 리팩토링 (Refactoring)

  • 정의: 외부 동작을 유지하면서 내부 코드 구조를 개선하여 가독성·유지보수성을 높이는 체계적인 코드 개선 활동
  • 키워드: 코드스멜, 외부 행위 유지, 코드 구조 개선, 점진적 개선, 자동화 테스트, 회귀 테스트
  • 분류: 설계 원리/패턴
  • 주요 리팩토링 기법:
