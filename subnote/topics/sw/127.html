[하위] 1-1. FTA (Fault Tree Analysis)

  • 정의: 시스템 최상위 고장 사건(Top Event)으로부터 하향식으로 원인을 AND/OR 게이트로 연결하여 결함 트리를 구성하는 귀납적 안전성 분석 기법
  • 키워드: Top Event, Fault Tree, 게이트(AND/OR), 기본사건(Basic Event), 최소 절단집합(MCS), 정성적·정량적 분석
  • 분류: 안전성 분석
  • FTA 주요 게이트:
    게이트기호의미
    AND 게이트반원 (∩)모든 입력 사건 발생 시 출력 사건 발생
    OR 게이트화살 곡선 (∪)하나 이상의 입력 사건 발생 시 출력 사건 발생
  • FTA 분석 결과:
    • 정성적 분석: 최소 절단집합(MCS, Minimal Cut Set) 도출 → 시스템 취약 경로 파악
    • 정량적 분석: 각 기본사건 발생 확률로 Top Event 발생 확률 계산
  • 인사이트: 최소 절단집합(MCS)이 FTA의 핵심 결과물로, MCS의 크기가 작을수록(단일 사건) 더 위험한 경로이며 우선적으로 개선 대상이 됨.
