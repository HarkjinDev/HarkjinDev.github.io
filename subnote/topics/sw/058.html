[하위] 5-1. 역공학 (Reverse Engineering)

  • 정의: 기존 소프트웨어의 소스코드·바이너리·데이터베이스를 분석하여 설계·명세 수준의 추상화된 표현을 추출하는 활동
  • 키워드: 소스코드 분석, 아키텍처 복구, 데이터 역공학, 디컴파일, 레거시 문서화
  • 분류: 설계 원리/패턴
  • 역공학 유형:
    유형대상산출물
    코드 역공학소스코드클래스 다이어그램, 아키텍처
    데이터 역공학DB 스키마ERD, 데이터 사전
    UI 역공학화면·인터페이스와이어프레임, UI 명세
    바이너리 역공학실행파일소스코드 복원(디컴파일)
  • 인사이트: 역공학은 아키텍처 드리프트 복구와 레거시 문서화에 필수적이며, 보안 분야에서 악성코드 분석에도 활용되나 저작권·윤리적 제한이 따름.
