5. 사용자 경험 (UX, User Experience)

  • 정의: 사용자가 제품·서비스와 상호작용하는 전 과정에서 느끼는 종합적인 경험
  • 키워드: 사용성(Usability), 접근성(Accessibility), UX 디자인, 페르소나, 사용자 여정 지도
  • 분류: 요구공학
  • UX 구성 요소 (Peter Morville의 허니콤 모델):
    요소설명
    유용성 (Useful)사용자 목표 달성에 기여하는가
    사용가능성 (Usable)쉽고 효율적으로 사용 가능한가
    바람직함 (Desirable)감성적으로 매력적인가
    발견가능성 (Findable)필요 정보를 찾기 쉬운가
    접근성 (Accessible)장애인 포함 모두가 사용 가능한가
    신뢰성 (Credible)신뢰를 줄 수 있는가
    가치 (Valuable)비즈니스·사용자 모두에게 가치 있는가
  • 인사이트: UX는 UI(화면)보다 넓은 개념으로 서비스 전 접점의 경험을 포괄하며, 페르소나·사용자 여정 지도가 UX 설계의 핵심 도구.
