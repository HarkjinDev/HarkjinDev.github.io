22. 적대적 테스트(Adversarial Testing)

  • 정의: 시스템에 의도적으로 악의적인 공격을 시뮬레이션하여, 예상치 못한 오류나 취약점을 분석하는 테스팅 기법
  • 키워드: 보안취약점 발견, 중독공격, 회피공격, 탐색적 공격, 교란, 적대적 예시, 추출공격, 전도공격
  • 분류: 테스트
  • 핵심 기법:
    구분핵심 기술설명
    중독공격 (Poisoning)서비스 거부잘못된 레이블 데이터 대량 주입
    백 도어 삽입트리거 감지 시 공격자 의도대로 동작
    회피공격 (Evasion)교란 (Perturbation)육안 불가 미세 노이즈로 예측 변경
    적대적 예시 (Adversarial Example)교란 추가 입력으로 모델 오인식 유발
    탐색적공격 (Exploratory)모델 추출공격 (Model Extraction)쿼리 결과 분석으로 모델 복제
    모델 전도 공격 (Model Inversion)출력값 분석으로 학습 데이터 역추출
